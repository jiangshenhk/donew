## 8）落到卖 put 策略

### QLD

**结论：可以观察，但不能急着卖。**

今天QLD跌3.43%，理论上权利金会变肥，但这不是普通回踩，而是宏观压估值回调。此时不能因为权利金变厚就冲进去。

| 项目 | 执行 |
|---|---|
| 是否卖Put | ⚠️ 只等确认，不接飞刀 |
| 仓位 | 计划仓位20%-30%以内 |
| Delta | 0.08-0.12优先，最高不超过0.15 |
| Strike | 远OTM，必须留足安全垫 |
| 入场 | 等QQQ/NVDA/SMH至少横住，不能边跌边卖 |
| 禁止 | VIX上20、10Y继续冲、WTI破100时提高仓位 |

### QLD 今晚动作清单

- [ ] QQQ是否停止放量下跌？
- [ ] NVDA是否从-4%以上跌幅中开始收回？
- [ ] SMH是否不再继续破位？
- [ ] VIX是否低于20？
- [ ] 10Y是否不再继续上冲？
- [ ] WTI是否没有突破100并继续拉升？
- [ ] 可选Put Delta是否≤0.12-0.15？

满足4项以上：只可小仓卖第一笔远OTM。  
满足6项以上：才允许分批第二笔。  
少于4项：不做，继续空仓等。
