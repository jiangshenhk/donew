## 8）落到卖 Put 策略

# QLD

## 结论

> **今晚不抢卖，只等开盘后 30–60 分钟确认承接；若没有承接，宁可不做。**

| 项目 | 操作 |
|---|---|
| 是否卖 Put | ⚠️ 条件允许后才做 |
| 仓位 | 正常仓位 20%–30% |
| Delta | 优先 0.08–0.12，最高不超过 0.15 |
| 到期 | 7–21 DTE |
| 行权价 | 必须远 OTM，不能贴近现价 |
| 最佳时机 | 开盘后恐慌释放、指数不再创新低时 |
| 禁止动作 | 盘前追权利金 / 高 Delta 接飞刀 |

### 今晚检查框

- [ ] QQQ 开盘后没有继续放量破低  
- [ ] NVDA / SMH 至少横住  
- [ ] 10Y 没有继续冲高  
- [ ] Brent 没有继续快速拉升  
- [ ] VIX 没有突破或快速接近 20  
- [ ] Put Delta ≤0.12 优先  

满足 4 项以上，才允许小仓远 OTM；少于 4 项，放弃。

# MSTR

## 结论

> **继续默认禁卖。**

MSTR 当前的问题不是单一价格问题，而是三重压力叠加：

- BTC 没有重新强于 Nasdaq
- 长端利率上冲压制高 beta
- 风险资产环境不支持加密链扩散

所以：

> **权利金越肥，越说明市场在给你风险补偿，不等于便宜。**

| 项目 | 操作 |
|---|---|
| 是否卖 Put | ❌ 默认不卖 |
| 唯一例外 | BTC 强于 QQQ + MSTR 强于 BTC + VIX 回落 |
| 仓位 | 即使满足也只能极轻仓 |
| Delta | ≤0.08–0.10 |
| 行权价 | 极远 OTM |

### MSTR 解禁条件

- [ ] BTC 重新站稳并强于 QQQ  
- [ ] MSTR 盘中强于 BTC  
- [ ] 10Y 停止上冲  
- [ ] Nasdaq 不再放量下跌  
- [ ] VIX 明显回落  

没有同时满足，就继续禁卖。
