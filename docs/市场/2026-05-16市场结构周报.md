## 1）资金流向异动｜本周最重要变化

本周真正重要的变化不是指数一度创新高，而是：

> **油价、10Y、美元、VIX在周末前同时转强，市场开始重新交易“通胀更粘、利率更高、估值要下修”。**

前半周市场仍然是AI抱团：S&P 500、Nasdaq继续刷新高位，资金集中买入AI基础设施、半导体和少数超大盘科技。后半周结构明显变了：油价快速上冲，美债收益率冲到4.5%以上，美元走强，黄金和BTC同步下跌，半导体、高beta、小盘、加密链被集中卖出。

本周资金真正做了三件事：

| 流出 | 流入 | 含义 |
|---|---|---|
| 半导体扩散、高beta科技、QLD、TSLA | 美元、现金、短久期资产 | 先降低风险敞口 |
| 黄金、GLD、无息资产 | 美元/收益率交易 | 高利率压制传统避险资产 |
| 小盘、非核心成长 | 少数现金流龙头 | 市场只保留确定性资产 |
| BTC、MSTR、COIN | 观望/现金 | 加密链没有确认risk-on |
| 航空、消费弹性、油价受损链 | 能源、油气现金流 | 油价重新成为通胀主线 |

一句话：

> **本周不是牛市结束，而是高位趋势被油价和利率强行拉回现实。**

### 谁在说真话？

**债券、油价、美元在说真话。**

股票前半周说：AI盈利和指数动能仍在。  
债券说：通胀压力回来了，降息空间被压缩。  
油价说：能源冲击和地缘风险没有解除。  
美元说：资金先买流动性，不急着扩散risk-on。  
BTC说：这不是全面风险偏好恢复。

所以本周的判断不是“全面转空”，而是：

> **指数多头仍有残余，但卖Put环境已经从顺风变成逆风。**

## 9）落到卖 put 策略

### QLD

**结论：可以观察，但不能急着卖。**

本周QLD的权利金会明显变肥，但这不是普通回踩，而是宏观压估值回调。此时不能因为权利金变厚就冲进去。

| 项目 | 执行 |
|---|---|
| 是否卖Put | ⚠️ 只等确认，不接第一刀 |
| 仓位 | 计划仓位20%–30%以内 |
| Delta | 0.08–0.12优先，最高不超过0.15 |
| Strike | 远OTM，必须留足安全垫 |
| 入场 | 等QQQ/NVDA/SMH至少横住，不能边跌边卖 |
| 禁止 | VIX上20、10Y继续冲、WTI突破100并延续时提高仓位 |

#### QLD 下周动作清单

- [ ] QQQ是否停止放量下跌？
- [ ] NVDA是否开始收回跌幅？
- [ ] SMH是否不再继续破位？
- [ ] VIX是否低于20？
- [ ] 10Y是否回到4.5%下方？
- [ ] WTI是否没有突破100并继续拉升？
- [ ] 可选Put Delta是否≤0.12–0.15？

满足4项以上：只可小仓卖第一笔远OTM。  
满足6项以上：才允许分批第二笔。  
少于4项：不做，继续空仓等。

### MSTR

**结论：继续禁卖。**

MSTR的问题不是“跌多了”，而是BTC、Nasdaq、VIX、美元这四个变量都没有给支持。MSTR不是BTC本体，而是BTC + 股市风险偏好 + 溢价 + IV的高beta组合。

| 项目 | 执行 |
|---|---|
| 是否卖Put | ❌ 不卖 |
| 唯一例外 | BTC强于QQQ + MSTR强于BTC + VIX回落 |
| Delta | ≤0.10 |
| Strike | 极远OTM |
| 仓位 | 即使满足也只能极小仓 |
| 禁止 | 因为权利金肥而卖、因为MSTR跌多了而卖、因为BTC横盘而卖 |

#### MSTR 下周动作清单

- [ ] BTC是否重新强于QQQ？
- [ ] MSTR是否强于BTC？
- [ ] COIN是否止跌？
- [ ] Nasdaq是否止跌？
- [ ] VIX是否回落？
- [ ] DXY是否停止上行？

全部满足才重新观察。  
当前默认：**不卖。**
