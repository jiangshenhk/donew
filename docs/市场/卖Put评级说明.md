# 用AI工具筛选合适的Put和价格：从市场恐慌里寻找风险溢价

> 这个页面用于解释我如何用 AI 辅助筛选 Sell Put。  
> 核心不是让 AI 预测涨跌，而是让 AI 帮我把 **市场环境、波动率、期权链、风险收益比** 标准化，判断当前有没有值得卖 Put 的风险溢价。

---

## 一句话理解这个方法

我的 Sell Put 筛选流程是：

```text
先看用AI看看市场有没有恐慌溢价
再看让AI今天适不适合卖 Put，确定风险系数，决定保守还是激进
最后在期权链里挑一个安全垫、Delta、年化收益都合适的价位
```

也可以理解为三句话：

1. **用Barchart上数据，让AI测温度**：用 IV、HV、IV Percentile、IV Rank、Expected Move 判断期权是否变贵；
2. **用AI获取13个主要指数，看环境**：结合今日市场趋势，判断这份高权利金是机会，还是陷阱；
3. **期权链选价位**：从预定的标的池子里面，根据 Delta、安全垫、年化收益和接货意愿，选择具体 Put。

---

## 第一步：从 Barchart 找期权市场温度

我会先去 Barchart 的标的页面，例如：

```text
https://www.barchart.com/stocks/quotes/QLD/overview
```

然后重点看 **Options Overview** 里的数据。

截图里的 QLD 示例数据大概是：

| 指标 | 示例值 | 作用 |
|---|---:|---|
| Implied Volatility | 46.53% | 当前期权市场给出的未来波动预期 |
| Historical Volatility | 31.56% | 标的过去实际波动水平 |
| IV Percentile | 89% | 当前 IV 在历史区间中处于较高位置 |
| IV Rank | 58.53% | 当前 IV 相对历史高低点的位置 |
| IV High | 60.85% | 过去区间 IV 高点 |
| IV Low | 26.32% | 过去区间 IV 低点 |
| Expected Move | 9.00 / 9.89% | 市场预期到期前大概波动范围 |
| Expected Range | 82.03 - 100.03 | 期权市场隐含的价格区间 |
| Put/Call Vol Ratio | 0.63 | 当日 Put / Call 成交量相对关系 |
| Put/Call OI Ratio | 0.27 | 存量持仓中 Put / Call 的关系 |

这些数据不是单独使用，而是一起看。

---

## 第二步：判断有没有“恐慌溢价”

Sell Put 的本质是卖保险。

所以我们希望在什么时候卖？

> **市场有点害怕，保险费变贵，但标的长期结构还没有坏掉。**

这时 Put 权利金可能变肥，存在风险溢价。

我重点看三个信号：

## 1. IV 是否明显高于 HV

```text
IV > HV
```

说明期权市场给出的未来波动预期，高于过去实际波动。

例如截图中：

```text
IV = 46.53%
HV = 31.56%
```

这说明当前期权价格里包含较高的波动预期，Put 权利金可能比平时更肥。

但要注意：

> IV 高，不等于一定要卖 Put。它可能是机会，也可能是市场真的在预警风险。

## 2. IV Percentile 是否处于高位

截图中：

```text
IV Percentile = 89%
```

这代表当前 IV 在历史观察区间里处于偏高位置。

我的理解是：

| IV Percentile | 含义 | Sell Put 观察 |
|---|---|---|
| 低于 30% | 权利金偏瘦 | 通常吸引力不足 |
| 30% - 60% | 正常区间 | 看标的和市场环境 |
| 60% - 80% | 偏高 | 可以开始重点观察 |
| 80% 以上 | 明显偏高 | 可能存在恐慌溢价，但必须看风险 |

所以 89% 不是“无脑卖”的信号，而是：

> **这里可能有风险溢价，值得让 AI 进一步分析。**

## 3. Expected Move 是否能帮助判断安全垫

截图中：

```text
Expected Move = 9.00 / 9.89%
Expected Range = 82.03 - 100.03
```

这表示期权市场认为，到期前价格大概会落在这个区间附近。

如果我想卖 Put，就会重点看：

```text
我选择的行权价，是否低于 Expected Range 下沿？
```

比如下沿是 82.03。

| Put 行权价位置 | 风险理解 |
|---|---|
| 高于 Expected Range 下沿 | 比较激进，更容易被打到 |
| 接近 Expected Range 下沿 | 中性偏进取 |
| 明显低于 Expected Range 下沿 | 更保守，安全垫更大 |

这就是为什么 Expected Move 很重要。

它不是预测价格一定会到哪里，而是帮我判断：

> 当前想卖的 Put，到底离市场预期波动范围有多远？

---

## 第三步：用 AI 判断“高权利金是机会还是陷阱”

Barchart 只能告诉我：

```text
期权贵不贵？
市场预期波动大不大？
```

但它不能直接告诉我：

```text
今天该不该卖 Put？
这份高权利金是不是值得拿？
```

所以我会把这些数据交给 AI，再结合今日市场分析做判断。

AI 主要看这几个维度：

| 维度 | 问题 |
|---|---|
| 趋势结构 | QQQ / QLD 是否仍在关键均线上方？ |
| 市场风险 | VIX、10Y、美债、美元是否同时压制风险资产？ |
| 波动溢价 | IV 是否明显高于 HV？IV Percentile 是否偏高？ |
| 事件风险 | 是否临近 CPI、FOMC、财报、重大地缘事件？ |
| 标的位置 | 当前价格是在支撑附近，还是已经破位？ |
| BTC / MSTR 联动 | 如果卖 MSTR，需要看 BTC 是否主动走强或破位 |
| 权利金补偿 | 收到的 premium 是否足够覆盖接货风险？ |

AI 的作用不是替我下单，而是帮我把这些信息变成一个结构化判断。

---

## 第四步：用“几个勾”判断今天适不适合卖 Put

我会把每天的 Sell Put 环境拆成几个勾。

| 判断项 | ✅ 有利于卖 Put | ⚠️ 需要谨慎 | ❌ 不适合 |
|---|---|---|---|
| 趋势结构 | 标的仍在关键支撑上方 | 横盘震荡 | 明显破位 |
| IV / HV | IV 高于 HV，权利金变肥 | 差距不大 | IV 低，权利金瘦 |
| IV Percentile | 60% 以上 | 30% - 60% | 低于 30% |
| Expected Move | 可选到足够远的行权价 | 安全垫一般 | 行权价太贴近当前价格 |
| 市场环境 | 没有系统性风险扩散 | 风险混杂 | VIX、10Y、美元同时压制 |
| 事件风险 | 没有重大隔夜事件 | 有普通数据 | 有重大财报 / CPI / Fed |
| 标的意愿 | 愿意接货 | 可以小仓接货 | 完全不想接货 |

简单理解：

```text
5个以上 ✅：可以重点看
3-4个 ✅：只适合小仓或远OTM
2个以下 ✅：不急，等待更好机会
```

---

## 第五步：在期权链里选择合适风险比例

确认市场环境之后，才进入期权链选具体合约。

我会看四个核心指标：

## 1. Delta

Delta 用来大致衡量被打到的概率。

| Delta 区间 | 风格 | 适合情况 |
|---|---|---|
| 0.10 - 0.15 | 防守型 | 市场风险较高，只想远 OTM 收小利息 |
| 0.15 - 0.25 | 平衡型 | 常用区间，权利金和安全垫比较平衡 |
| 0.25 - 0.35 | 进取型 | 看好支撑，愿意接货 |
| 0.35 以上 | 偏贴价 | 不适合无脑卖，容易被打到 |

我的常用思路是：

```text
市场不稳 → 选 0.10 - 0.15 Delta
市场正常 → 选 0.15 - 0.25 Delta
特别愿意接货 → 才考虑更高 Delta
```

## 2. 安全垫

```text
安全垫 = (当前价格 - 行权价) / 当前价格
```

安全垫越大，被打到的概率越低，但权利金也会少。

我会把安全垫和 Expected Move 一起看：

```text
如果安全垫 > Expected Move，说明行权价在市场预期波动区间外，更保守。
如果安全垫 < Expected Move，说明行权价仍在市场预期波动范围内，风险更高。
```

## 3. 年化收益

```text
年化收益 ≈ 权利金 / 行权价 / 到期天数 × 365
```

年化收益只是参考，不是越高越好。

我更关心的是：

> **在足够安全垫下，还有没有合理年化？**

## 4. 接货价格

Sell Put 最后一定要回到一个问题：

> 如果真的被指派，我愿不愿意用这个价格买入？

如果答案是否定的，再高的年化也不应该卖。

---

## 第六步：形成最终下单区间

AI 最后应该输出的不是一个单一答案，而是一个区间建议。

例如：

| 风格 | 适合情况 | 选择方式 |
|---|---|---|
| 保守 | 市场有风险，但权利金变肥 | 低 Delta、远 OTM、低仓位 |
| 平衡 | 市场结构正常，IV 有溢价 | 0.15 - 0.25 Delta，安全垫够用 |
| 进取 | 明确愿意接货，支撑强 | 更靠近现价，但仓位要小 |
| 不做 | 系统性风险太高 | 等下一次机会 |

最终我不是问：

```text
哪个 Put 年化最高？
```

而是问：

```text
哪个 Put 在当前市场下，风险收益比最合理？
```

---

## 我的 AI 筛选 Put 标准流程

```text
第一步：打开 Barchart 标的 overview 页面
第二步：记录 IV、HV、IV Percentile、IV Rank、Expected Move、Expected Range
第三步：判断是否存在恐慌溢价
第四步：结合今日市场趋势，判断今天适不适合卖 Put
第五步：进入期权链，按 Delta、安全垫、年化收益筛选
第六步：检查是否愿意接货
第七步：选择合适行权价，小仓位下单
第八步：开仓后跟踪 Delta、价格位置和市场结构变化
```

---

## 评级总表

| 评级 | 含义 | 行动 |
|---|---|---|
| ❌ | 不做 | 结构不支持，放弃新仓 |
| ⚠️ | 观察 | 只看不动，或极小仓、极远 OTM |
| ✅ | 可以做 | 小仓、远 OTM、等回踩 |
| ✅✅ | 主策略 | 可以作为当日主策略，但仍需分批和安全垫 |
| ✅✅✅ | 最佳窗口 | 趋势、位置、波动率、宏观都配合，可偏积极 |

---

## 评级看什么？

我主要看五个维度：

1. **趋势结构**：QQQ / SPY / BTC 是否仍在关键均线或结构支撑上方。
2. **波动率价格**：VIX、IV、权利金是否足够补偿风险。
3. **宏观压力**：10Y、美债收益率、美元、油价是否压制风险资产。
4. **资金流向**：资金是在进攻科技/宽基，还是撤向防御/现金/能源冲击。
5. **事件风险**：财报、Fed、CPI、就业、地缘风险是否可能造成跳空。

---

## QLD 与 MSTR 的评级差异

| 标的 | 特点 | 评级原则 |
|---|---|---|
| QLD | 跟随纳指，波动高但结构较清晰 | 可作为主策略，但必须看 QQQ / 10Y / VIX |
| MSTR | BTC beta + 公司自身波动 + 财务结构 | 不能只看权利金，必须看 BTC 是否主动走强 |

一句话：

> **QLD 是主战场，MSTR 是加速器。**

---

## 为什么很少给 ✅✅✅？

✅✅✅ 必须同时满足：

- 市场回踩支撑，而不是高位追涨；
- VIX / IV 抬升，权利金变肥；
- 油价、10Y、美元没有同时压制；
- 主线资产有承接，科技或宽基没有破位；
- 没有重大隔夜事件风险。

多数时候，市场只适合 ✅ 或 ✅✅。

---

## 我的执行纪律

- 评级越高，不代表可以满仓；
- 评级越低，不代表一定会跌；
- 卖 Put 的核心不是预测，而是**拿足够的权利金，承担可控的接货风险**；
- 任何时候，仓位管理优先于方向判断。
