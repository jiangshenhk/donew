## 1）资金流向异动 | 今日最重要变化

**最重要变化：风险资产“止跌但没有资金追”，属于被动企稳而非主动修复。**

- **美股收盘跌幅收敛**：盘中下跌后尾盘企稳，说明抛压释放但没有强买盘。
- **油价高位震荡而非继续冲高**：WTI维持在105-106区间 → 通胀压力未进一步恶化。
- **10Y未继续上冲**：维持在高位但未创新高 → 利率压力边际缓和。
- **美元涨势放缓**：DXY高位横盘 → 流动性不再进一步收紧。
- **BTC未破位但无反弹**：维持在78-79k区间 → 风险偏好没有恢复。

**结论：卖压释放，但没有新资金接力 → 典型“弱修复窗口”。**

## 8）落到卖 Put 策略

### QLD

**结论：允许试探，但必须极保守。**

| 条件 | 动作 |
|---|---|
| 10Y回落 + NVDA转强 + QQQ强于SPY | 可卖第一笔远OTM |
| 只看到指数止跌，但利率未回落 | 只允许挂很远OTM低价单 |
| VIX继续上行 | 不卖 |
| WTI重新上冲 | 不卖或降仓 |

执行口径：

- 仓位：≤25%
- Delta：0.08–0.12优先，最高不超过0.15
- Strike：远OTM
- 节奏：只做第一笔，不连续加仓
- 目标：不是赚满权利金，而是测试市场承接
