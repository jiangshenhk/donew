## 1）资金流向异动｜今晚最重要变化

今晚最重要的变化不是“美股涨”，而是**油价风险溢价被快速挤出**。

美股期货盘前明显走强，道指、标普500、纳指100期货均上涨，主线来自两个方向：一是美伊接近停火/谈判备忘录的预期，二是 AMD 财测强化 AI 需求。

更关键的是跨资产：Brent 一度大跌约 10% 以上，WTI 也大幅回落，说明资金正在撤出“战争导致能源供应中断”的交易。Bitcoin 小涨至 82,000 美元上方，10年美债收益率回落至约 4.35%，VIX期货下跌。

**资金正在从：**

- 能源冲击交易：USO / XLE / 油气股
- 美元避险：DXY走弱
- 利率上行压力：美债收益率回落
- 纯防御仓位：部分现金、防御股

**流向：**

- AI / 半导体：AMD、SMCI、ARM、NVDA
- 纳指成长股：QQQ / QLD
- 全球风险资产：欧洲股市、亚洲科技股
- 黄金仍有承接：说明并非纯粹 risk-on，而是“风险资产反弹 + 尾部风险保险仍在”

## 3）资金流向与资产联动

### 当前联动关系

| 组合 | 信号 |
|---|---|
| 油价大跌 + 纳指上涨 | 地缘风险溢价释放，成长股估值压力下降 |
| 美债收益率下行 + 科技上涨 | 利率端支持成长股扩张 |
| 美元走弱 + 风险资产上涨 | 避险需求下降，全球资金重新承担风险 |
| 黄金上涨 + 股票上涨 | 市场不是完全乐观，仍在买尾部保护 |
| BTC小涨 + MSTR财报后波动 | Crypto 风险偏好恢复不够强，MSTR不应重仓卖 Put |

### 谁在说真话？

**油价和美债在说真话。**

油价暴跌说明战争溢价正在被真实定价下修；美债收益率下行说明市场认为油价回落会缓解通胀压力。股票上涨是结果，不是原因。

黄金上涨则提醒：这不是“风险彻底解除”，而是“风险下降但还没消失”。

## 8）落到卖 Put 策略

### QLD

**结论：可以卖，但只做远 OTM，分批，不追近端高权利金。**

| 项目 | 执行 |
|---|---|
| 是否卖 Put | ✅ 可以 |
| 仓位 | 中低仓，最多正常仓位的 30%-50% |
| 执行方式 | 高开后等 30-60 分钟，看 QQQ 是否承接 |
| 行权价 | 远 OTM，优先选择下方关键支撑外侧 |
| 到期 | 短周期可以，但不要压太近；避免开盘情绪最高点一次性打满 |
| 禁忌 | 不卖 ATM，不追高卖满仓，不在第一根冲高K线盲目成交 |

**今晚 QLD 行动清单：**

- [ ] QQQ高开后不快速回补缺口  
- [ ] NVDA / AMD / SMCI 至少两个保持红盘  
- [ ] 10Y不重新上破 4.40%  
- [ ] VIX不反抽  
- [ ] 满足后才卖第一笔远 OTM Put  
- [ ] 若开盘冲高回落，暂停第二笔
