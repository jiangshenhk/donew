## 1）资金流向异动｜今晚最重要变化

今晚最重要的变化不是单纯股指高开，而是：

**油价继续下跌，释放通胀与利率压力；AI/半导体继续吸资金；黄金也同步上涨。**

这说明资金不是单一 risk-on，而是两条线同时走：

| 资金流出 | 资金流入 |
|---|---|
| 原油、能源风险溢价 | AI、半导体、成长股 |
| 中东战争溢价 | 黄金、部分避险资产 |
| 高油价通胀交易 | 久期资产、科技估值 |

美股期货盘前接近纪录高位，核心驱动是油价下跌、美国与伊朗和平协议预期、AI 交易和强企业盈利。黄金同时上涨至两周高位，受美元走弱、油价下跌和收益率回落支撑。

**市场现在不是防守盘。  
但也不是低位修复盘。  
这是高位继续挤空 + 资金追 AI 的盘。**

### 关键联动

### 谁在说真话？

**今晚说真话的是油价和半导体。**

油价继续跌，说明中东风险溢价正在被撤掉；NVDA、AI 链继续强，说明资金仍愿意进攻成长资产。  
但 BTC 没有确认，黄金也在涨，所以市场不是“全资产狂欢”，而是：

> **股市进攻，宏观降温，避险仍留一手。**

## 8）落到卖 Put 策略

# QLD

结论：**可以卖，但只卖第一笔，远 OTM，分批。**

QLD 当前盘前强，AI 和纳指趋势都支持卖 Put。但因为 QLD 已经高位大涨，不能把它当成低位恐慌盘来卖。

| 项目 | 建议 |
|---|---|
| 是否卖 Put | **可以** |
| 激进程度 | **中等偏保守** |
| 仓位 | 计划总仓位的 **25%–35%** |
| 到期 | **21–35 DTE** |
| Delta | **0.12–0.18** |
| Strike | 现价下方 **8%–12%** |
| 执行 | 不开盘立刻追，等 30–60 分钟确认 |

QLD 今晚动作：

- [ ] QQQ 高开后不快速回补缺口  
- [ ] NVDA / 半导体继续强  
- [ ] US 10Y 不重新上冲 4.35%–4.40%  
- [ ] Brent 不重新站回 100 上方并急拉  
- [ ] VIX 不盘中快速反抽  

满足 3 条以上：**卖第一笔远 OTM QLD Put。**  
满足不足 3 条：**继续空仓，等尾盘。**

---

# MSTR

结论：**不做主仓，不因权利金高而追。**

MSTR 的问题不是 IV，而是确认度不够。BTC 盘前偏弱，MSTR 也没有明显跟随科技股上攻。今晚 MSTR 不是最优卖 Put 标的。

| 项目 | 建议 |
|---|---|
| 是否卖 Put | **暂不主动卖** |
| 激进程度 | **低** |
| 仓位 | 若做，不超过 QLD 仓位的 **10%–20%** |
| 到期 | 21–35 DTE |
| Delta | **0.08–0.12** |
| Strike | 必须极远 OTM |
| 条件 | BTC 重新站回 82k–85k 且 MSTR 不冲高回落 |

MSTR 今晚动作：

- [ ] BTC 站回 82k  
- [ ] BTC 强于 QQQ  
- [ ] MSTR 开盘后不弱于纳指  
- [ ] MSTR 没有冲高回落  
- [ ] Put spread 不离谱  

全部满足才考虑小仓。  
否则：**不做。**
