# 风险评分说明

> 市场日报中的风险评分用于回答一个问题：今天卖 Put 是偏进攻、偏防守，还是应该等待？

---

## 评分区间

| 分数 | 市场状态 | 策略倾向 |
|---|---|---|
| 0–3 | 低风险 / 顺风 | 可以正常执行，但仍要控制仓位 |
| 3–5 | 中低风险 | 可以卖 Put，优先远 OTM |
| 5–7 | 中高风险 | 只适合分批、小仓、远 OTM |
| 7–8.5 | 高风险 | 观察为主，极远 OTM 或不做 |
| 8.5–10 | 极端风险 | 不开新仓，先保命 |

---

## 评分看什么？

我主要看六个变量：

| 变量 | 影响 |
|---|---|
| VIX / IV | 决定卖 Put 权利金是否合理 |
| 10Y 美债收益率 | 决定科技估值压力 |
| 油价 | 决定通胀和地缘风险 |
| 美元 DXY | 决定全球流动性压力 |
| 股市广度 | 判断是否只有少数权重股在撑指数 |
| BTC / 黄金 | 判断风险偏好和尾部对冲是否一致 |

---

## 分数不是方向预测

风险评分不是预测市场一定涨或一定跌。

它只是告诉我：

- 这时候卖 Put 是否值得；
- 安全垫要留多远；
- 仓位要放多少；
- 是做 QLD，还是做 SPY；
- MSTR 是否应该降级。

---

## 例子

| 情况 | 评分倾向 |
|---|---|
| QQQ上涨，VIX下降，10Y回落，油价稳定 | 5–6 |
| QQQ上涨，但油价暴涨、10Y上冲、黄金也涨 | 6.5–7.5 |
| QQQ下跌，VIX暴涨，油价与美元同时上行 | 8+ |
| 市场回踩支撑，VIX抬升但结构未破 | 4.5–6 |

---

## 我的使用方式

- **风险分数越低**：可以更正常地执行卖 Put。
- **风险分数越高**：strike 越远，仓位越小。
- **超过 7 分**：默认进入防守模式。
- **超过 8.5 分**：优先不做，等待市场重新定价。

一句话：

> **风险评分决定仓位和距离，不决定我有没有观点。**
