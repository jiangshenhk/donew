# QLD 卖 Put 操作框架

> QLD 是 2 倍纳指 ETF，适合用来做“纳指主线 + 波动率收租”的核心标的。但它不是无风险资产，必须用安全垫、仓位和节奏控制。

---

## 一句话框架

> **QLD 卖 Put 的核心：只在纳指趋势未破时，利用回调和波动率抬升，卖远 OTM Put，赚时间价值和恐慌溢价。**

---

## 什么时候适合卖？

| 条件 | 判断 |
|---|---|
| QQQ / Nasdaq 主趋势未破 | 必要条件 |
| VIX / IV 有一定抬升 | 权利金更有性价比 |
| 10Y 没有快速上冲 | 科技估值压力可控 |
| 油价没有重新暴涨 | 通胀冲击可控 |
| 市场不是重大事件前夜 | 避免隔夜跳空 |

---

## 什么时候不适合卖？

- QQQ 高开急拉，没有回踩；
- 10Y 快速上冲；
- 油价暴涨并带动通胀预期；
- VIX 太低，权利金不够；
- 即将公布 CPI、Fed、重大财报；
- QQQ / NVDA / 半导体同时破位。

---

## Strike 选择

| 市场状态 | Strike 选择 |
|---|---|
| ✅✅✅ 最佳窗口 | 可中远 OTM，但仍不贴价 |
| ✅✅ 主策略窗口 | 远 OTM / 中远 OTM |
| ✅ 普通窗口 | 远 OTM，小仓 |
| ⚠️ 观察窗口 | 极远 OTM 或不做 |
| ❌ 风险窗口 | 不做 |

---

## OTM、Delta 与概率理解

这里要分清楚两个概念：

> **✅ 的数量 = 当天适不适合卖 Put 的环境评级。**  
> **远 OTM / 中远 OTM = 具体 strike 离现价有多远，通常用 Delta 或大致到期价内概率来近似。**

### OTM 分层

| 说法 | 大致 Delta | 大致被行权 / 到期价内概率 | 使用场景 |
|---|---:|---:|---|
| 极远 OTM | 0.05–0.08 | 5%–8%左右 | 防守仓、观察仓 |
| 远 OTM | 0.10–0.15 | 10%–15%左右 | 正常安全卖 Put |
| 中远 OTM | 0.15–0.20 | 15%–20%左右 | 主策略窗口才考虑 |
| 中等 OTM | 0.20–0.25 | 20%–25%左右 | 已经偏进攻 |
| 近 OTM / 贴价 | 0.30以上 | 30%以上 | 不适合作为长期收租主策略 |

> Delta 不是精确概率，但实战中可以把 Put Delta 粗略理解为“到期进入价内的概率近似值”。

### 评级与 Delta 对应

| 市场评级 | 可以卖吗 | 建议 Delta | Strike 选择 |
|---|---|---:|---|
| ❌ | 不做 | 不看 | 不开新仓 |
| ⚠️ | 观察 | 0.05–0.10 | 极远 OTM 或不做 |
| ✅ | 可以小仓 | 0.08–0.12 | 远 OTM |
| ✅✅ | 主策略窗口 | 0.10–0.15，最多 0.18 | 远 OTM / 中远 OTM |
| ✅✅✅ | 最佳窗口 | 0.15–0.20 | 可中远 OTM，但仍不贴价 |

重点：

> **✅✅ 不代表一定卖 20% 概率的 Put。**  
> 它只是说明“市场环境允许你卖”，但如果指数已经高位，仍然应该偏远。

### QLD 实战用法

| 市场位置 | QLD Delta 倾向 |
|---|---:|
| 高位上涨、刚拉升后 | 0.10–0.12 |
| 回踩不破、VIX稍微抬升 | 0.12–0.15 |
| 明显恐慌回踩但结构未破 | 0.15–0.18 |
| 真正最佳窗口 | 0.18–0.20 |

如果日报给出 **⚠️ / ✅**，QLD 只看 **0.10–0.15**，优先 **0.10–0.12**。  
如果日报给出 **✅✅**，也应先看市场位置：高位用低 Delta，回踩才提高 Delta。

### 最重要的一句话

> **远 OTM 可以理解为 10%左右概率，中远 OTM 可以理解为 15%–20%左右概率；但高位市场用低 Delta，回踩市场才可以提高 Delta。**

---

## 仓位原则

| 情况 | 仓位 |
|---|---|
| 高位上涨 | 小到中小仓 |
| 回踩支撑 | 可分批提高 |
| VIX 抬升但结构未破 | 可以分批 |
| 油价 / 10Y 同时压制 | 自动减半 |
| 重大事件前 | 降仓或不做 |

---

## 执行节奏

1. 开盘先观察 30–60 分钟；
2. 看 QQQ 是否有承接；
3. 看 10Y、VIX、油价是否反向冲击；
4. 第一笔只做小仓；
5. 若盘中回踩不破，再补第二笔；
6. 不在情绪最热的时候一次打满。

---

## 平仓与滚动

| 情况 | 动作 |
|---|---|
| 盈利 50%–70% | 可提前买回 |
| Delta 快速升高 | 评估是否滚动 |
| QQQ 结构破位 | 减仓或止损 |
| 到期前 21 天左右 | 检查是否换月 |
| 权利金快速衰减 | 不贪最后一点 |

---

## 关键提醒

> **QLD 是主战场，但不是提款机。**

真正的优势来自：

- 等回调；
- 卖远 OTM；
- 分批；
- 不贴价；
- 有现金接货；
- 不在风险事件前硬卖。
