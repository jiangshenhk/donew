# QLD 卖 Put 操作框架

> QLD 是 2 倍纳指 ETF，适合用来做“纳指主线 + 波动率收租”的核心标的。但它不是无风险资产，必须用安全垫、仓位和节奏控制。

---

## 一句话框架

> **QLD 卖 Put 的核心：只在纳指趋势未破时，利用回调和波动率抬升，卖远 OTM Put，赚时间价值和恐慌溢价。**

---

## 什么时候适合卖？

| 条件 | 判断 |
|---|---|
| QQQ / Nasdaq 主趋势未破 | 必要条件 |
| VIX / IV 有一定抬升 | 权利金更有性价比 |
| 10Y 没有快速上冲 | 科技估值压力可控 |
| 油价没有重新暴涨 | 通胀冲击可控 |
| 市场不是重大事件前夜 | 避免隔夜跳空 |

---

## 什么时候不适合卖？

- QQQ 高开急拉，没有回踩；
- 10Y 快速上冲；
- 油价暴涨并带动通胀预期；
- VIX 太低，权利金不够；
- 即将公布 CPI、Fed、重大财报；
- QQQ / NVDA / 半导体同时破位。

---

## Strike 选择

| 市场状态 | Strike 选择 |
|---|---|
| ✅✅✅ 最佳窗口 | 可中远 OTM，但仍不贴价 |
| ✅✅ 主策略窗口 | 远 OTM / 中远 OTM |
| ✅ 普通窗口 | 远 OTM，小仓 |
| ⚠️ 观察窗口 | 极远 OTM 或不做 |
| ❌ 风险窗口 | 不做 |

---

## 仓位原则

| 情况 | 仓位 |
|---|---|
| 高位上涨 | 小到中小仓 |
| 回踩支撑 | 可分批提高 |
| VIX 抬升但结构未破 | 可以分批 |
| 油价 / 10Y 同时压制 | 自动减半 |
| 重大事件前 | 降仓或不做 |

---

## 执行节奏

1. 开盘先观察 30–60 分钟；
2. 看 QQQ 是否有承接；
3. 看 10Y、VIX、油价是否反向冲击；
4. 第一笔只做小仓；
5. 若盘中回踩不破，再补第二笔；
6. 不在情绪最热的时候一次打满。

---

## 平仓与滚动

| 情况 | 动作 |
|---|---|
| 盈利 50%–70% | 可提前买回 |
| Delta 快速升高 | 评估是否滚动 |
| QQQ 结构破位 | 减仓或止损 |
| 到期前 21 天左右 | 检查是否换月 |
| 权利金快速衰减 | 不贪最后一点 |

---

## 关键提醒

> **QLD 是主战场，但不是提款机。**

真正的优势来自：

- 等回调；
- 卖远 OTM；
- 分批；
- 不贴价；
- 有现金接货；
- 不在风险事件前硬卖。
