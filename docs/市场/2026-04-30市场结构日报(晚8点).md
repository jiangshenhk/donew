## 1）资金流向异动｜今晚最重要变化

今晚最重要的变化是：

**科技财报重新托住盘前情绪，但油价仍然是压在市场头上的核心变量。**

Reuters 报道，美股盘前原本受油价冲击压制，但后续因企业盈利乐观情绪转强，道指、标普、纳指期货都转向上涨；Alphabet 因云业务强劲盘前上涨，Amazon 也因云收入好于预期走高，但 Meta 和 Microsoft 因资本开支担忧承压。

油价这边不是小扰动。Brent 一度冲到 **126.41**，后回落到 **113.90**；WTI 一度到 **110.93**，后结算在 **104.92**。这说明市场虽然愿意买科技财报，但仍必须重新定价中东供应风险和通胀压力。

资金流一句话：

| 流出 / 降温 | 流入 / 承接 | 本质 |
|---|---|---|
| 高估值但财报不够强的科技 | 财报兑现的大科技 | 资金开始挑盈利质量 |
| 远端成长估值 | 能源 / 防御 / 短久期 | 油价推高通胀预期 |
| 黄金部分资金 | 美元 / 利率资产 | 不是纯避险，是利率压力 |
| 高 beta | 稳定盈利资产 | 风险偏好没有全面扩散 |

| 科技财报强 + 纳指期货反弹 | 有承接 | QQQ / QLD 不是不能做 |

| Meta / Microsoft 承压，Alphabet / Amazon 强 | 财报分化 | 市场只奖励兑现，不奖励故事 |

### 谁在说真话？

**今晚说真话的是：油价 + 科技财报分化。**

股指期货反弹说明市场还愿意买盈利；但油价高位说明通胀和供应风险没有解除。  
所以今晚不是“全面风险偏好恢复”，而是：

> **盈利好的科技可以托指数，但油价限制估值扩张。**
