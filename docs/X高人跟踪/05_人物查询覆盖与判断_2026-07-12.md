> 本文件同时记录两件事：一是人物本身是否值得跟踪；二是公开搜索引擎实际能查到多少原始资料。两者必须分开，避免“搜不到=没水平”或“只搜到神预测=胜率很高”的误判。

## 一、本轮查询方法

本轮使用公开搜索引擎与可公开访问页面进行测试，主要组合包括：

- `site:x.com/账号/status + BTC/Bitcoin/MSTR/美股/加密`
- `"@账号" + 关键词`
- `人物名 + 关键词`
- 直接打开 X 个人主页
- 查找 TradingView、媒体、YouTube、Telegram、个人网站等同步或二次引用

### 本轮实际情况

- 多数中文和中小型 X 账号，精确账号搜索返回 0 条可确认原帖。
- 部分 X 主页可以打开，但抓取结果为空，无法读取时间线正文。
- 搜索引擎容易收录热门帖、被引用帖和成功预测，普通帖、回复帖、图片帖、失败判断常常缺失。
- 因此本轮只能做“搜索覆盖度测试 + 人物功能定位”，不能把搜索结果数量当成完整发帖数量，也不能据此计算真实胜率。
