# X 上牛人跟踪与交叉验证总控

> 目的：建立一个可复盘、可统计、可交叉验证的投资观点库，而不是简单收藏“大V”。

## 一、核心原则

1. **只记录提前公开的观点**：必须保留原帖链接、发布时间和当时市场价格。
2. **预测必须可验证**：至少包含方向、价格区域、时间窗口、触发条件或失效条件中的三项。
3. **命中与失败都记录**：不能只收集事后截图和成功案例。
4. **观点不等于交易指令**：分析师的宏观判断、技术路径和实际入场点必须分开评价。
5. **多人交叉验证**：单人观点只作为辅助；多人、不同方法出现同方向结论时才提高权重。
6. **按周期评价**：短线、波段和大周期不能混在一起计算准确率。

## 三、统一评价框架

每位人物按以下八项评分，每项 0–10 分：

| 维度 | 评价重点 |
|---|---|
| 原始性 | 是否提前公开，而非行情发生后的解释 |
| 明确度 | 是否给出方向、价位、时间、条件和失效位 |
| 准确率 | 同周期全部可验证观点的命中比例 |
| 盈亏比 | 即使胜率一般，正确时是否能覆盖错误损失 |
| 时效性 | 是否过早、过晚，能否转化为实际操作 |
| 一致性 | 是否频繁同时发布多空观点，或事后删除错误判断 |
| 风控意识 | 是否明确止损、失效条件和仓位风险 |
| 独立价值 | 是否提供市场共识之外的有效信息 |

### 综合评级建议

- **S**：长期可验证，方法稳定，可作为核心辅助信号。
- **A**：有明确优势，适合较高权重参考。
- **B**：有能力，但限制明显；适合作为第二意见。
- **C**：偶有亮点，稳定性不足。
- **D**：以情绪、流量或事后解释为主。
- **观察中**：样本不足，不做结论。

## 四、预测记录标准

每条预测至少记录：

| 字段 | 内容 |
|---|---|
| 发布时间 | 原帖公开时间，统一注明时区 |
| 原帖链接 | X、TradingView、YouTube 或其他原始来源 |
| 标的与周期 | BTC 日线、BTC 周线、MSTR 小时线等 |
| 当时价格 | 发帖时附近的市场价格 |
| 原始观点 | 尽量忠实概括，不擅自强化措辞 |
| 方向 | 看涨 / 看跌 / 震荡 / 条件路径 |
| 目标区域 | 明确价位或区间 |
| 时间窗口 | 小时、数日、数周、季度或周期内 |
| 触发条件 | 例如突破、收复、跌破或指标确认 |
| 失效条件 | 什么情况说明观点错误 |
| 最大有利波动 | 观点发布后最有利方向的幅度 |
| 最大不利波动 | 观点发布后先经历的反向幅度 |
| 最终结果 | 命中 / 部分命中 / 错误 / 无法验证 / 尚未到期 |
| 交易可用性 | 即便最终碰到目标，过程中是否可实际持有 |

## 五、多人交叉验证矩阵

当同一市场出现重要变化时，建立一次横向对照：

| 人物 | 方法类型 | 方向 | 时间周期 | 关键位置 | 失效位 | 可信度 | 是否独立信号 |
|---|---|---|---|---|---|---:|---|
| A | 宏观/流动性 |  |  |  |  |  |  |
| B | 链上数据 |  |  |  |  |  |  |
| C | 技术结构 |  |  |  |  |  |  |
| D | 衍生品/资金流 |  |  |  |  |  |  |

### 共识强度

- **一级：单人信号**——只做观察，不改变主策略。
- **二级：同类两人同向**——提高关注，但可能来源和方法高度相关。
- **三级：不同方法三方同向**——可调整仓位、Delta 或安全垫。
- **四级：多方法同向 + 市场价格确认**——可以作为正式操作信号。

> 不能简单数人头。十个互相转发同一套逻辑的账号，本质上仍然只有一个信息源。

## 六、与你的投资体系结合

### 对 BTC / IBIT 定投

- 大周期高手转空，只影响情绪仓和突破仓，不轻易停止固定定投。
- 多人看空且周线 E90、流动性、链上数据同时恶化，可降低加速投入。
- 多人看多但价格尚未确认，不追高，等待技术触发。

### 对 MSTR / IBIT 卖 Put

- 单人警告：降低目标年化要求，选择更低 Delta。
- 三类信号共振看空：减少张数、扩大安全垫、避开短期限。
- 高度一致看空并有价格确认：暂停新增卖 Put，优先处理已有风险。
- 观点分歧很大：说明市场处于路径不清阶段，更适合小仓位和远虚值。

## 七、维护流程

1. 每发现一位候选人，先建立个人档案，评级为“观察中”。
2. 回溯至少 10–20 条可验证预测，包含正确与错误案例。
3. 按短线、波段、大周期分别统计。
4. 每月更新一次预测结果和评分。
5. 每季度淘汰没有独立价值、长期模糊或只展示成功案例的人物。
