# 方法二：卖Put收利息

> Sell Put 不是单纯赚权利金，而是把“看好但不追高”变成一个有价格、有补偿、有纪律的买入计划。

## 适合谁？

| 类型 | 是否适合 |
|---|---|
| 想用更低价格买入长期看好的资产 | 适合继续研究 |
| 能接受被行权、愿意接货 | 适合继续研究 |
| 只想赚权利金、不想接货 | 不适合 |
| 没有仓位控制和现金管理 | 先不要做 |
| 看不懂期权风险 | 先学习基础 |

## 风险提示

Sell Put 不是低风险套利。标的暴跌、波动率变化、流动性不足、仓位过重、保证金压力都可能导致亏损。本站内容仅用于投资研究与学习，不构成任何投资建议。
