# 心安复利

> 复利不是短期暴富，而是让时间、纪律和优质资产长期站在自己这一边。

这个频道是本站的底层价值观。它负责回答一个更根本的问题：普通人为什么要选择长期复利，而不是一直追逐短期暴富？

## 三层关系

```text
财富观 = 知道自己为什么投资
复利方法 = 知道长期怎么执行
佛学心态 = 知道波动中如何保持清醒
```

这三层不是三件分散的事。财富观决定你追求什么，复利方法决定你怎么做，佛学心态决定你能不能在波动、贪婪和恐惧里坚持下来。

```text
不要把暴富当成必然
不要把一次涨跌当成命运
不要在焦虑中做决定
不要把投资变成执念
```
