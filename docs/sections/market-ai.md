# AI市场趋势判断

> 每日市场分析不是预测明天涨跌，而是判断当前市场环境更适合进攻、观望，还是防守。

这个频道是本站的高频更新区，核心任务是把宏观利率、油价、美元、VIX、AI主线、半导体、加密链和高 beta 资产放在同一个框架里，判断今天是否适合执行 ETF 定投、Sell Put 或继续观望。

## 核心观察框架

| 模块 | 主要看什么 | 用来判断什么 |
|---|---|---|
| 利率 | 美债10Y、实际利率 | 成长股估值压力 |
| 油价 | Brent、能源风险 | 通胀和风险偏好 |
| 美元 | DXY | 全球风险资产流动性 |
| VIX | 恐慌指数 | 是否进入防守状态 |
| AI / 半导体 | NVDA、芯片链、AI硬件 | 进攻主线是否还在 |
| 加密链 | BTC、MSTR 等 | 高 beta 风险是否扩散 |
| 中概 / 券商 | FUTU、TIGR 等 | 政策型风险是否释放 |

## 判断结果怎么用？

```text
风险低 + 主线清晰 = 可以考虑小仓分批
风险中等 + 主线分化 = 只看远 OTM，不追价
风险高 + 多因子压制 = 暂停高 beta，先观察
```

市场日报不是交易指令，而是执行前的环境检查。真正下单前，还要结合仓位、现金、标的价格、隐含波动率和自己能承受的最大回撤。
