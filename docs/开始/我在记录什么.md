# 站点说明：我在记录什么

---

## **这个网站，记录的是一套围绕“长期复利”展开的个人投资知识体系。**

这里不是喊单站，也不是追热点的情绪站。  
我希望把平时零散看到的市场信息、投资工具、交易方法与长期认知，逐步整理成一套可以长期复用的知识结构。

?> **一句话来说：这里记录的，不只是怎么买，更是为什么买、什么时候买、买了之后如何坚持。**

---

## **我为什么要做这个站？**

很多人接触投资，最先接触到的是“结论”，比如：

- [ ] 现在是不是该买？
- [ ] 哪个ETF更好？
- [ ] 这个点位可不可以定投？
- [ ] Sell Put是不是稳赚？
- [ ] 美债、黄金、BTC、纳指之间到底怎么看？

但真正决定长期结果的，往往不是某一次结论，而是你背后有没有一套稳定的方法。

所以，我想把平时的思考分成几类，长期整理下来：

> **第一类：长期复利框架**  
> 比如复利、定投、仓位纪律、长期持有这些底层逻辑。

> **第二类：策略工具**  
> 比如 Sell Put、ETF、港美股、美债、黄金、比特币等工具和资产。

> **第三类：市场结构判断**  
> 比如每天市场到底在交易什么，利率、美债、黄金、BTC、纳指之间是如何联动的。

> **第四类：实操资料**  
> 比如券商、开户、银行卡、入金路径与常见问题。

---

## **我理解的“复利”，不只是一个数学公式。**

很多人提到复利，想到的只是：

?> 复利终值 = 本金 × (1 + 收益率)^时间

这个公式当然重要，但在现实投资里，真正能让复利发生的，并不只是公式本身，而是下面这些东西：

- [x] 持续投入的能力
- [x] 对波动的承受能力
- [x] 对规则的执行能力
- [x] 对情绪的克制能力
- [x] 在不同阶段选择合适工具的能力

所以在我看来，<b><font color="blue">复利不是单纯“买入并等待”，而是规则、节奏、耐心与长期纪律的共同结果。</font></b>

## **这个网站主要会写什么？**

### **第一部分：长期复利框架**
这部分会重点讲：

- 什么是复利
- 为什么时间比预测更重要
- 为什么长期持有能放大收益
- 定投的底层逻辑
- 仓位控制与再平衡

### **第二部分：Sell Put 策略**
这部分会重点讲：

- 什么是 Sell Put
- 为什么 Sell Put 可以服务长期复利
- 行权价怎么选
- 安全垫怎么计算
- 年化收益怎么看
- 什么情况下提前买回

### **第三部分：资产与市场**
这部分会重点讲：

- ETF 的作用
- 港美股的基础工具
- 美债在组合里的意义
- 黄金的防守属性
- 比特币与高波动资产的角色
- 不同资产之间的联动关系

### **第四部分：市场结构观察**
这部分会重点讲：

- 今天市场在交易什么
- 利率是否在主导市场
- 风险偏好是在修复还是收缩
- 黄金、BTC、纳指、美债谁更强
- 当前环境更适合定投、观望，还是 Sell Put

---

## **我希望这个站，解决什么问题？**

> 不是帮人“猜明天涨跌”，而是帮自己建立一个长期可复用的判断系统。

因为短期结论总会变，但方法会留下来。

当一个人真正建立起自己的投资框架后，他面对市场时会更稳定：

- 市场大涨时，不容易盲目追高；
- 市场大跌时，不容易情绪崩溃；
- 市场震荡时，也知道该观察什么、等待什么。

这就是我做这个站的真正原因。
