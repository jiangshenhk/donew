# 新手阅读顺序

---

## **如果你是第一次来到这个网站，建议不要一上来就看具体标的和每日判断。**

因为市场上的“结论”每天都在变，  
但一个人真正能长期拿得住、看得懂、用得上的，是背后的方法。

?> **所以更好的阅读方式，不是先找答案，而是先建立框架。**

---

## **第一步：先理解什么是复利。**

建议先看：

- [x] 什么是复利
- [x] 站点说明：我在记录什么

因为这是整个网站最底层的起点。

你需要先明白：

- 为什么投资不能只看短期涨跌；
- 为什么时间、收益率和本金会共同决定长期结果；
- 为什么长期投资最难的不是“知道”，而是“坚持”。

如果这一层没想清楚，后面的定投、Sell Put、ETF、美债，都会变成零散知识，而不是系统。

---

## **第二步：再理解长期定投的角色。**

建议重点看：

- [x] 定投的底层逻辑
- [x] ETF 定投方法
- [x] 为什么时间比择时更重要
- [x] 资产配置与仓位纪律

因为对大多数投资者来说，长期底仓的建立，永远比短期技巧更重要。

?> **先学会慢慢积累，再去追求效率；先学会长期持有，再去讨论策略增强。**

这一部分的目标，是帮你建立一个核心认知：

> **复利的基础，不是聪明，而是持续。**

---

## **第三步：然后再看什么是 Sell Put。**

建议重点看：

- [x] 什么是 Sell Put
- [x] 为什么我把定投和 Sell Put 放在一起
- [x] 行权价怎么选
- [x] 安全垫怎么计算
- [x] 年化收益怎么看

很多人一看到 Sell Put，就只关心：

- 权利金高不高；
- 年化高不高；
- 能不能快速赚钱。

但真正重要的是：

- 你愿不愿意接货；
- 这个价格你是不是真的想买；
- 你有没有能力承受波动；
- 这个策略是不是在服务你的长期目标。

所以，Sell Put 不适合放在最前面学，  
而更适合放在你已经理解“长期复利”和“底仓逻辑”之后再看。

---

## **第四步：再去理解资产本身。**

建议重点看：

- [x] ETF 观察
- [x] 港美股基础
- [x] 美债投资指南
- [x] 黄金与避险资产
- [x] 比特币与高波动资产

因为投资工具再多，最后也还是要落到资产本身。

不同资产的角色并不一样：

> **ETF 更适合作为长期配置工具；**  
> **美债更偏防守和利率观察；**  
> **黄金偏避险和抗不确定性；**  
> **比特币与高波动资产更需要仓位边界。**

如果不理解这些差异，就容易把所有资产都用同一种方式去看，结果自然容易出错。

---

## **第五步：最后再看“每日市场判断”。**

建议重点看：

- [x] 每日市场判断
- [x] 周度复盘
- [x] 利率与美债跟踪
- [x] 黄金 / BTC / 纳指联动

这是因为：

### **每日判断更像“应用题”，不是“基础题”。**

如果你前面的框架已经有了，  
那么你再看每天的市场记录，就会知道自己该看什么：

- 市场现在在交易利率，还是在交易增长？
- 风险偏好是在修复，还是在退潮？
- 当前适合继续定投，还是更适合等待？
- Sell Put 的安全垫是不是足够？
- 哪类资产更强，哪类资产更脆弱？

但如果前面的基础没打好，  
你看到的就只会是一堆零散结论，很难真正内化成自己的判断。

## **最后想说的是：**

这个网站最适合的打开方式，不是“急着找答案”，而是“慢慢建立框架”。

因为真正能让人长期受益的，不是某一篇文章告诉你“今天该买什么”，  
而是你慢慢形成了自己的判断顺序、节奏和纪律。

*** 当你开始按框架看市场，而不是按情绪看市场时，投资才真正开始变得稳定。***
