## 使用规则

1. 图片统一放在本目录：`docs/assets/xhs/`
2. 文件名必须和上表完全一致。
3. 推荐使用 `.png` 格式。
4. 图片放好后，`docs/sections/etf-dca.md` 页面会自动显示对应卡片图片。
5. 卡片点击后会跳转到对应的小红书播放页。
