# 卖 Put + ABC/KDJ/MACD 综合判断模型

这个模型把两套框架合并：

1. **股票短线动量与历史行为画像模型**：决定这只股票/指数是否适合卖 put。
2. **ABC + K线成交量 + KDJ/MACD 趋势模型**：判断当前趋势处于 A跌、B弹、C跌还是反转修复。

核心目标：

> 先判断当前趋势结构，再判断卖 put 是否有统计优势。  
> 不用历史支撑去对抗当前破位，也不用指标超卖去硬接放量下跌。

---

## 一、总判断顺序

```text
当前K线与成交量
↓
均线结构
↓
KDJ / MACD 动能
↓
ABC阶段
↓
相对强弱
↓
股票历史行为画像
↓
事件 / 季节性 / 估值 / 期权波动
↓
卖 put 结论
```

核心原则：

> 当前动量是第一优先级，历史行为是过滤器，ABC/KDJ/MACD是趋势识别工具。

---

## 二、两个模型的分工

| 模型 | 负责回答 | 不能做什么 |
|:---|:---|:---|
| 短线动量与历史行为画像 | 这只股票/指数历史上是否适合在当前位置卖 put | 不能替代当前价格行为 |
| ABC/KDJ/MACD趋势模型 | 当前是下跌、反弹、修复还是反转 | 不能单独决定卖 put |

正确用法：

> 先用 ABC/KDJ/MACD 判断“现在能不能接”，再用历史行为画像判断“这只标的值不值得接”。

---

## 三、第一层：当前趋势状态

| 状态 | K线成交量 | KDJ | MACD | ABC阶段 | 初步结论 |
|:---|:---|:---|:---|:---|:---|
| 上行延续 | 站上20日线，回踩不破 | 强势但未死叉 | 红柱稳定 | 非A跌 | 可进入下一层 |
| 上涨衰竭 | 长上影、放量滞涨 | 高位死叉 | 红柱缩短 | 可能转A | 降级观察 |
| A段下跌 | 放量大阴，跌破20日线 | 向下死叉 | 绿柱扩大 | A段形成 | 暂停 |
| B段反弹 | 不破A低，反弹修复 | 低位金叉 | 绿柱缩短 | B段 | 观察，不急卖 |
| C段下跌 | 反弹失败，跌破A低 | 二次死叉 | 绿柱扩大 | C段 | 暂停 |
| 反转修复 | 收回20日线/关键位 | 金叉上行 | 绿柱缩短或翻红 | 修复确认 | 可进入卖put评估 |

硬规则：

- A段刚形成，不卖 put。
- C段跌破A低，不卖 put。
- B段反弹如果缩量，不当反转。
- 只有反转修复或强势回踩，才进入卖 put 评估。

## 四、第二层：宏观流动性事件过滤器

在判断 BTC、QQQ、黄金、七姐妹等风险资产时，必须加入宏观流动性事件。

重点事件：

- 日本央行加息或释放鹰派信号
- 美联储FOMC、CPI、PCE、非农
- 美债收益率快速上行
- 美元指数快速走强
- 日元急升或套息交易平仓
- 大型IPO/指数纳入导致资金再平衡
- BTC ETF大额流入或流出

### 日本央行加息与风险资产

日本央行加息对 BTC 和纳指的影响，不只看加息当天，而要看预期交易窗口。

观察经验：

| 时间 | 事件 | 风险资产含义 |
|:---|:---|:---|
| 2024年3月 | 日本央行退出负利率 | 全球流动性预期变化 |
| 2024年7月 | 日本央行继续加息 | 日元套息交易压力上升 |
| 2025年1月 | 日本央行再加息 | 风险资产去杠杆压力 |
| 2025年12月19日 | 日本央行加息至0.75% | 事件当天可能已被市场吸收，需看前期是否已提前下跌 |

交易规则：

> 日本央行加息不是只看当天反应，而要看加息前4-8周风险资产是否已经提前下跌。

判断方式：

| 情况 | 判断 | 卖 put 动作 |
|:---|:---|:---|
| 加息预期升温，BTC/QQQ已开始放量破位 | 预期交易开始 | 暂停 |
| 加息前已从高点跌20%-30% | 可能已提前释放 | 等事件落地后确认 |
| 加息当天风险资产不跌反涨 | 卖压可能已释放 | 观察B段反弹质量 |
| 加息后日元继续走强、美债收益率上行 | 流动性压力未结束 | 暂停 |
| 加息后日元走弱、风险资产收回关键位 | 宏观压力缓和 | 可重新评估低Delta |

注意：

> 不能把“日本加息后历史上常跌”机械当作做空信号。真正有用的是：加息预期是否触发套息交易平仓，以及这个压力是否已经提前反映在K线里。

## 五、第三层：历史行为画像

趋势状态过关后，再看标的自身历史习惯。

| 模块 | 判断问题 | 结论格式 |
|:---|:---|:---|
| 长期均线行为 | 50日/100日/200日线附近是否经常反弹？ | 支撑有效 / 支撑一般 / 容易破位 / 当前不可依赖 |
| 大涨后行为 | 大涨后是延续还是回吐？ | 趋势延续型 / 容易回吐型 / 需要确认型 |
| 大跌后行为 | 大跌后是修复还是连续下杀？ | 修复型 / 连续下杀型 / 等确认型 |
| 事件行为风险 | 财报、IPO、政策、宏观数据是否未落地？ | 低 / 中 / 高 |
| 季节性位置 | 当前月份/季度是否顺风？ | 顺风 / 中性 / 逆风 |
| 估值压力 | 当前估值是否容易被杀？ | 低 / 中 / 高 |
| 期权波动习惯 | IV高后是否容易回落？ | 卖方友好 / 中性 / 卖方不友好 |

原则：

> 历史上经常反弹，只能在当前已经止跌时加分；不能在当前放量破位时当作接盘理由。

## 六、第四层：相对强弱

必须比较相对强弱。

| 标的 | 比较对象 |
|:---|:---|
| 个股 | 行业ETF + QQQ/SPY |
| 科技股 | QQQ / XLK / SMH / SOXX |
| 半导体 | SOXX / SMH |
| 纳指 | QQQ / SPY / VIX或VXN |
| BTC相关 | BTC / MSTR / COIN / BTC ETF资金流 |

判断：

| 相对强弱 | 含义 | 卖 put 动作 |
|:---|:---|:---|
| 强于行业和大盘 | 个股有承接 | 可加分 |
| 跟随行业回调 | 中性 | 看趋势结构 |
| 弱于行业和大盘 | 个股独立走弱 | 降级 |
| 行业和大盘同时破位 | 系统性风险 | 暂停 |

---

## 七、第五层：卖 Put 执行映射

| 综合状态 | 条件 | 动作 |
|:---|:---|:---|
| 正向共振 | 上行延续 + 历史支撑有效 + 事件风险低 | 可等回踩卖，Delta 0.08-0.12 |
| 修复共振 | B段强反弹或反转修复 + 历史大跌后常修复 | 小仓远OTM，Delta 0.05-0.08 |
| 中性共振 | 震荡消化 + 历史区间震荡 | 极低Delta，0.05-0.08 |
| 冲突 | 当前动量弱，但历史支撑好 | 等确认，不主动卖 |
| 负向共振 | A段/C段下跌 + 历史也容易续跌 | 暂停 |
| 事件高风险 | 财报/宏观/IPO/监管未落地 | 暂停或只观察 |

## 八、评分系统

用于减少主观判断。

### 1. 趋势结构分

| 条件 | 分数 |
|:---|---:|
| 价格在20日线上方 | +1 |
| 价格在50日线上方 | +1 |
| 价格在200日线上方 | +1 |
| 收回20日线 | +2 |
| 跌破20日线 | -1 |
| 放量跌破20日线 | -3 |
| 跌破50日线 | -3 |
| 跌破A段低点 | -4 |

### 2. 动能指标分

| 条件 | 分数 |
|:---|---:|
| KDJ低位金叉 | +1 |
| MACD绿柱缩短 | +1 |
| MACD金叉 | +2 |
| KDJ高位死叉 | -1 |
| MACD死叉 | -2 |
| MACD绿柱扩大 | -2 |

### 3. 历史画像分

| 条件 | 分数 |
|:---|---:|
| 长期均线支撑有效 | +1 |
| 大跌后常修复 | +1 |
| IV高后常回落 | +1 |
| 当前处于季节性顺风 | +1 |
| 容易破位续跌 | -2 |
| 大涨后常回吐 | -1 |
| 估值压力高 | -1 |
| 事件风险高 | -3 |

### 4. 宏观流动性分

| 条件 | 分数 |
|:---|---:|
| 日本央行/美联储事件已落地且风险资产收回关键位 | +1 |
| BTC ETF或风险资产资金流恢复 | +1 |
| 加息预期升温但价格仍强于20日线 | 0 |
| 加息预期升温且价格跌破20日线 | -2 |
| 日元急升或套息交易平仓迹象明显 | -3 |
| 事件前4-8周已放量破位 | -3 |

### 5. 结论映射

| 总分 | 结论 |
|:---|:---|
| +5及以上 | 可卖 |
| +2到+4 | 谨慎卖 |
| -1到+1 | 只观察 |
| -2及以下 | 暂停 |

硬性否决：

- 放量跌破A段低点：直接暂停。
- 放量跌破50日线：直接暂停。
- 事件风险高且未落地：不重仓卖 put。
- Put skew急剧扩大且价格破位：暂停。

## 十一、实战决策句

最终结论必须写成一句能执行的话。

示例：

> 当前处于A段下跌，放量跌破20日线，KDJ和MACD同步转弱，虽然长期仍在50日线上方，但当前动量与卖put冲突，结论为暂停。只有当价格不破A低并收回20日线，且MACD绿柱缩短时，才重新考虑低Delta远OTM。

---

## 十二、操盘纪律

1. 不在A段第一根放量大阴线卖 put。
2. 不用KDJ超卖当作卖 put 理由。
3. 不用历史支撑对抗当前破位。
4. B段反弹必须分强弱，弱B不能当反转。
5. C段跌破A低时，不加仓摊平。
6. 事件未落地时，不因为权利金高就重仓。
7. 卖 put 的前提是风险正在收缩，不是风险正在扩张。
8. 当前动量优先，历史习惯辅助，指标只做确认。
9. 宏观事件要看预期交易窗口，不只看事件当天。
10. 如果加息当天不跌反涨，要判断是压力释放，还是反弹诱多。
