# 日线动量评估框架

这份文档不是为了发明一个复杂指标，而是把一个朴素但实用的想法固定下来：

> 一旦趋势形成，只要没有足够强的反向力量，价格在相对短时间内更容易延续原方向，而不是立刻反转。

这套框架的目标，是把这个想法拆成可观察、可记录、可回测的日线研究方法。

---

## 一、核心假设

先把思路写成一句可验证的话：

> 当价格以较强动量冲击前高或前低后，如果回踩或反抽过程中没有出现明显反向力量，则原趋势延续概率更高。

这里面有三层意思：

1. 先看价格是否已经形成方向性运动。
2. 再看价格是否正在冲击关键结构位。
3. 最后看回踩或反抽时，市场有没有给出反向证据。

所以这不是单纯看涨跌幅，也不是单纯看均线，而是同时看：

- 价格推进速度
- 前高前低结构
- 成交量在突破、跌破、回踩、反抽时的表现

一句话概括：

> 结构定方向，动量定强弱，成交量定真假。

---

## 二、什么叫动量

这里的动量，不是一个指标，而是一组现象。

### 1. 价格动量

价格在一定时间内推进得有多快。

常用定义：

- `ROC20 = (Close - Close[-20]) / Close[-20]`
- `ROC60 = (Close - Close[-60]) / Close[-60]`

直观理解：

- `ROC20` 看短波推进
- `ROC60` 看中段趋势

### 2. 结构动量

价格是否持续抬高低点，或者持续压低高点。

多头结构：

- 更高的高点 `Higher High`
- 更高的低点 `Higher Low`

空头结构：

- 更低的高点 `Lower High`
- 更低的低点 `Lower Low`

### 3. 成交量动量

价格运动是否有量能支持。

常用定义：

- `Volume Ratio = Volume / MA(Volume, 20)`

直观理解：

- 突破放量，说明资金愿意追随趋势
- 回踩缩量，说明反向力量有限
- 跌破放量，说明抛压在集中释放
- 反抽缩量，说明修复意愿不足

所以我们这里说的动量，本质上是：

> 动量 = 价格推进 + 结构推进 + 量能配合

---

## 三、什么叫反向事情

如果说趋势有惯性，那么趋势改变也一定要有证据。

这里把反向事情定义为四类。

### 1. 结构破坏

上升趋势中的反向事情：

- 跌破最近关键低点
- 跌回突破位下方并无法收回

下降趋势中的反向事情：

- 突破最近关键高点
- 收回破位线之上并站稳

### 2. 假突破 / 假跌破

这就是 2B 的核心。

- 顶部 2B：突破前高后迅速跌回前高下方
- 底部 2B：跌破前低后迅速收回前低上方

本质：

> 市场表面上完成了突破或跌破，但真实力量并没有跟上。

### 3. 量价背离

例如：

- 创新高但量能不增，甚至缩量
- 跌破关键位但没有持续放量
- 反抽看起来很强，但量能跟不上

含义：

> 价格表面在动，但推动它的力量不够。

### 4. 时间拖延

强趋势通常有一个特点：快。

如果出现以下现象，就要小心惯性衰减：

- 突破后迟迟不能远离突破位
- 跌破后迟迟不能继续下行
- 回踩拖太久，始终没有二次发力

结论：

> 趋势不是因为“出现一点回调”而结束，而是因为出现了足够级别的结构破坏、2B反向确认、量价背离或时间拖延。

## 四、日线评估的四问法

每天看一个标的，只做四个判断。

### 1. 趋势方向是什么

重点观察：

- 20 日结构
- 60 日结构
- 是否站上关键均线
- 是否仍在抬高低点或压低高点

可输出：

- 上行延续
- 上行衰减
- 震荡整理
- 下行延续
- 下行衰减

### 2. 当前处在什么位置

重点观察：

- 是否接近前高 / 前低
- 是否刚完成突破 / 跌破
- 是否处于回踩 / 反抽

可归类为：

- 突破前
- 突破中
- 突破回踩
- 顶部 2B 观察
- 跌破中
- 破位反抽
- 底部 2B 观察
- 箱体震荡

### 3. 当前动量强不强

重点观察：

- `ROC20`
- `ROC60`
- 当日实体大小
- 最近 3 到 5 根 K 线斜率
- 成交量是否显著高于 20 日均量

可输出：

- 强
- 中
- 弱

### 4. 有没有反向证据

重点观察：

- 是否跌回关键位内部
- 是否出现 2B
- 是否回踩放量下杀
- 是否反抽缩量无效
- 是否拖延失速

可输出：

- 无明显反向证据
- 有轻微反向证据
- 反向证据明确

## 五、一个实用的日线评分法

如果后面我们要做半量化跟踪，可以先用一个简单版本。

### 1. 指标层

- `ROC20`
- `ROC60`
- `Volume Ratio`
- `Breakout Distance = (Close - PrevHigh20) / PrevHigh20`
- `Breakdown Distance = (PrevLow20 - Close) / PrevLow20`

补充一个量价推进指标：

- `VPT = cumulative(Volume * (Close - PrevClose) / PrevClose)`
- `VPT Slope10 = VPT - VPT[-10]`

### 2. 方向分

如果当前价格：

- 在 `MA20` 之上且 `MA20 > MA60`，方向分偏多
- 在 `MA20` 之下且 `MA20 < MA60`，方向分偏空

### 3. 结构分

可按结构给基础分：

| 结构 | 分值倾向 |
|:---|:---|
| 突破回踩确认 | `+2` |
| 放量突破前高 | `+2` |
| 底部 2B | `+1` |
| 箱体中性 | `0` |
| 顶部 2B | `-1` |
| 破位反抽 | `-2` |
| 放量跌破前低 | `-2` |

### 4. 一个简单总分

先不追求复杂，先做一个可解释版本：

```text
Daily Momentum Score
= 0.30 * z(ROC20)
+ 0.25 * z(ROC60)
+ 0.20 * z(VPT Slope10)
+ 0.15 * z(Volume Ratio)
+ 0.10 * Structure Score
```

解释：

- 分数越高，说明顺势推进越明显
- 分数越低，说明下行动量或结构破坏越明显
- 真正使用时，仍然要让结构判断优先于总分

也就是说：

> 总分用来排序，结构用来下结论。

## 六、把结构语言固定下来

你后面每天判断时，尽量只用固定词，不要每次临时发挥。

### 偏多结构

- 放量突破前高
- 突破回踩确认
- 低位 ABC
- 底部 2B

### 偏空结构

- 放量跌破前低
- 破位反抽
- 高位 ABC
- 顶部 2B

### 中性结构

- 箱体震荡
- 趋势衰减但未破坏
- 回踩中但尚未确认

其中有一个很重要的区分：

> 跌破前低后，反弹到前低附近站不上，不叫底部 2B，叫破位反抽。  
> 只有跌破后快速收回并站稳，才进入底部 2B 观察。

## 八、研究和回测怎么落地

建议分三步，不要一开始做太大。

### 第一步：先做人工跟踪

先选少数标的，比如：

- `QQQ`
- `MSTR`

每天记录：

- 结构
- 动量
- 成交量
- 是否出现反向证据

目的不是预测，而是训练眼睛和语言。

### 第二步：做历史回测

先验证几个最核心的结构：

- 放量突破前高
- 突破回踩确认
- 顶部 2B
- 放量跌破前低
- 破位反抽
- 底部 2B

回测重点不要只看 20 日后涨跌幅，还要看：

- 命中率
- 平均收益
- 最大回撤
- 信号后 5 日 / 10 日 / 20 日表现

### 第三步：再做评分系统

等我们确认哪些结构长期更有用，再把这些结构和量价要素正式并入打分。

顺序应该是：

> 先理解现象，再抽象规则，最后再压缩成分数。

不要反过来。

---

## 九、现阶段最重要的原则

这套体系里，最重要的不是公式，而是顺序。

### 原则 1

先看结构，再看指标。

### 原则 2

先看关键位冲击，再看普通波动。

### 原则 3

先看量价是否配合，再讨论真假突破。

### 原则 4

先承认趋势有惯性，再去找趋势结束的证据。

### 原则 5

不要用“看起来很低”对抗“正在变弱”。

---

## 十、一句话总纲

如果把这套方法压缩成一句话：

> 先用结构确定方向，再用动量判断强弱，最后用成交量和回踩确认真假；只要没有足够强的反向证据，就默认趋势更容易延续。

这句话，就是后面所有观察、回测和执行的总纲。
