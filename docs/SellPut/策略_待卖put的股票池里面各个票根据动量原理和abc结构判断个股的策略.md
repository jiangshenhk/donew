# 卖 Put + ABC/KDJ/MACD 综合判断模型

> **公共规则优先**：分析标的、每日必分析池和禁用清单读取 `策略辅助_公用的股票池.md`；DTE、Delta、年化与安全垫读取 `策略辅助_分析指标指引.md`。本文中旧的低 Delta 示例仅用于解释历史结构，不得作为新开 Sell Put 参数。

这个模型把两套框架合并：

1. **股票短线动量与历史行为画像模型**：决定这只股票/指数是否适合卖 put。
2. **ABC + K线成交量 + KDJ/MACD 趋势模型**：判断当前趋势处于 A跌、B弹、C跌还是反转修复。

核心目标：

> 先判断当前趋势结构，再判断卖 put 是否有统计优势。  
> 不用历史支撑去对抗当前破位，也不用指标超卖去硬接放量下跌。

配套研究稿：

- [研究_日线动量评估框架](./研究_日线动量评估框架.md)

## 一、总判断顺序

```text
当前K线与成交量
↓
均线结构
↓
KDJ / MACD 动能
↓
ABC阶段
↓
相对强弱
↓
股票历史行为画像
↓
事件 / 季节性 / 估值 / 期权波动
↓
卖 put 结论
```

核心原则：

> 当前动量是第一优先级，历史行为是过滤器，ABC/KDJ/MACD是趋势识别工具。

---

## 二、两个模型的分工

| 模型 | 负责回答 | 不能做什么 |
|:---|:---|:---|
| 短线动量与历史行为画像 | 这只股票/指数历史上是否适合在当前位置卖 put | 不能替代当前价格行为 |
| ABC/KDJ/MACD趋势模型 | 当前是下跌、反弹、修复还是反转 | 不能单独决定卖 put |

正确用法：

> 先用 ABC/KDJ/MACD 判断“现在能不能接”，再用历史行为画像判断“这只标的值不值得接”。

## 三、第一层：当前趋势状态

| 状态 | K线成交量 | KDJ | MACD | ABC阶段 | 初步结论 |
|:---|:---|:---|:---|:---|:---|
| 上行延续 | 站上20日线，回踩不破 | 强势但未死叉 | 红柱稳定 | 非A跌 | 可进入下一层 |
| 上涨衰竭 | 长上影、放量滞涨 | 高位死叉 | 红柱缩短 | 可能转A | 降级观察 |
| A段下跌 | 放量大阴，跌破20日线 | 向下死叉 | 绿柱扩大 | A段形成 | 暂停 |
| B段反弹 | 不破A低，反弹修复 | 低位金叉 | 绿柱缩短 | B段 | 观察，不急卖 |
| C段下跌 | 反弹失败，跌破A低 | 二次死叉 | 绿柱扩大 | C段 | 暂停 |
| 反转修复 | 收回20日线/关键位 | 金叉上行 | 绿柱缩短或翻红 | 修复确认 | 可进入卖put评估 |

硬规则：

- A段刚形成，不卖 put。
- C段跌破A低，不卖 put。
- B段反弹如果缩量，不当反转。
- 只有反转修复或强势回踩，才进入卖 put 评估。

## 四、第二层：ABC交易法则精确定义

ABC法则不是用来预测底部和顶部，而是用来确认趋势是否真正发生变化。

核心原则：

> 不在最低点抄底，也不在最高点逃顶。等市场自己证明趋势已经改变，再行动。

### 1. 低位ABC

低位ABC用于判断下跌趋势是否可能反转向上。

结构：

```text
                B
               / \
              /   \
             /     \        突破B：趋势确认
            /       C  ->  ->  ->
           /
          A
```

定义：

| 点位 | 含义 |
|:---|:---|
| A | 阶段低点 |
| B | 第一轮反弹高点 |
| C | 回踩形成的次低点，必须高于A |

低位ABC成立必须同时满足：

| 条件 | 含义 |
|:---|:---|
| 突破原下降趋势线 | 旧的下跌节奏被破坏 |
| C点高于A点 | 市场不再创新低，卖压减弱 |
| 向上突破B点 | 买方力量占优，新的上涨趋势得到确认 |

注意：

> 仅仅出现反弹，不代表趋势反转。  
> 仅仅站上下降趋势线，也不代表趋势反转。  
> 真正的确认信号，是突破B点。

### 2. 高位ABC

高位ABC用于判断上涨趋势是否可能结束。

结构：

```text
          A
         / \
        /   \
       /     \
      /       B  -> 跌破B：趋势确认
     /       /
            C
```

定义：

| 点位 | 含义 |
|:---|:---|
| A | 阶段高点 |
| B | 第一轮明显回调低点 |
| C | 反弹形成的次高点，必须低于A |

高位ABC成立必须同时满足：

| 条件 | 含义 |
|:---|:---|
| 跌破原上升趋势线 | 原有上涨节奏被破坏 |
| C点低于A点 | 反弹无法创新高，多头动能下降 |
| 向下跌破B点 | 空方占优，下跌结构得到确认 |

注意：

> C点低于A点，只能说明上涨开始变弱。  
> 真正的趋势转弱确认，是价格跌破B点。

### 3. 成交量配合

低位ABC最理想的成交量变化：

| 阶段 | 价格变化 | 理想成交量 |
|:---|:---|:---|
| A到B | 从低点反弹 | 成交量逐步放大 |
| B到C | 回踩确认 | 成交量逐步缩小 |
| C突破B | 第二次上攻 | 成交量再次放大，最好超过A到B阶段 |

含义：

> 第一次上涨有人买，回调时卖的人减少，第二次上涨时买盘更强。

如果价格突破B点，但成交量明显不足，需要警惕假突破。

成交量在本模型里不是简单辅助项，而是结构确认项。

核心口诀：

> 结构定方向，成交量定真假，动量定强弱。

### 4. 量价结构确认

突破、跌破、反弹、回踩都必须结合成交量判断真假。

| 结构 | 价格动作 | 成交量验证 | 含义 |
|:---|:---|:---|:---|
| 低位ABC | A反弹到B，回踩C不破A，再突破B | A到B放量，B到C缩量，突破B再放量 | 偏多确认 |
| 底部2B | 跌破前低后快速收回 | 跌破时可放量恐慌，收回时量能跟上 | 假跌破，偏多 |
| 高位ABC | A回调到B，反弹C不过A，再跌破B | A到B放量下跌，B到C缩量反弹，跌破B再放量 | 偏空确认 |
| 顶部2B | 突破前高后快速跌回 | 突破放量但无法维持，回落放量更危险 | 假突破，偏空 |
| 破位反抽 | 跌破前低后反弹到前低，但站不上 | 跌破放量，反抽缩量，二次下跌放量 | 偏空延续 |
| 突破回踩 | 突破压力后回踩压力不破 | 突破放量，回踩缩量，再上攻放量 | 偏多延续 |

判断顺序：

```text
当前是高位还是低位
↓
是突破、跌破、反弹还是回踩
↓
突破/跌破有没有放量
↓
反弹/回踩有没有缩量
↓
二次确认时量能有没有重新放大
↓
归类：真突破 / 假突破 / 真破位 / 假破位 / 反抽确认 / 回踩确认
```

卖Put提醒：

> 价格到了历史支撑，但跌破放量、反抽缩量、二次下跌再放量时，说明风险仍在扩张，不能因为权利金变肥就卖 put。

### 5. 两线定乾坤

当底部反复震荡，不能只看是否突破下降趋势线。

| 线条 | 作用 |
|:---|:---|
| S1：下降趋势线 | 判断旧趋势是否被破坏 |
| S2：水平压力线 | 判断新趋势是否真正启动 |

常见路径：

```text
下跌 -> 止跌 -> 横盘筑底 -> 放量突破平台 -> 主升浪
```

规则：

> 只突破S1，仍可能只是底部整理；放量突破S2，新的上涨空间才真正打开。

### 6. 2B法则

2B法则用于提前识别假突破或假跌破。

| 类型 | 信号 | 含义 |
|:---|:---|:---|
| 顶部2B | 短暂突破前高后快速跌回前高下方 | 追涨资金被套，顶部风险增加 |
| 底部2B | 短暂跌破前低后快速站回前低上方 | 恐慌盘释放，卖方力量衰减 |

ABC与2B的分工：

| 方法 | 特点 | 用法 |
|:---|:---|:---|
| 2B法则 | 更早、更灵敏，但假信号较多 | 进入观察名单，试探性观察 |
| ABC法则 | 更慢、更稳健 | 确认趋势反转，增加信心 |
| 两线定乾坤 | 适合震荡筑底 | 等平台突破后跟进 |

注意：

> 底部2B偏多，顶部2B偏空。  
> 跌破前低后反弹到前低附近站不上，不是底部2B，而是破位反抽。

交易节奏：

```text
2B出现 -> 进入观察名单
C点形成 -> 准备交易
突破B点 -> 趋势确认，正式执行
```

### 7. 破位反抽

破位反抽用于识别下跌结构是否延续。

结构：

```text
前低 / 原支撑
────────────
      ↓ 跌破
        新低
      ↑ 反弹
回抽前低失败
      ↓
继续下跌
```

定义：

| 点位 | 含义 |
|:---|:---|
| 前低 | 原来的支撑位，跌破后变成压力位 |
| 新低 | 跌破前低后的低点 |
| 反抽点 | 价格反弹回前低附近，但不能重新站上 |

成立条件：

| 条件 | 含义 |
|:---|:---|
| 放量跌破前低 | 原支撑被真实卖压击穿 |
| 反弹到前低附近缩量 | 买盘只是技术反弹，承接不足 |
| 站不上前低并再次转弱 | 支撑转压力，空方仍占优 |
| 二次下跌重新放量 | 下跌结构延续确认 |

交易含义：

> 破位反抽是偏空延续结构，不是抄底结构。  
> 只有重新站回前低并稳住，破位有效性才减弱；如果跌破新低后快速收回，才可能转入底部2B观察。

卖Put动作：

| 情况 | 动作 |
|:---|:---|
| 放量破位后第一次反抽 | 不卖 put，观察前低能否收回 |
| 反抽缩量且站不上前低 | 暂停，避免卖在二次下跌前 |
| 重新站回前低并缩量回踩不破 | 从红色转黄色观察 |
| 跌破新低后快速收回 | 进入底部2B观察名单 |

### 8. 有效突破标准

为了减少假突破，突破B点最好满足：

| 项目 | 建议标准 |
|:---|:---|
| 收盘确认 | 日线收盘站上B，而不是盘中短暂刺穿 |
| 突破幅度 | 超过B点约2%-3% |
| 成交量 | 突破日成交量高于近期均量，最好明显放大 |
| 回踩验证 | 突破后回踩B不破，再次转强 |
| 大盘环境 | 大盘或所属行业不处于明显下跌趋势 |
| 时间级别 | 周线优先级高于日线，日线高于小时线 |

这会牺牲一点入场价格，但能减少假突破损失。

### 9. 红黄绿状态

| 状态 | 结构判断 | 现货操作 | 卖 Put 操作 |
|:---|:---|:---|:---|
| 绿色 | 低位ABC已成立，突破B后站稳 | 可持有或分批加仓 | 可选择远虚值Put，优先等回调时卖 |
| 黄色 | 只突破下降趋势线，但B尚未突破；或仍在箱体内 | 小仓观察 | 降低仓位，避免一次铺满 |
| 红色 | 高位ABC成立，跌破B；同时跌破中期均线 | 减仓，等待修复 | 暂停新增Sell Put |

卖Put关键提醒：

> 有些很肥的权利金，不是市场送来的利息，而是提前支付给你的接飞刀补偿费。

## 五、第三层：宏观流动性事件过滤器

在判断 BTC、QQQ、黄金、七姐妹等风险资产时，必须加入宏观流动性事件。

重点事件：

- 日本央行加息或释放鹰派信号
- 美联储FOMC、CPI、PCE、非农
- 美债收益率快速上行
- 美元指数快速走强
- 日元急升或套息交易平仓
- 大型IPO/指数纳入导致资金再平衡
- BTC ETF大额流入或流出

### 日本央行加息与风险资产

日本央行加息对 BTC 和纳指的影响，不只看加息当天，而要看预期交易窗口。

观察经验：

| 时间 | 事件 | 风险资产含义 |
|:---|:---|:---|
| 2024年3月 | 日本央行退出负利率 | 全球流动性预期变化 |
| 2024年7月 | 日本央行继续加息 | 日元套息交易压力上升 |
| 2025年1月 | 日本央行再加息 | 风险资产去杠杆压力 |
| 2025年12月19日 | 日本央行加息至0.75% | 事件当天可能已被市场吸收，需看前期是否已提前下跌 |

交易规则：

> 日本央行加息不是只看当天反应，而要看加息前4-8周风险资产是否已经提前下跌。

判断方式：

| 情况 | 判断 | 卖 put 动作 |
|:---|:---|:---|
| 加息预期升温，BTC/QQQ已开始放量破位 | 预期交易开始 | 暂停 |
| 加息前已从高点跌20%-30% | 可能已提前释放 | 等事件落地后确认 |
| 加息当天风险资产不跌反涨 | 卖压可能已释放 | 观察B段反弹质量 |
| 加息后日元继续走强、美债收益率上行 | 流动性压力未结束 | 暂停 |
| 加息后日元走弱、风险资产收回关键位 | 宏观压力缓和 | 可重新评估低Delta |

注意：

> 不能把“日本加息后历史上常跌”机械当作做空信号。真正有用的是：加息预期是否触发套息交易平仓，以及这个压力是否已经提前反映在K线里。

## 六、第四层：历史行为画像

趋势状态过关后，再看标的自身历史习惯。

| 模块 | 判断问题 | 结论格式 |
|:---|:---|:---|
| 长期均线行为 | 50日/100日/200日线附近是否经常反弹？ | 支撑有效 / 支撑一般 / 容易破位 / 当前不可依赖 |
| 大涨后行为 | 大涨后是延续还是回吐？ | 趋势延续型 / 容易回吐型 / 需要确认型 |
| 大跌后行为 | 大跌后是修复还是连续下杀？ | 修复型 / 连续下杀型 / 等确认型 |
| 事件行为风险 | 财报、IPO、政策、宏观数据是否未落地？ | 低 / 中 / 高 |
| 季节性位置 | 当前月份/季度是否顺风？ | 顺风 / 中性 / 逆风 |
| 估值压力 | 当前估值是否容易被杀？ | 低 / 中 / 高 |
| 期权波动习惯 | IV高后是否容易回落？ | 卖方友好 / 中性 / 卖方不友好 |

原则：

> 历史上经常反弹，只能在当前已经止跌时加分；不能在当前放量破位时当作接盘理由。

## 七、第五层：相对强弱

必须比较相对强弱。

| 标的 | 比较对象 |
|:---|:---|
| 个股 | 行业ETF + QQQ/SPY |
| 科技股 | QQQ / XLK / SMH / SOXX |
| 半导体 | SOXX / SMH |
| 纳指 | QQQ / SPY / VIX或VXN |
| BTC相关 | BTC / MSTR / COIN / BTC ETF资金流 |

判断：

| 相对强弱 | 含义 | 卖 put 动作 |
|:---|:---|:---|
| 强于行业和大盘 | 个股有承接 | 可加分 |
| 跟随行业回调 | 中性 | 看趋势结构 |
| 弱于行业和大盘 | 个股独立走弱 | 降级 |
| 行业和大盘同时破位 | 系统性风险 | 暂停 |

## 八、第六层：卖 Put 执行映射

| 综合状态 | 条件 | 动作 |
|:---|:---|:---|
| 正向共振 | 上行延续 + 历史支撑有效 + 事件风险低 | 可等回踩后进入公共参数筛选 |
| 修复共振 | B段强反弹或反转修复 + 历史大跌后常修复 | 小仓候选，仍须满足公共参数 |
| 中性共振 | 震荡消化 + 历史区间震荡 | 只观察，不以降低 Delta 代替结构确认 |
| 冲突 | 当前动量弱，但历史支撑好 | 等确认，不主动卖 |
| 负向共振 | A段/C段下跌 + 历史也容易续跌 | 暂停 |
| 事件高风险 | 财报/宏观/IPO/监管未落地 | 暂停或只观察 |

| 事件风险高 | -3 |

### 4. 宏观流动性分

| 条件 | 分数 |
|:---|---:|
| 日本央行/美联储事件已落地且风险资产收回关键位 | +1 |
| BTC ETF或风险资产资金流恢复 | +1 |
| 加息预期升温但价格仍强于20日线 | 0 |
| 加息预期升温且价格跌破20日线 | -2 |
| 日元急升或套息交易平仓迹象明显 | -3 |
| 事件前4-8周已放量破位 | -3 |

### 5. 结论映射

| 总分 | 结论 |
|:---|:---|
| +5及以上 | 可卖 |
| +2到+4 | 谨慎卖 |
| -1到+1 | 只观察 |
| -2及以下 | 暂停 |

硬性否决：

- 放量跌破A段低点：直接暂停。
- 放量跌破50日线：直接暂停。
- 事件风险高且未落地：不重仓卖 put。
- Put skew急剧扩大且价格破位：暂停。

## 十、图表输入要求

## 十一、判断标准

### 1. 结构判断顺序

默认判断顺序固定为：

```text
先看裸K结构
↓
确定前高前低和区间
↓
判断当前位置
↓
打开EMA确认趋势约束
↓
看MACD/RSI/ATR/MAVOL
↓
给出结构分类和执行结论
```

### 2. 前高前低的定义

| 名称 | 定义 |
|:---|:---|
| 前高 | 当前结构中最重要的阶段高点，通常是反弹失败高点、平台上沿或最近一次明显冲高点 |
| 前低 | 当前结构中最重要的阶段低点，通常是明显止跌点、恐慌低点或最近一次结构锚点 |

角色转换规则：

- 前高未突破前，默认是压力。
- 前高有效突破后，回踩不破，压力可转支撑。
- 前低未跌破前，默认是支撑。
- 前低有效跌破后，反抽不过，支撑可转压力。

口诀：

> 前高未破是压力，破后回踩成支撑。  
> 前低未破是支撑，破后反抽成压力。

### 3. 第一压力区和第二压力区的定义

| 名称 | 定义 |
|:---|:---|
| 第一压力区 | 离当前价最近、最可能先挡住反弹的一段阻力区域 |
| 第二压力区 | 第一压力区之上、更大一级、更重的阻力区域 |

命名顺序按 `离当前价格的远近` 排，不按画图先后排。

默认取法：

1. 先找最近关键前高。
2. 从前高往下看，中间哪些平台、反抽、震荡区曾经卡住价格。
3. 离当前价最近的一层，定义为第一压力区。
4. 再上一层，定义为第二压力区。

### 4. 前低观察区和失效位

| 名称 | 定义 |
|:---|:---|
| 前低观察区 | 前低锚点附近的一段测试区，一般用 `前低 ± 合理波动` 表示 |
| 结构失效位 | 跌破关键前低并无法快速收回时的确认失效条件 |

默认不把盘中瞬时跌破直接当失效，优先看：

- 是否跌破后 `快速收回`
- 是否跌破后 `反弹站不回`

### 5. 动量判断标准

| 指标 | 偏多 | 中性 | 偏空 |
|:---|:---|:---|:---|
| `MACD` | 红柱扩大、零轴上金叉 | 柱体缩短、纠缠 | 绿柱扩大、零轴下死叉 |
| `RSI14` | `>50` | `40-50` | `<40` |
| `ATR14` | 反弹超过 `1 ATR` 且配合放量，才更像有效修复 | 普通波动 | 跌破关键位超过 `1 ATR`，破坏可信度更高 |
| `VOL/MAVOL` | `VOL > MA5 > MA20`，更像放量确认 | 量能普通 | 反弹量始终压在 `MA20` 下，偏弱 |

### 6. 结构分类标准

| 结构 | 核心标准 |
|:---|:---|
| 突破回踩 | 突破前高或平台后，回踩不破，量能缩后再放 |
| 破位反抽 | 跌破前低或平台后，反弹回原位附近站不上 |
| 底部2B | 跌破前低后快速收回前低上方 |
| 顶部2B | 突破前高后快速跌回前高下方 |
| 低位ABC | A为低点，B为首个反弹高点，C高于A，突破B确认 |
| 高位ABC | A为高点，B为首个回调低点，C低于A，跌破B确认 |
| 箱体震荡 | 价格在明确定义的上下沿区间内来回波动，未给出有效突破或跌破 |

### 7. 今日图表结论口径

默认要回答五个问题：

1. 当前在什么结构位置？
2. 当前是偏多、偏空还是观察？
3. 上方最近先会撞到哪一个压力区？
4. 下方最重要的观察区和失效位在哪里？
5. 当前动作更像突破回踩、破位反抽、低位修复，还是趋势延续？

| 态度 | 含义 | 卖 put 动作 |
|:---|:---|:---|
| `crash_risk` | 大跌或二次下杀风险更高 | `pause` |

| `up` | 上涨或趋势延续概率更高 | `can_sell_put` |

| 态度 | 含义 |
|:---|:---|

- `小跌` 不等于 `大跌`，小跌可参与，但必须区分是正常回调，还是正在向大跌演化。

- `上涨、震荡、可控小跌` 都可以进入后续参数比较；`大跌` 先排除。

## 十六、操盘纪律

1. 不在A段第一根放量大阴线卖 put。
2. 不用KDJ超卖当作卖 put 理由。
3. 不用历史支撑对抗当前破位。
4. B段反弹必须分强弱，弱B不能当反转。
5. C段跌破A低时，不加仓摊平。
6. 事件未落地时，不因为权利金高就重仓。
7. 卖 put 的前提是风险正在收缩，不是风险正在扩张。
8. 当前动量优先，历史习惯辅助，指标只做确认。
9. 宏观事件要看预期交易窗口，不只看事件当天。
10. 如果加息当天不跌反涨，要判断是压力释放，还是反弹诱多。
11. 放量破位、缩量反抽、二次下跌放量时，不把反弹当反转。
12. 突破要看放量，回踩要看缩量，再上攻要看重新放量。
