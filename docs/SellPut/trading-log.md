# 卖Put日常交易记录

> 用AI和数据，重新理解复利投资。

这个页面用于记录我日常 Sell Put 的交易、观察和复盘。

这里不是荐股，也不是操作建议，而是把每一次卖 Put 的思考过程记录下来，方便以后复盘：

```text
当时市场环境是什么？
为什么选择这个标的？
为什么选择这个行权价？
安全垫有多少？
权利金是否来自市场恐慌溢价？
如果被接货，是否愿意长期持有？
结果是否符合预期？
```

---

## 一、记录原则

每一笔 Sell Put 交易，都尽量记录六个要素：

| 项目 | 说明 |
|---|---|
| 日期 | 按美股交易日记录 |
| 标的 | QLD / MSTR / IBIT / 其他 |
| 合约 | 到期日、行权价、Put 类型 |
| 权利金 | 成交价或中间价 |
| 安全垫 | 当前价格到行权价的距离 |
| 当时判断 | 为什么这笔交易值得做 |
