# 策略辅助_分析指标指引

> 用途：本文件是 Sell Put 每日分析的唯一参数来源。总控、市场风向、动量结构、期权链 SOP 和日报模板均必须读取本文件；如与其他文件的旧阈值冲突，以本文件为准。
> 
> 更新原则：参数变更只修改本文件，并在每日结论中注明例外原因。它是研究与风险控制规则，不构成自动下单指令。

---

## 一、统一候选参数

所有进入“可卖 / 小仓可卖”候选表的 Put，默认必须同时满足：

| 指标 | 统一目标 | 口径 |
|---|---:|---|
| DTE | 4-10 个自然日 | 以期权到期日减分析日计算；到期日当天不新开仓 |
| Put Delta | `-0.12` 至 `-0.18` | 使用绝对值 `0.12-0.18`；优先接近 `0.15` |
| 目标年化 | `>= 30%` | 统一按 Mid 计算；`Mid = (Bid + Ask) / 2` |
| 安全垫 | 优先 `>= 7%` | `(现价 - Strike) / 现价`；不足 7% 必须单独写明原因并降级 |
| 流动性 | Bid、Ask、OI、成交量可核验 | 价差不是硬门槛，但必须限价成交、不得按 Ask 假设成交 |
| 账户占用 | 先通过账户风险规则 | 期权参数合格不覆盖仓位、相关性或接货能力限制 |

年化统一计算：

```text
Mid = (Bid + Ask) / 2
单次收益率 = Mid / Strike
简单年化 = 单次收益率 × 365 / DTE
```

Bid 与 Ask 用于验证流动性和设置限价；年化候选门槛统一使用 Mid。实际成交价必须在交易记录中单独保存，不回写为候选年化口径。

---

## 二、例外与降级

以下情况不能擅自更改 Delta、DTE 或年化门槛，只能给出“不适合 / 观察”：

1. 市场风向为逆风、日元套息交易风险升级、重大事件未落地，或个股放量破位。
2. 标的在公共禁用清单中。
3. 现金担保金额、同类风险暴露或账户桶限制不合格。
4. 关键期权字段缺失，或正股与期权链时间戳不一致。

唯一允许的执行例外是：安全垫低于 7% 但其他条件满足时，结论最多为“**小仓适合**”，并必须同时写明结构支撑、失效位、账户占用和为什么接受较低安全垫。不得把该例外写成默认规则。

## 三、强势高波动股的技术优先分支

适用：INTC、HOOD、SMCI 及公共股票池中标记为“强势高波动”的标的。

高 IV、新高、事件后上涨或 RSI 偏高本身都不是一票否决；但也不能因为权利金高而在加速段直接卖 Put。必须按下表处理：

| 技术状态 | Sell Put 结论 |
|---|---|
| 突破后缩量回踩，守住突破位；或形成更高低点并重新走强；Strike 在明确支撑下方 | 可进入参数筛选，合格后才可“适合 / 小仓适合” |
| 高位连续加速、长上影后未消化、RSI 极端但尚无回踩确认 | 只观察，等待回踩或横盘消化 |
| 跌回突破位下方、放量跌破支撑、相对行业明显转弱 | 不适合，暂停新卖 Put |

RSI 只用于识别“是否需要等待消化”，不替代结构判断。任何强势股仍须满足本文件第一节的 DTE、Delta、年化、账户与流动性要求。

---

## 四、输出要求

每日候选表必须至少展示：`symbol / 到期日 / DTE / Strike / Bid / Ask / Mid / Delta / IV / OI / 安全垫 / Mid 年化 / 现金担保金额 / 结论`。

固定结论顺序：

```text
市场风向 -> 个股技术结构 -> 期权参数 -> 账户风险 -> 适合 / 小仓适合 / 不适合
```
