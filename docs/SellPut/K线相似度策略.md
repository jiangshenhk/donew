# K线相似度策略

## 1. 策略目标

K线相似度策略用于回答一个问题：

> 当前标的最近一段K线形态，和历史上哪些阶段最像？这些历史相似阶段之后，未来上涨或下跌的概率、收益空间和回撤风险如何？

该方法的核心假设是：市场行为具有一定重复性。若当前价格结构、震荡节奏、上影线/下影线分布与历史某些阶段高度相似，则这些历史阶段之后的收益分布，可以作为当前决策的一个统计参考。

该策略不直接预测确定性涨跌，而是输出概率和风险收益分布。

## 2. 适用场景

- 判断指数或个股当前形态是否处于历史相似的强势区间。
- 辅助确认买入、持有、减仓、止盈或止损决策。
- 与行业分析、基本面判断、量价关系、支撑阻力位结合使用。
- 用于比较多个标的：寻找当前形态更接近历史上涨阶段的标的。

不建议将该方法作为单一交易依据。突发政策、业绩变化、产业催化、流动性冲击等因素，可能使历史形态失效。

## 3. 数据口径

基础数据使用日K线：

- Open：开盘价
- High：最高价
- Low：最低价
- Close：收盘价
- Volume：成交量，可作为后续增强特征

本次样例使用东方财富日K数据接口：

- 上证指数：`1.000001`
- 绿的谐波：`1.688017`

截至样例运行日，数据更新至 `2026-06-09`。

## 4. 形态窗口

常用窗口长度：

- 20日：偏短线，约一个交易月。
- 30日：偏短中线，适合观察一段完整震荡或上行结构。
- 60日：偏中期，适合观察趋势结构和阶段性资金行为。

实际使用时不建议只看一个窗口，应同时观察多个窗口的稳定性。

如果 20日、30日、60日窗口都指向同一方向，信号更强；如果短窗口偏强、长窗口偏弱，则说明当前更可能是短线机会，而不是中期趋势确认。

## 5. 特征处理

为了避免价格绝对高低影响相似度，窗口内价格需要先做归一化处理。

以窗口第一天的收盘价为基准：

```text
normalized_price = 当前价格 / 窗口首日收盘价 - 1
```

分别对 Open、High、Low、Close 做归一化，得到四组相对价格序列。

这样比较的是形态，而不是价格本身。

## 6. 相似度评分

### 6.1 OHLC综合相关系数

分别计算当前窗口与历史窗口的四个相关系数：

```text
Corr(Open)
Corr(High)
Corr(Low)
Corr(Close)
```

综合相似度：

```text
综合相似度 = [Corr(Open) + Corr(High) + Corr(Low) + Corr(Close)] / 4
```

相似度越接近 1，说明两段K线形态越接近。

常用阈值：

- `>= 90%`：高度相似，样本通常较少，信号更纯但稳定性需要检查。
- `>= 85%`：较高相似，样本适中。
- `>= 80%`：宽松相似，样本较多，但噪声也更大。

### 6.2 归一化欧氏距离

在综合相关系数之外，可计算归一化欧氏距离作为辅助排序指标：

```text
distance = sqrt(mean((Zscore(当前窗口) - Zscore(历史窗口))^2))
```

相关系数看方向和形状，欧氏距离看具体贴合程度。实操中可优先按综合相似度排序，再用距离过滤。

### 6.3 DTW动态时间归整

DTW适合处理节奏略有错位的形态，例如一段行情用 25 天完成，另一段行情用 30 天完成。

但DTW容易提高拟合弹性，也更容易过拟合。建议作为增强模块，而不是第一版策略的唯一评分方法。

## 7. 回测方法

对每一个历史窗口：

1. 取历史窗口长度为 `N` 的K线片段。
2. 与当前最近 `N` 根K线计算相似度。
3. 排除距离当前太近的窗口，避免未来收益数据重叠。
4. 统计历史窗口结束之后的未来收益：
   - 未来5日收益
   - 未来10日收益
   - 未来20日收益
5. 同时统计未来区间内：
   - 最大盈利空间：MFE，Maximum Favorable Excursion
   - 最大亏损/回撤空间：MAE，Maximum Adverse Excursion

收益计算：

```text
未来收益 = 未来第H日收盘价 / 窗口结束日收盘价 - 1
```

最大盈利空间：

```text
MFE = 未来H日内最高价 / 窗口结束日收盘价 - 1
```

最大回撤空间：

```text
MAE = 未来H日内最低价 / 窗口结束日收盘价 - 1
```

## 8. 信号解释框架

### 8.1 强信号

满足以下条件时，K线相似度信号较强：

- 30日或60日窗口中存在足够多的 `90%+` 样本。
- 未来10日、20日上涨概率明显高于 50%。
- 平均收益和中位收益同时为正。
- 平均最大盈利空间明显大于平均最大回撤空间。
- 不同窗口长度结论一致。

### 8.2 弱信号

出现以下情况时，信号偏弱：

- 只有少数样本达到 `90%+`，样本数不足。
- 平均收益为正，但中位收益为负，说明结果可能被少数极端行情拉高。
- 20日窗口看多，但60日窗口转弱，说明更像短线弹性而非中期趋势。
- 最大回撤空间较大，风险收益比不稳定。

### 8.3 无效或谨慎信号

出现以下情况时，不宜据此交易：

- 高相似度样本数为 0。
- 上涨概率接近 50%，收益中位数接近 0。
- 不同窗口结果明显冲突。
- 当前标的存在重大基本面或政策事件，历史形态可比性下降。

## 11. 当前策略判断模板

以绿的谐波样例为例：

```text
短线5日：略偏多，但波动较大。
中短线10日：相似K线统计优势最明显。
中期20日：结果分化，不能简单外推。
策略倾向：若已有基本面和行业逻辑支撑，可将K线相似度作为持有或低吸辅助；若只靠形态，不宜追高重仓。
```

更稳健的交易解释：

- 如果价格回踩但不破关键支撑，且行业强度仍在，可考虑继续观察10日窗口的弹性。
- 如果放量跌破当前窗口低点，说明历史相似路径失效，应降低K线相似度信号权重。
- 如果后续上涨但成交量不能跟上，需警惕相似度信号兑现后回落。

## 12. 增强方向

后续可把策略从“形态相似”升级为“形态 + 量价 + 行业强弱”：

### 12.1 加入成交量

增加以下特征：

- 成交量均线
- 量比
- 放量上涨/缩量回调
- 成交额分位数

### 12.2 加入波动率

增加以下特征：

- 20日波动率
- ATR
- 振幅
- 上下影线比例

### 12.3 加入趋势指标

增加以下特征：

- MA5、MA10、MA20、MA60斜率
- 均线多头/空头排列
- 收盘价相对均线位置

### 12.4 加入行业相对强度

对个股而言，建议同步比较：

- 个股相对所属行业指数的超额收益
- 行业指数相对上证指数/沪深300的超额收益
- 板块内强势股数量

如果个股K线相似度偏强，同时行业相对强度也偏强，信号质量更高。

## 13. 风险控制

K线相似度策略必须配合风险控制：

- 不因相似度高而忽视基本面变化。
- 不在样本数过少时重仓。
- 不把平均收益当作确定收益。
- 重点关注中位收益和最大回撤。
- 高波动个股应降低单笔仓位。
- 当价格走出与历史样本明显不同的路径时，应及时判定信号失效。

## 14. 可复用脚本

当前目录下已有脚本：

```text
kline_similarity.py
```

常用运行方式：

```bash
python3 kline_similarity.py --window 20 --top 8
python3 kline_similarity.py --window 30 --top 12
python3 kline_similarity.py --window 60 --top 8
```

建议每次分析至少跑三个窗口：

```text
20日：短线节奏
30日：短中线结构
60日：中期趋势
```

最终结论应以多窗口一致性、样本数量、收益中位数和最大回撤共同判断。
