# 股票短线动量与历史行为画像模型

这个模型用于分析某只股票在未来 1-7 个交易日内，是否存在继续下冲、继续上涨或震荡消化的可能，并结合这只股票自己的历史行为习惯，判断是否适合卖 put。

核心目标不是预测长期价值，而是回答：

> 当前卖 put 是顺着短线动量和历史惯性，还是在逆着趋势硬接？

---

## 一、模型总原则

### 1. 当前动量优先，历史习惯辅助

历史行为很重要，但不能替代正在发生的价格行为。

正确顺序：

```text
当前短线动量
↓
股票历史行为画像
↓
估值 / 事件 / 季节性位置
↓
当前是否符合历史惯性
↓
卖 put 是否有统计优势
```

核心规则：

> 当前动量是第一优先级，历史习惯是过滤器，不是反向硬接的理由。

例如：

- 如果当前已经企稳，而历史上这个位置经常有承接，卖 put 可以升级。
- 如果当前正在放量破位，即使历史上这里常反弹，也不能提前接。
- 如果当前大涨后放量滞涨，而历史上该股大涨后经常回吐，不追卖贴价 put。

---

## 二、股票历史行为画像

每只股票都要建立自己的历史行为画像。

重点观察 7 类规律：

| 模块 | 观察内容 | 对卖 put 的意义 |
|:---|:---|:---|
| **长期均线反应** | 50日 / 100日 / 200日线附近是反弹还是破位 | 判断支撑是否真有用 |
| **大涨后回吐习惯** | 单日大涨后未来3-7天平均表现 | 防止大阳线后追卖 put |
| **大跌后修复习惯** | 跌超5%、10%后是否容易反弹 | 判断恐慌后能不能卖 |
| **财报 / 分红 / 除权行为** | 事件前后是否习惯性下跌或波动 | 防止跨事件卖 put |
| **季节性** | 某些月份 / 季度是否经常调整 | 过滤短期风向 |
| **估值区间反应** | PE / PB / P/S 到某区间是否容易杀估值 | 防止高估值卖 put |
| **期权波动习惯** | IV高后是回落还是继续扩张 | 判断卖波动率是否有优势 |

---

## 三、长期均线反应

观察股票在关键均线附近的历史表现：

- 50日线
- 100日线
- 200日线
- 周线级别均线
- 前期平台下沿
- 长期上升趋势线

需要回答：

| 问题 | 判断目的 |
|:---|:---|
| 跌到长期均线附近是否经常反弹？ | 判断支撑有效性 |
| 跌破后是否经常快速收回？ | 判断是假破位还是趋势转弱 |
| 跌破后是否容易连续下杀？ | 判断不能接的位置 |
| 均线附近成交量是否放大？ | 判断是否有资金承接 |
| 当前是否第一次跌破长期均线？ | 第一次破位风险更高 |

结论格式：

> **长期均线行为：支撑有效 / 支撑一般 / 容易破位 / 当前不可依赖。**

---

## 四、大涨后回吐习惯

有些股票大涨后会继续趋势，有些股票大涨后容易快速回吐。

观察：

| 事件 | 后续表现 |
|:---|:---|
| 单日涨幅 > 5% | 未来3-7天是否继续上涨 |
| 单日涨幅 > 10% | 是否容易冲高回落 |
| 跳空高开 | 缺口是否容易回补 |
| 放量大阳线 | 是趋势启动还是短线高潮 |
| 高位大涨 | 是否容易成为阶段顶部 |

卖 put 含义：

| 历史规律 | 动作 |
|:---|:---|
| 大涨后常继续上行 | 可等回踩卖 put |
| 大涨后常回吐 | 不追卖贴价 put |
| 跳空后常补缺口 | 等缺口回补后再评估 |
| 高位大涨后常滞涨 | 降级为观察 |

结论格式：

> **大涨后行为：趋势延续型 / 容易回吐型 / 需要确认型。**

---

## 五、大跌后修复习惯

有些股票大跌后容易反弹，有些股票大跌只是第一段。

观察：

| 事件 | 后续表现 |
|:---|:---|
| 单日跌幅 > 5% | 未来3-7天是否修复 |
| 单日跌幅 > 10% | 是否继续下杀 |
| 跳空低开 | 缺口是否容易回补 |
| 放量大阴线 | 是否意味着趋势转弱 |
| 长下影 | 次日是否继续承接 |

卖 put 含义：

| 历史规律 | 动作 |
|:---|:---|
| 大跌后常快速修复 | 可等次日确认后小仓远OTM |
| 大跌后常继续下冲 | 暂停卖 put |
| 长下影后常反弹 | 等不破低点再考虑 |
| 跳空下跌后常补跌 | 不接第一天 |

结论格式：

> **大跌后行为：修复型 / 连续下杀型 / 等确认型。**

---

## 六、事件行为画像

短线卖 put 必须看事件前后的历史习惯。

重点事件：

- 财报
- 分红
- 除权除息
- 增发 / 配股
- 可转债 / 融资
- 股东减持
- 监管事件
- 产品发布
- 政策补贴
- 行业会议

需要回答：

| 问题 | 判断目的 |
|:---|:---|
| 财报前是否习惯性上涨或下跌？ | 判断预期交易 |
| 财报后是否经常跳空？ | 判断跨财报风险 |
| 分红 / 除权前是否容易下跌？ | 判断事件风 |
| 增发 / 融资后是否持续承压？ | 判断真实供给压力 |
| 事件前IV是否经常虚高？ | 判断是否能卖波动 |

原则：

> 事件未落地时，不因为权利金高就重仓卖 put。

结论格式：

> **事件行为风险：低 / 中 / 高。**

---

## 七、季节性行为

有些股票或行业在某些月份、季度容易调整或走强。

观察：

| 维度 | 示例 |
|:---|:---|
| 月份 | 每年5月、8月、9月是否容易调整 |
| 季度 | 财报季前后是否固定波动 |
| 行业周期 | 半导体、能源、银行、消费是否有季节性 |
| 税务 / 调仓 | 年底税损卖出、季末再平衡 |
| 加密相关 | 周末、ETF资金流、减半周期 |

卖 put 含义：

| 季节性 | 动作 |
|:---|:---|
| 当前处于历史强势窗口 | 可加分 |
| 当前处于历史弱势窗口 | 降级 |
| 季节性不明显 | 中性 |
| 季节性与当前动量冲突 | 以当前动量为先 |

结论格式：

> **季节性位置：顺风 / 中性 / 逆风。**

---

## 八、估值区间反应

估值不是短线唯一变量，但能解释某些股票为什么涨到某个区间后容易被杀。

可观察指标：

- PE
- Forward PE
- PB
- P/S
- EV/EBITDA
- 市销率分位
- 市净率分位
- 相对行业估值溢价

需要回答：

| 问题 | 判断目的 |
|:---|:---|
| 这个股票历史上市场主要看哪个估值指标？ | 避免用错估值 |
| 当前估值处于历史什么分位？ | 判断估值压力 |
| 过去估值到这个分位后是否容易回撤？ | 判断杀估值风险 |
| 当前基本面是否已经改变？ | 判断历史区间是否失效 |
| 行业整体估值是否也高？ | 判断是个股问题还是行业问题 |

注意：

- PE / PB 对银行、周期、传统制造、成熟芯片股更有用。
- P/S 对高增长软件、亏损成长股更常用。
- MSTR 不能简单用传统PE/PB分析，要看BTC、溢价、融资和资本结构。
- AI高增长股的历史估值区间可能阶段性失效，但失效不等于没有回撤风险。

结论格式：

> **估值压力：低 / 中 / 高。**

---

## 九、期权波动习惯

观察该股历史上 IV 抬升后的行为。

| 观察项 | 判断目的 |
|:---|:---|
| IV 高后是否容易回落？ | 是否适合卖波动 |
| IV 高后股价是否继续破位？ | 高IV是否代表真实风险 |
| 财报前IV是否常被高估？ | 是否有卖波动优势 |
| Put skew 是否常常提前提示下跌？ | 是否尊重保护需求 |
| 周权是否流动性稳定？ | 是否适合7天内卖put |

卖 put 含义：

| 波动习惯 | 动作 |
|:---|:---|
| IV高后常回落，股价稳定 | 可考虑卖put |
| IV高后常继续扩张 | 不把高权利金当机会 |
| Put skew扩大后常下跌 | 尊重风险，降级 |
| 周权流动性差 | 不卖或极小仓 |

结论格式：

> **期权波动习惯：卖方友好 / 中性 / 卖方不友好。**

---

## 十、历史行为与当前动量共振

历史行为画像最终要和当前动量结合。

| 当前动量 | 历史行为 | 共振判断 | 卖 put 动作 |
|:---|:---|:---|:---|
| 上行动量延续 | 历史大涨后常延续 | 正向共振 | 可等回踩卖 |
| 上行动量延续 | 历史大涨后常回吐 | 冲突 | 不追，等确认 |
| 下行动量延续 | 历史跌到均线常反弹 | 冲突 | 等止跌确认 |
| 下行动量延续 | 历史破位后常续跌 | 负向共振 | 暂停 |
| 恐慌后修复 | 历史大跌后常修复 | 正向共振 | 小仓远OTM |
| 震荡消化 | 历史区间内常震荡 | 中性共振 | 低Delta |

共振结论：

| 共振类型 | 含义 |
|:---|:---|
| **正向共振** | 当前动量和历史习惯都支持卖put |
| **中性共振** | 没有明显优势，只能低Delta |
| **冲突** | 当前动量和历史习惯不一致，必须等确认 |
| **负向共振** | 当前动量和历史习惯都不利，暂停卖put |

## 十一、单股分析输出模板

每次分析某只股票，必须输出：

| 模块 | 判断 |
|:---|:---|
| **当前短线动量** | 上行延续 / 上涨衰竭 / 震荡消化 / 下行延续 / 恐慌后修复 |
| **长期均线行为** | 支撑有效 / 支撑一般 / 容易破位 / 当前不可依赖 |
| **大涨后行为** | 趋势延续型 / 容易回吐型 / 需要确认型 |
| **大跌后行为** | 修复型 / 连续下杀型 / 等确认型 |
| **事件行为风险** | 低 / 中 / 高 |
| **季节性位置** | 顺风 / 中性 / 逆风 |
| **估值压力** | 低 / 中 / 高 |
| **期权波动习惯** | 卖方友好 / 中性 / 卖方不友好 |
| **历史与当前是否共振** | 正向共振 / 中性共振 / 冲突 / 负向共振 |
| **卖 put 结论** | 可卖 / 谨慎卖 / 只观察 / 暂停 |

---

## 十二、卖 Put 执行映射

| 共振结果 | 当前动量 | 卖 put 动作 |
|:---|:---|:---|
| 正向共振 | 上行动量延续 | 可等回踩卖，Delta 0.08-0.12 |
| 正向共振 | 恐慌后修复 | 小仓远OTM，Delta 0.05-0.10 |
| 中性共振 | 震荡消化 | 极低Delta，0.05-0.08 |
| 冲突 | 任意状态 | 等确认，不主动卖 |
| 负向共振 | 下行动量延续 | 暂停 |
| 负向共振 | 上涨衰竭 | 暂停或只观察 |

---

## 十三、失效条件

所有分析必须给出失效条件。

| 判断 | 失效条件 |
|:---|:---|
| 长期均线支撑有效 | 放量跌破均线并连续两日收不回 |
| 大涨后延续 | 放量冲高回落并跌破前一日低点 |
| 大跌后修复 | 次日跌破恐慌低点 |
| 季节性顺风 | 当前价格行为明显转弱 |
| 估值压力低 | 估值快速抬升至历史高分位 |
| 期权卖方友好 | Put IV继续扩张且股价破位 |
| 正向共振 | 当前动量转为破位或事件风险升高 |

卖 put 风控触发：

| 触发条件 | 动作 |
|:---|:---|
| Put价格变为开仓价2倍 | 重新评估，通常止损或滚动 |
| Delta从0.08升到0.20以上 | 不再视为远OTM，必须处理 |
| 股价放量跌破关键位 | 暂停新增 |
| 个股弱于行业ETF | 降级 |
| 出现独立负面催化剂 | 暂停 |
| 大盘风向转为逆风 | 降低仓位或暂停 |

---

## 十四、写作要求

必须：

- 结论前置
- 不写新闻摘要
- 不只看单日涨跌
- 当前动量优先
- 历史习惯作为过滤器
- 不用历史支撑对抗当前破位
- 必须比较相对强弱
- 必须判断事件和季节性
- 必须观察估值压力
- 必须观察期权波动习惯
- 必须给出卖 put 结论和失效条件

最终目标：

> 判断某只股票当前短线动量是否与自身历史行为习惯共振，并把结论转化为7天内卖 put 的执行建议。
