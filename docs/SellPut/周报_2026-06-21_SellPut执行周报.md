## 1. 本周市场背景

### 1.1 美股整体：跌一天、修复一天，结构没有坏，但波动来源变多

2026-06-18，美股在前一日 Fed 偏鹰冲击后反弹。AP 统计显示，当日 S&P 500 上涨 1.1% 至 7,500.58，Nasdaq Composite 上涨 1.9% 至 26,517.93，Russell 2000 上涨 2.1% 至 2,979.77。由于 2026-06-19 Juneteenth 休市，6月18日是本周最后一个美股交易日。

这说明短线风险承接仍在，但不是“无风险上行”。真正的变化是：指数被半导体和风险偏好拉回，但利率和美元同时上行，会压制卖 Put 的安全垫。

### 1.2 Fed：不是降息逻辑，而是“可能再加息”的逻辑

Fed 在 2026-06-17 维持联邦基金利率在 3.50%-3.75%，但释放偏鹰信号，市场开始重新定价年内加息可能。Reuters 报道显示，Fed 维持利率不变，但暗示年内可能加息，10年美债收益率一度至约 4.46%，2年期收益率升至 2025年2月以来高位。

对 Sell Put 的含义：

```text
利率不再是风险资产的顺风。
如果下周10Y继续上行，QLD/MSTR/CRCL的近价 Put 风险会明显变大。
```

### 1.3 美元与日元：日元弱到 160 附近，日本加息也没有明显救回来

本周日本央行在 2026-06-16 将政策利率从 0.75% 提高到 1.0%，为约31年来高位。但日元并未明显走强，美元/日元仍在 160 附近徘徊。Reuters 和 MarketWatch 均提到，日元弱势仍引发日本官方干预风险。

这对我们这套策略很重要，因为它会影响三层东西：

1. **美元走强**：压制 EEM、新兴市场、部分风险资产估值。
2. **日元弱势**：市场会盯着日本干预和日债波动，容易触发跨资产去杠杆。
3. **日本利率上行**：如果 JGB 收益率继续上行，全球资金成本和套利交易会被重新定价。

简化成交易语言：

```text
日元/日本利率不是直接决定 QLD、MSTR 的因素，
但它是“全球杠杆是否舒服”的背景变量。
一旦日元突然大幅反弹，可能对应 carry trade unwind，
这时卖 Put 要暂停追单。
```

## 3. 核心池周度评级

### 3.1 QLD：仍是主力，但只等回调，不追涨

QLD 跟随 Nasdaq 100。当前科技和半导体修复强，结构上对 QLD 有利。但 QLD 是 2 倍 ETF，利率上行时会放大回撤。

周度判断：

```text
技术/风向等级：B+
执行权限：可以扫描
交易方式：只卖远虚值 Put，不卖追涨近价 Put
```

下周条件：

- QQQ / Nasdaq 不放量跌破短期均线；
- 10Y 不继续快速冲高；
- QLD 回调时 Put 权利金抬升；
- Delta 优先 0.10-0.15；
- 安全垫优先 8%-12% 以上。

结论：

```text
QLD 是下周第一候选，但不是今天价格越涨越卖。
最好等红盘回落、IV抬升、Bid变厚时做。
```

---

### 3.2 EEM：加入核心池后第一次周度判断，适合“稳健远虚值”，但美元是关键敌人

EEM 的好处是波动低于 MSTR/CRCL，适合做稳健 ETF 收租；坏处是它对美元和美债利率非常敏感。本周美元指数走强、日元弱、亚洲资产压力仍在，所以 EEM 不能因为“看起来稳”就随便卖。

周度判断：

```text
技术/风向等级：B-
执行权限：允许扫描，但必须更保守
交易方式：只在 IV Percentile 明显偏高时，用远 Strike 换安全垫
```

下周条件：

- DXY 不继续突破上行；
- 10Y 不快速上冲；
- EEM 不放量跌破近期支撑；
- Barchart IV Percentile 偏高；
- Strike 必须低于近期箱体下沿或关键支撑区。

结论：

```text
EEM 可以做，但它不是进攻票，是“用高IV换安全垫”的防守票。
美元继续强时，EEM只能观察或卖更远。
```

---

### 3.3 MSTR：BTC代理，仍然只能小仓

MSTR 的核心风险不是普通股票波动，而是 BTC + 公司溢价 + 融资叙事三重波动。本周 BTC 仍有压力，市场利率走高也不利于高波动资产。

周度判断：

```text
技术/风向等级：C+ 至 B-
执行权限：仅顺风小仓
交易方式：不做近价，不做重仓，不和 CRCL 同时放大风险
```

下周条件：

- BTC 不跌破关键支撑；
- IBIT 不出现放量下跌；
- MSTR 自身没有融资/监管/重大新闻冲击；
- 只考虑 Delta 0.10 附近；
- 仓位小于 QLD。

结论：

```text
MSTR 只能作为高权利金小加速，不是本周主仓。
```

---

### 3.4 INTC：半导体强修复，事件驱动改善，但还要防“消息后冲高回落”

本周半导体是最强主线之一。Reuters 报道 2026-06-18 半导体显著上涨，Philadelphia Semiconductor Index 涨 6.4%，并提到 Apple 与 Intel 在美国芯片设计/制造合作消息推动 Intel 上涨 10.6%。

这对 INTC 是明显利好，但它属于“事件驱动修复”，不是已经完全进入长期趋势牛。

周度判断：

```text
技术/风向等级：B
执行权限：观察后扫描
交易方式：只在事件消化后，等回踩支撑再看 Put
```

下周条件：

- 消息后不放量冲高回落；
- 股价回踩不破关键支撑；
- 期权价差不能太宽；
- 不在单日大涨后追卖近价 Put。

结论：

```text
INTC 下周值得重点观察，但更适合“等回调后的结构确认”，不是追涨卖 Put。
```

---

### 3.5 CRCL：仍然最需要谨慎

CRCL 属于 Crypto / Fintech 高波动个股，和 MSTR 在风险因子上有重叠。当前美元强、利率高、BTC不算强顺风时，CRCL 不适合重仓卖 Put。

周度判断：

```text
技术/风向等级：C+
执行权限：谨慎扫描
交易方式：除非权利金极肥且安全垫极大，否则不优先
```

结论：

```text
CRCL 暂时排在 QLD、EEM、INTC、MSTR 之后。
如果已经有 MSTR 风险，就不应再主动加 CRCL。
```

## 5. 下周卖 Put 操作规则

### 5.1 可以卖 Put 的条件

满足以下至少 3 条，才进入期权链比较：

```text
1. SPY / QQQ 不出现放量破位；
2. VIX 不突然上冲；
3. 10Y 美债收益率不快速突破上行；
4. DXY 不继续强势突破；
5. USDJPY 不出现剧烈反转引发 carry unwind；
6. 标的回调但没有破结构；
7. Barchart IV Percentile 明显偏高；
8. Put Bid 抬升，价差可接受，OI 足够。
```

### 5.2 暂停卖 Put 的条件

出现以下任意 2 条，暂停新开仓：

```text
1. QQQ / SPY 同时放量下跌；
2. 10Y 快速冲高并压制成长股；
3. DXY继续突破，EEM/亚洲资产走弱；
4. USDJPY 快速跌破或升破关键位，引发日元干预/套息交易反向波动；
5. BTC 跌破支撑，MSTR/CRCL同步放量下跌；
6. VIX 快速上行但指数尚未充分下跌；
7. 个股重大事件未落地。
```

## 7. AI交易拥挤度与接力真空观察

本周 AI / 半导体仍然是最强主线，半导体指数单日大涨说明资金仍在追逐 AI 硬件链。但这也带来一个问题：

```text
AI硬件继续涨，市场很好看；
但如果AI硬件暂停，谁来接力？
```

当前仍需观察：

- 半导体是否继续强于软件；
- 软件/防御股是否出现补涨；
- 小盘 Russell 2000 的强势是否能持续；
- AI硬件链是否出现高位放量分歧；
- 如果半导体冲高回落，QLD 的 Put 安全垫会迅速变薄。

策略含义：

```text
不能因为半导体一天大涨，就把 QLD / INTC 当成无风险票。
越是拥挤主线，卖 Put 越要等回调，而不是追在兴奋点。
```

## 8. 电子段永平式复盘

本周最重要的不是判断“会不会涨”，而是判断“我愿不愿意接货”。

对当前核心池的本分排序：

```text
QLD：愿意长期轮动接货，但必须接受杠杆ETF波动。
EEM：可以作为稳健分散，但美元强时不要硬做。
INTC：需要确认商业和事件修复，不因一天大涨改变能力圈。
MSTR：懂 BTC 才能做，且只能小仓。
CRCL：看不清监管和估值时，不做也是好决策。
```

Stop Doing List：

```text
1. 不因为权利金肥就跳过市场风向；
2. 不在事件刚刺激大涨后追卖近价 Put；
3. 不同时放大 MSTR 和 CRCL 两个 Crypto 风险；
4. 不在美元强、利率冲高、日元剧烈波动时重仓；
5. 不把“没跌”误判成“安全”。
```
