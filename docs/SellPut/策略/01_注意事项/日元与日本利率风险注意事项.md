# 日元与日本利率风险注意事项

> 本文件用于每日总控中的跨资产风险检查。  
> 日元和日本利率不直接决定单个标的价值，但会影响全球套息交易、美元流动性、美债收益率和高 beta 风险资产。

---

## 一、为什么要单独看日元

过去很长时间，日本低利率是全球 carry trade 的资金来源之一。

当以下情况出现时，全球风险资产可能受到冲击：

```text
日本加息 / 日本长债收益率上行 / 日元快速升值 / 日元极端贬值引发干预预期
```

这些变量会影响：

- 美股科技股；
- AI 与半导体；
- 新兴市场；
- BTC 与高 beta 个股；
- 杠杆 ETF；
- 期权隐含波动率。

---

## 二、每日必须检查的指标

| 指标 | 观察重点 | 风险含义 |
|---|---|---|
| USDJPY | 是否快速上涨或快速下跌 | 日元极端波动可能触发 carry trade 平仓 |
| 日本10Y国债 / JGB | 是否快速上行 | 日本资金成本上升，可能外溢冲击全球债市 |
| 日本30Y / 40Y 国债 | 是否异常上行 | 日本财政与长期利率压力上升 |
| 日本央行表态 | 是否释放加息或缩表信号 | 政策转向可能影响全球流动性 |
| 日本财务省表态 | 是否出现干预汇率信号 | 汇率干预可能引发短线剧烈波动 |
| Nikkei / TOPIX | 日本股市是否剧烈波动 | 亚洲风险偏好信号 |

---

## 三、风险状态分级

| 状态 | 判断 | 对交易的影响 |
|---|---|---|
| 无明显苗头 | USDJPY 与 JGB 稳定 | 不单独降级 |
| 有苗头 | USDJPY 接近极端区间，JGB 上行，日本官员开始口头干预 | 风险等级降一级 |
| 已触发降级 | USDJPY 快速波动，日本长债快速上行，亚洲股市同步走弱 | 高 beta 交易暂停或只观察 |

---

## 四、对不同标的的影响

| 标的 / 资产 | 影响逻辑 |
|---|---|
| QQQ / QLD / TQQQ | 流动性收缩时，成长股和杠杆 ETF 先受压 |
| SMH / SOXX / AI ETF | AI / 半导体拥挤交易可能被平仓 |
| EEM / FXI / KWEB | 美元强与亚洲风险偏好下降会压制新兴市场 |
| BTC / MSTR / IBIT | 风险偏好下降时，Crypto 代理资产波动放大 |
| 高 IV 个股 | 市场风险真实化时，权利金变肥但不一定值得交易 |

---

## 五、降级规则

出现以下任意情况，总控必须降级：

1. USDJPY 快速冲高并接近极端区间；
2. USDJPY 快速下跌，疑似 carry trade 平仓；
3. 日本 10Y / 30Y 国债收益率快速上行；
4. 日本财务省或日本央行释放强烈汇率 / 加息信号；
5. 日经、韩国、台湾等亚洲科技市场同步大跌；
6. 美股盘前同时出现 QQQ、SMH、SOXX 大幅下跌。

---

## 六、交易口径

日元风险触发时：

```text
不因为 Put 权利金变肥而加仓。
不因为大跌后看似便宜而抄底。
先看是否是 carry trade 平仓或全球去杠杆。
```

如果涉及卖 Put：

- Delta 降低；
- Strike 拉远；
- 张数减少；
- 优先等待盘中确认；
- 高 beta 个股暂停；
- 杠杆 ETF 降级。

---

## 七、一句话原则

```text
日元不是标的，但它可能是全球流动性的开关。
```
