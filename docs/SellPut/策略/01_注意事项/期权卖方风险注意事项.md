# 期权卖方风险注意事项

> 本文件只在涉及期权卖方策略时调用。  
> 它不决定标的是否优秀，只判断卖方是否正在承担不合理风险。

---

## 一、核心原则

```text
卖期权不是赚时间价值，而是承担尾部风险并要求足够补偿。
```

任何卖方策略都必须同时满足：

1. 市场环境允许；
2. 技术结构没有破坏；
3. 波动率补偿合理；
4. 流动性足够；
5. 仓位可承受；
6. 愿意接受最坏情景。

---

## 二、必须检查的期权参数

| 参数 | 用途 |
|---|---|
| DTE | 判断时间风险 |
| Strike | 判断安全垫 |
| Bid / Ask / Mid | 判断成交质量 |
| Delta | 判断行权概率和风险距离 |
| IV | 判断市场隐含波动 |
| HV | 判断历史真实波动 |
| IV Rank | 判断当前 IV 相对一年区间的位置 |
| IV Percentile | 判断过去一年有多少交易日 IV 低于当前水平 |
| Expected Move | 判断市场预计波动范围 |
| OI / Volume | 判断流动性 |
| Put/Call | 判断保护需求或方向拥挤 |

---

## 三、卖 Put 特别风险

以下情况不应主动卖 Put：

1. 市场风向为逆风或事件风；
2. 标的跌破关键支撑；
3. 技术结构从回调变成破位；
4. IV 高但来自财报、监管、诉讼、融资、地缘等真实事件；
5. Bid/Ask 价差过大；
6. OI / Volume 太低；
7. 接货金额超过账户承受能力；
8. 与已有仓位风险因子高度重叠；
9. 不愿意按 Strike 接货；
10. 只是因为年化收益率高而心动。

---

## 四、卖 Call / Covered Call 特别风险

以下情况不应盲目卖 Call：

1. 标的刚刚突破并进入主升段；
2. Call 权利金不足以补偿被行权风险；
3. 持仓是长期核心仓，不愿意被卖飞；
4. IV 不高，Call 价格偏低；
5. 临近重大利好事件；
6. 技术结构仍处于强势 C 段。

---

## 五、IV 判断原则

```text
IV 高不等于值得卖。
```

必须区分：

| IV来源 | 含义 | 动作 |
|---|---|---|
| 市场恐慌后波动率抬升 | 可能有补偿 | 等止跌后观察 |
| 财报 / 监管 / 融资 / 诉讼 | 真实事件风险 | 默认降级 |
| 技术破位导致 IV 上升 | 风险正在变真 | 不急于卖 |
| 价格稳定但 IV 偏高 | 较优机会 | 可进入候选 |

---

## 六、仓位原则

```text
先算接货金额，再算年化收益率。
```

必须检查：

- 单张接货金额；
- 保证金占用；
- 已有同类风险因子；
- 最坏情况下是否能继续持有；
- 是否会影响下一次更好的机会。

---

## 七、一句话原则

```text
卖方最怕的不是少赚，而是为了多赚一点权利金，把自己卖给尾部风险。
```
