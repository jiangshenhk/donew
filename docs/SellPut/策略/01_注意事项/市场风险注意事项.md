# 市场风险注意事项

> 本文件用于每日投资总控中的风险预检。  
> 它不判断具体交易动作，只判断当前市场环境是否存在需要降级或暂停的风险。

---

## 一、核心原则

市场风险优先于单个标的机会。

```text
如果市场风向逆转，个股安全垫会快速失效。
```

任何策略分析前，必须先检查市场是否处于以下状态：

- 顺风；
- 横风；
- 逆风；
- 事件风。

---

## 二、必须观察的市场变量

| 变量 | 用途 | 风险含义 |
|---|---|---|
| SPY / S&P 500 | 判断美股整体风险偏好 | 大盘破位时，多数策略降级 |
| QQQ / Nasdaq | 判断科技股与成长股风险偏好 | 影响 QLD、TQQQ、AI、半导体 |
| IWM | 判断市场宽度 | 小盘弱说明风险承接不足 |
| VIX | 判断波动率风险 | 快速上升时，期权卖方风险放大 |
| 10Y / 30Y 美债 | 判断利率压力 | 利率冲高压制成长股、科技股、新兴市场 |
| DXY 美元指数 | 判断美元流动性 | 美元强会压制 EEM、BTC、风险资产 |
| WTI / Brent | 判断油价与通胀风险 | 油价冲高会推升通胀和利率压力 |
| BTC / IBIT | 判断 Crypto 风险偏好 | 影响 MSTR、MARA、RIOT、IBIT |
| SMH / SOXX | 判断半导体与 AI 主线 | 影响 QLD、INTC、SMCI、NVDA |

---

## 三、降级条件

出现以下情况，应将当日整体风险等级至少降一级：

1. QQQ 明显弱于 SPY；
2. SPY、QQQ、IWM 同时走弱；
3. VIX 快速上升；
4. 10Y 美债收益率快速冲高；
5. DXY 快速上行；
6. 油价因地缘或供给冲击快速上涨；
7. 半导体 / AI 主线退潮；
8. BTC 跌破关键支撑；
9. 亚洲或欧洲市场先行大跌；
10. 临近 CPI、FOMC、非农、重大财报或重大地缘事件。

---

## 四、暂停条件

出现以下情况，应暂停新增高风险交易：

```text
市场风向为逆风或事件风。
```

具体包括：

- QQQ / SPY 盘前或盘中大跌；
- VIX 快速跳升；
- SMH / SOXX 放量跌破关键支撑；
- BTC 快速下跌并拖累 Crypto 代理股；
- DXY 与 10Y 同时上行；
- 日元 carry trade 风险触发；
- 大量高 beta 股票同步杀跌。

---

## 五、执行提醒

```text
权利金变肥，不等于风险被补偿。
```

市场逆风时，应优先问：

1. 这是慢跌，还是系统性去杠杆？
2. 权利金增加来自 IV 溢价，还是来自真实破位风险？
3. 当前价格是否已经跌破关键结构？
4. 如果被迫接货，是否愿意持有？
5. 是否存在隔夜或盘后无法处理的风险？

---

## 六、一句话原则

```text
先保市场生存权，再谈策略收益率。
```
