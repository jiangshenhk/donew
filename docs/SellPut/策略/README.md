# 投资策略目录说明

本目录已经重构为通用投资决策架构，不再把所有内容绑定到“卖 Put”。

## 当前结构

```text
00_总控文件/          # 每日投资决策入口，负责调度
01_注意事项/          # 风险提醒与降级条件
02_标的池子/          # 只回答“看什么”，不回答“怎么交易”
03_策略1_市场风向判断/ # 判断市场是否适合承担风险
04_策略2_技术结构判断/ # 判断标的动量、均线、ABC结构
05_策略3_波动率与期权参数判断/ # 判断期权与波动率定价是否合理
99_输出模板/          # 日报、单票分析、交易复盘模板
历史文件夹/           # 旧策略文件归档
```

## 核心原则

```text
标的池只决定看什么。
策略只决定怎么看。
总控才决定做什么。
```

## 使用方式

每日分析从以下文件开始：

```text
00_总控文件/投资策略_每日执行总控.md
```

旧文件已从主目录移出，归档到：

```text
历史文件夹/老策略文件_20260708/
```
