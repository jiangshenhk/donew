# 策略_强势股ABC模块_总控接入规则

本文件规定强势股 ABC 模块在每日 Sell Put 总控中的调用位置。详细判断规则读取：

```text
策略/策略_强势股ABC上涨结构下的SellPut模式.md
```

固定位置：放在“动量与 ABC 技术结构判断文件”之后，“期权链参数判断文件”之前。

执行顺序应理解为：

```text
市场风向判断
动量与 ABC 技术结构判断
强势股 ABC 上涨结构判断
期权链参数判断
```

核心规则：

```text
强势股 ABC 上涨结构判断，是市场风向和普通技术结构之后、期权链判断之前的独立关卡。
```

执行要求：强势高波动标的必须通过本关卡，才可进入期权链分析；普通标的不适用本关卡。
