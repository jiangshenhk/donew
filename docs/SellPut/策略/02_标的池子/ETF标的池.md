## 二、ETF 使用规则

ETF 可以用于四类分析：

1. 大盘风向；
2. 行业主线；
3. 跨资产风险；
4. 交易候选。

但只有满足以下条件时，才可以进入交易候选：

- 市场风向允许；
- 技术结构没有破位；
- 流动性充足；
- 交易参数合格；
- 接货金额或仓位风险可承受；
- 用户愿意持有或承担相关风险。

## 三、ETF 风向观察优先级

每日优先观察：

```text
QQQ / SPY / IWM / SMH / SOXX / EEM / VIX / 10Y / DXY / BTC
```

其中：

- QQQ 影响 QLD / TQQQ；
- SMH / SOXX 影响 INTC / SMCI / AI 主线；
- EEM 受 DXY、10Y、亚洲市场影响；
- IBIT / BTC 影响 MSTR / MARA / RIOT；
- GDX / GLD 反映避险、实际利率与美元压力。

```text
ETF 先用来判断风向，再决定是否进入交易候选。
```
