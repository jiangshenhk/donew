# 核心标的池

> 标的池只回答“看什么”，不回答“怎么交易”。  
> 本文件不绑定买入、卖 Put、Covered Call 或定投。交易动作由总控文件根据策略结果决定。

---

## 一、核心原则

```text
标的池 = universe
策略模块 = 判断方法
总控文件 = 行动决策
```

标的池中的标的可以用于：

- 市场风向观察；
- 技术结构判断；
- 买入或加仓分析；
- 卖 Put 候选分析；
- Covered Call 候选分析；
- 定投节奏判断；
- 风险预警。

## 三、使用规则

1. 标的进入池子，不代表可以买；
2. 标的进入池子，不代表可以卖 Put；
3. 标的进入池子，只代表它值得被跟踪；
4. 每次交易动作必须重新经过总控判断；
5. 如果注意事项触发降级，标的池权限自动降低。

## 四、字段说明

| 字段 | 说明 |
|---|---|
| 标的 | 股票或 ETF 代码 |
| 类型 | ETF、个股、BTC代理、AI观察、行业ETF等 |
| 主要风险因子 | 影响价格的核心变量 |
| 观察用途 | 为什么要看它 |
| 默认状态 | 可分析、条件分析、只观察、暂停、禁用 |
| 备注 | 账户、流动性、接货金额、特殊风险 |
