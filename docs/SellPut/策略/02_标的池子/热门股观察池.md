# 热门股观察池

> 热门股观察池用于跟踪高关注、高波动、高主题弹性的个股。  
> 本文件不绑定买入或卖 Put。所有交易动作必须由总控文件决定。

---

## 一、定位

热门股观察池主要用于：

1. 判断市场风险偏好；
2. 判断主题资金是否扩散；
3. 捕捉高波动标的的潜在机会；
4. 识别不能碰的事件风险。

## 三、使用规则

热门股只在以下情况下进入进一步分析：

- 市场风向不是逆风；
- 标的技术结构没有明显破位；
- 没有重大财报、监管、诉讼、融资事件临近；
- 账户能承受最坏情景；
- 与已有仓位风险因子不高度重叠。
