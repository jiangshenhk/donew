# 策略_强势股ABC上涨结构下的SellPut模式

> 本文件用于补充《操作策略_每日卖Put执行总控.md》和《策略_待卖put的股票池里面各个票根据动量原理和abc结构判断个股的策略.md》。  
> 适用对象：INTC、NVDA、PLTR、CRCL、HOOD、SMCI、MSTR 等出现强趋势、事件催化或资金重新定价的高波动标的。

---

## 一、为什么需要这个模式

普通 Sell Put 框架容易把“高位”直接理解成“风险大、等待、不做”。

但强势股经常会出现：

```text
高位之后还有高位；
超买之后继续超买；
风险提示之后继续上涨。
```

因此，对强势股不能简单使用：

```text
高位 = 风险大 = 不卖 Put
```

而应该改成：

```text
高位 + 弱势 = 不做；
高位 + 强趋势 = 等 B 段回调卖 Put；
高位 + C 段确认 = 已有仓位持有，不追近价；
高位 + 放量破位 = 暂停。
```

核心原则：

```text
A 段确认趋势；
B 段卖 Put；
C 段吃收益；
破 B 段撤退。
```

## 二、强势上涨股的 ABC 结构定义

强势上涨股的 ABC 结构，不是传统下跌反弹里的 ABC，而是上涨趋势中的三段式节奏。

```text
A 段：启动上涨段
B 段：回调 / 横盘消化段
C 段：再次上冲段
```

---

### 2.1 A 段：启动上涨段

A 段通常具备以下特征：

```text
1. 放量大阳线；
2. 突破前高或箱体上沿；
3. 收盘接近高点；
4. 明显强于 QQQ / SPY；
5. 明显强于所属行业，例如 SMH / SOXX；
6. 有消息、财报、政策、评级、产业逻辑等催化；
7. 市场开始重新定价。
```

A 段的交易含义：

```text
方向最强，但 Put 不一定最好卖。
```

原因：

```text
1. 股价刚刚大涨，离技术支撑较远；
2. 远虚值 Put 权利金可能太薄；
3. 近价 Put 容易被正常回调打到；
4. 此时卖 Put 容易变成追高接风险。
```

A 段正确动作：

```text
确认强势，不急卖近价 Put；
标记 A 段高点、低点、中位；
标记突破位和前高；
等待 B 段回踩或横盘消化。
```

---

### 2.2 B 段：回调 / 横盘消化段

B 段是强势股 Sell Put 的核心区域。

健康 B 段通常具备以下特征：

```text
1. 回调缩量；
2. 不跌破 A 段大阳线中位；
3. 不跌破 A 段突破位；
4. 不跌破 B 段低点；
5. 能守住 5 日线 / 10 日线 / 20 日线中的关键支撑；
6. 回调时仍强于 QQQ / SPY / 所属行业；
7. 没有放量长阴线；
8. Put 权利金因为回调变肥，但 Delta 没有失控。
```

B 段交易含义：

```text
股价回调 → Put 权利金变肥；
结构不破 → 风险仍可控；
支撑明确 → Strike 有位置可以放。
```

强势股最佳 Sell Put 位置通常是：

```text
B 段中后段，价格接近支撑但没有跌破支撑；
或者 B 段止跌横盘，准备重新向上。
```

---

### 2.3 C 段：再次上冲段

C 段通常具备以下特征：

```text
1. B 段回调没有破位；
2. 缩量横盘后重新放量；
3. 突破 B 段小平台上沿；
4. 重新站上短期均线；
5. 再次强于 QQQ / SPY / 所属行业；
6. 前高被重新挑战或突破。
```

C 段交易含义：

```text
方向确认更强，但 Put 权利金往往开始变薄。
```

C 段正确动作：

```text
已有 Put：持有，吃 Theta 和趋势修复；
未卖 Put：不追近价，等待下一次回踩；
若 C 段刚确认且尚未加速，可以低 Delta 小仓跟随。
```

## 三、ABC 各阶段 Sell Put 动作表

| 阶段 | 技术特征 | Sell Put 动作 | Delta 建议 | 说明 |
|---|---|---|---|---|
| A 段刚启动 | 放量突破、大阳线、强于板块 | 观察，不追 | 0.05–0.08，仅极小仓 | 确认强势，等待 B 段 |
| A 段连续大涨 | 离均线和支撑较远 | 不卖近价 | 不追 | 防止正常回调打到 Put |
| B 段刚回调 | 第一根或第二根回调阴线 | 先等 | 不急 | 还不知道是健康回调还是派发 |
| B 段缩量接近支撑 | 缩量、不破中位、不破突破位 | 重点卖 Put | 0.08–0.12 | 最佳区域 |
| B 段止跌横盘 | 高位横住、支撑明确 | 可卖 Put | 0.10–0.15 | C 段可能启动 |
| C 段刚突破 | B 段确认后重新上冲 | 小仓跟随 | 0.08–0.12 | 方向确认，但不追近价 |
| C 段加速大涨 | 连续大涨、远离支撑 | 不追 | 不卖近价 | 等下一次回踩 |
| 跌破 B 段低点 | 放量跌破支撑 | 暂停 | 不开新仓 | 强势结构失败 |

一句话：

```text
最好的卖 Put 点 = B 段后半段，不是 A 段刚拉升，也不是 C 段大涨之后。
```

## 四、技术位如何决定 Strike

强势股卖 Put 时，Strike 不能由权利金决定，而应由技术位决定。

优先参考以下技术位：

```text
第一：B 段低点；
第二：A 段突破位；
第三：A 段大阳线中位；
第四：10 日线；
第五：20 日线；
第六：前高回踩位 / 箱体上沿；
第七：0.5–1 个 ATR 下方。
```

理想 Put Strike 应该放在：

```text
关键支撑下方一档；
或者支撑下方 2%–5%；
或者 0.5–1 个 ATR 下方。
```

核心原则：

```text
把 Strike 放到市场证明你错的位置下面。
```

如果股价跌到 Strike，通常意味着：

```text
1. B 段支撑已经被打穿；
2. 强趋势可能已经失效；
3. 需要重新判断是否真的愿意接货；
4. 不能只因为权利金肥就继续滚动或加仓。
```

## 五、强趋势模式开启条件

满足多数条件时，标的进入“强趋势 Sell Put 模式”：

```text
1. 股价明显强于 QQQ / SPY；
2. 股价明显强于所属行业；
3. 出现放量突破；
4. 站上 5 日 / 10 日 / 20 日均线；
5. 回调时缩量；
6. 消息面或基本面催化仍在；
7. 没有跌破 A 段突破位；
8. 没有跌破 B 段低点；
9. Put IV 仍有溢价，但 Delta 没有失控；
10. 用户愿意在支撑下方接货。
```

只要强趋势模式开启，就不能因为“高位”两个字直接排除。

正确表达应该是：

```text
强趋势股，不是不做；
而是只在 B 段健康回调或 B 段止跌横盘时做。
```

## 六、强势股 Sell Put 提示机制

### 6.1 强势启动提示

当 A 段刚启动时，提示：

```text
标的进入强趋势观察，不因高位直接排除。
等待 B 段健康回调，重点观察 5 日线、10 日线、突破位和 A 段中位。
```

动作：

```text
不急着卖近价 Put，只标记技术位。
```

---

### 6.2 B 段回踩中提示

当股价进入回调但尚未确认健康时，提示：

```text
标的正在进入 B 段回调，开始接近可卖 Put 区域。
需要观察是否缩量、是否守住关键支撑、Put 权利金是否变肥。
```

动作：

```text
开始看期权链，但不急下单。
```

---

### 6.3 B 段健康确认提示

当 B 段健康回调确认时，提示：

```text
B 段健康回调确认，适合进入 Sell Put 筛选。
优先选择支撑位下方 Strike，Delta 0.08–0.12；强势非常明确时可放宽到 0.15。
```

动作：

```text
重点提示可以卖 Put。
```

---

### 6.4 C 段启动提示

当 C 段刚启动时，提示：

```text
C 段启动，已有 Put 可以继续持有。
新开 Put 不追近价，只考虑低 Delta 或等待下一次回踩。
```

动作：

```text
已有仓位继续吃收益；新增仓位谨慎。
```

---

### 6.5 结构失败提示

当跌破 B 段低点或放量跌破关键支撑时，提示：

```text
跌破 B 段低点 / 放量跌破关键支撑，强趋势结构失败。
暂停卖 Put，不因权利金变肥而接刀。
```

动作：

```text
停止新开仓，等待新结构形成。
```

## 七、强势股模式下的最终判断模板

每日分析强势股时，必须回答以下问题：

```text
1. 当前是否属于强趋势股？
2. 当前处于 A 段、B 段，还是 C 段？
3. 如果是 B 段，是健康回调还是派发转弱？
4. B 段低点在哪里？
5. A 段突破位在哪里？
6. A 段大阳线中位在哪里？
7. 当前是否强于 QQQ / SPY / 行业？
8. 成交量是缩量回调，还是放量下跌？
9. 哪个 Strike 位于关键支撑下方？
10. 如果跌到 Strike，我是否愿意接货？
11. 当前应该卖 Put、观察，还是暂停？
```

输出时可以使用以下格式：

```text
强势股 ABC 判断：

标的：
是否强趋势：是 / 否
当前阶段：A 段 / B 段 / C 段 / 结构失败
A 段高点：
A 段低点：
A 段中位：
B 段低点：
突破位：
当前支撑：
当前压力：
成交量状态：
相对强弱：
建议动作：卖 Put / 观察 / 暂停
建议 Delta：
建议 Strike 区域：
失效条件：
```

## 八、与总控文件的衔接规则

本模式不替代总控文件，而是作为强势股的额外判断层。

执行顺序为：

```text
1. 先按总控文件判断市场风向；
2. 再按 ABC 技术结构文件判断趋势位置；
3. 如果标的属于强趋势股，则调用本文件；
4. 若本文件判断处于 B 段健康回调或 B 段止跌横盘，才进入期权链筛选；
5. 若处于 A 段启动或 C 段加速，原则上不追近价 Put；
6. 若跌破 B 段低点，直接暂停。
```

最终冲突处理：

```text
市场风向 C/D > 本模式；
技术结构破位 > 本模式；
一票否决规则 > 本模式；
权利金和年化收益率不能反过来覆盖结构失败。
```

---

## 九、最终原则

```text
强势上涨股，不因高位直接排除；
但只在 B 段健康回调或 B 段止跌横盘时卖 Put。
A 段不追，C 段不追高，跌破 B 段低点暂停。
```

最简口诀：

```text
A 段确认趋势，B 段卖 Put，C 段吃收益，破 B 段撤退。
```
