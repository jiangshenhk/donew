# 策略2_动量原理与ABC结构判断

> 本策略只判断标的的技术结构，不绑定买入、卖 Put 或卖 Call。  
> 它回答：当前标的是上涨、回调、修复、破位，还是震荡？

---

## 一、策略目标

通过动量、均线、成交量和 ABC 结构，判断某个标的当前是否处于健康结构。

本策略不直接输出交易动作，只输出：

```text
技术结构等级：A / B / C / D
```

交易动作由总控文件结合市场风向、注意事项和交易类型决定。

---

## 二、核心观察项

| 项目 | 说明 |
|---|---|
| 当前位置 | 当前处在 A点附近、A后弱弹、B到C回踩、C点观察、C点确认、突破B、破位反抽、底部2B、强趋势回踩等哪一类位置 |
| 趋势方向 | 上涨、横盘、下跌、修复 |
| 均线结构 | 5日、10日、20日、50日、90日位置关系 |
| ABC结构 | 当前处于 A 段、B 段、C 段，还是结构不清 |
| 123法则 | 趋势线是否被突破或跌破，是否不再创新高/新低，是否突破前高或跌破前低 |
| 2B法则 | 是否出现假突破或假跌破，尤其是跌破前低后快速收回，或突破前高后快速跌回 |
| 成交量 | 放量上涨、缩量回调、放量下跌 |
| 相对强弱 | 强于 QQQ / SPY / 行业，还是弱于 |
| 支撑位 | 最近关键支撑区 |
| 压力位 | 最近关键压力区 |
| 失效位 | 跌破哪里说明结构失败 |

---

## 三、技术结构等级

| 等级 | 状态 | 含义 |
|---|---|---|
| A | 正向共振 | 趋势健康，回调可控，强于大盘或行业 |
| B | 修复共振 | 有修复迹象，但尚未完全确认 |
| C | 中性震荡 | 方向不清，适合观察或等待更好位置 |
| D | 负向破位 | 技术结构破坏，不适合承担新风险 |

---

## 四、ABC结构定义

| 阶段 | 含义 | 判断重点 |
|---|---|---|
| A段 | 第一段启动或主升 | 动量是否真实、是否放量、是否突破关键位 |
| B段 | 回调或整理 | 是否缩量、是否守住关键均线、是否健康回踩 |
| C段 | 第二段上涨或延续 | 是否突破前高、是否强于大盘、是否过热 |
| C段后回撤 | 高位回撤验证 | 是否守住 EMA5 / EMA10 / EMA20 |
| 结构失败 | 跌破关键支撑 | 不再用原上涨结构解释 |

### 1. 低位ABC结构

低位ABC用于判断下跌后是否开始修复。

| 点位 | 含义 | 判断重点 |
|---|---|---|
| A | 阶段低点 | 只是低点，不等于底部确认 |
| B | 第一轮反弹高点 | 观察反弹力度和压力区 |
| C | 回踩低点 | 必须高于A，且最好缩量企稳 |
| 突破B | 反转确认 | 更安全，但卖Put权利金通常会变薄 |

低位ABC的关键不是抄A点，而是观察：

```text
A点出现
↓
B段反弹形成压力
↓
回踩到C，C高于A
↓
C点附近缩量企稳并重新转强
↓
突破B后，低位ABC确认
```

### 2. 高位ABC结构

高位ABC用于判断上涨趋势是否可能结束。

| 点位 | 含义 | 判断重点 |
|---|---|---|
| A | 阶段高点 | 不等于顶部确认 |
| B | 第一轮回调低点 | 判断回调是否破坏趋势 |
| C | 反弹次高点 | 如果低于A，说明动能减弱 |
| 跌破B | 转弱确认 | 高位ABC成立，结构降级 |

高位ABC的关键不是看见回调就判空，而是必须等：

```text
A点出现
↓
跌到B
↓
反弹到C，但C低于A
↓
再跌破B
↓
高位ABC转弱确认
```

## 五、当前位置标签系统

每次分析必须先判断“当前价格处在什么位置”，再判断技术等级。

| 当前位置标签 | 含义 | 技术含义 | 初步等级倾向 |
|---|---|---|---|
| A点附近 | 刚跌到阶段低点 | 看似便宜，但还不知道是否继续破位 | C / D |
| A点后弱弹 | 从低点弹起一点，但B点未清晰形成 | 只是弱修复，不能当反转 | C |
| B点压力区 | 第一轮反弹到压力区 | 价格变高，容易遇阻回落 | C |
| B到C回踩途中 | 从B往C回踩，但C未确认 | 不要猜底，可能继续跌破A | C / D |
| C点观察区 | 回踩接近支撑，暂未跌破A | 候选区域，需要等企稳证据 | B / C |
| C点确认转强 | C高于A，缩量企稳，开始站回短均线 | 较好的技术位置 | A / B |
| 突破B | 价格突破B点或前高 | 结构确认度更高，但可能短线过热 | A / B |
| 破位反抽 | 跌破前低后反弹，但站不回原支撑 | 支撑变压力，偏空延续 | D |
| 底部2B | 跌破前低后快速收回 | 假跌破，进入重点观察 | B / C |
| 顶部2B | 突破前高后快速跌回 | 假突破，风险上升 | C / D |
| 强趋势回踩 | 上升趋势中回踩EMA10/20/50但不破结构 | 健康回调候选 | A / B |
| 结构不清 | 没有明确前高前低或区间 | 等待更清楚信号 | C |

核心原则：

```text
先定位，再判断。
位置错了，指标再多也没用。
```

## 六、123法则

123法则用于确认趋势是否真的发生变化。它比单纯看一根K线更稳健。

### 1. 上涨转弱的123法则

上涨趋势可能转弱时，通常满足：

| 步骤 | 信号 | 含义 |
|---|---|---|
| 1 | 上升趋势线被跌破 | 原上涨节奏被破坏 |
| 2 | 反弹不再创新高 | 多头动能下降 |
| 3 | 再跌破前低 | 转弱确认 |

判断含义：

```text
趋势线跌破
↓
反弹不创新高
↓
跌破前低
↓
高位ABC或结构失败确认
```

### 2. 下跌转强的123法则

下跌趋势可能转强时，通常满足：

| 步骤 | 信号 | 含义 |
|---|---|---|
| 1 | 下降趋势线被突破 | 原下跌节奏被破坏 |
| 2 | 回踩不再创新低 | 卖压减弱 |
| 3 | 再突破前高 | 转强确认 |

判断含义：

```text
下降趋势线突破
↓
回踩不创新低
↓
突破前高
↓
低位ABC或修复共振确认
```

### 3. 123法则与ABC的对应关系

| 123法则 | ABC结构里的含义 |
|---|---|
| 前高 | B点或压力确认点 |
| 前低 | A点或支撑确认点 |
| 不再创新低 | C点高于A |
| 不再创新高 | 高位C点低于A点/前高 |
| 突破前高 | 低位ABC确认增强 |
| 跌破前低 | 高位ABC或结构失败确认 |

一句话：

```text
ABC负责画结构，123负责确认趋势。
```

## 七、2B法则

2B法则用于识别假突破和假跌破。它比123法则更灵敏，但假信号也更多。

### 1. 底部2B

| 条件 | 含义 |
|---|---|
| 价格跌破前低 | 市场尝试继续杀跌 |
| 跌破后快速收回前低上方 | 空头杀不动，假跌破 |
| 收回后不再快速跌破 | 卖压衰减，进入观察 |

判断含义：

```text
跌破前低
↓
快速收回
↓
底部2B观察
↓
再看是否形成C点确认或低位ABC
```

底部2B不是立即确认反转，只是说明：

```text
继续下跌的力量可能减弱，可以进入重点观察。
```

### 2. 顶部2B

| 条件 | 含义 |
|---|---|
| 价格突破前高 | 市场尝试继续上攻 |
| 突破后快速跌回前高下方 | 追高资金被套，假突破 |
| 回落时放量 | 风险更高 |

判断含义：

```text
突破前高
↓
快速跌回
↓
顶部2B
↓
降级观察或判断高位风险
```

### 3. 2B与123法则的分工

| 方法 | 特点 | 用途 |
|---|---|---|
| 2B法则 | 更早、更灵敏 | 发现假突破/假跌破，进入观察名单 |
| 123法则 | 更慢、更稳健 | 确认趋势是否真正反转 |
| ABC结构 | 更直观 | 标注当前位置和结构阶段 |

一句话：

```text
2B负责提前预警，123负责确认趋势，ABC负责定位结构。
```

## 八、均线判断

| 状态 | 含义 |
|---|---|
| 价格在 5/10/20/50/90 日均线上方 | 强势或趋势健康 |
| 价格跌破 5日但守住10/20日 | 短线回调，需观察 |
| 价格跌破20日 | 趋势降级 |
| 价格跌破50日且无法修复 | 中期结构转弱 |
| 价格跌破90日 | 大级别趋势风险 |

## 十、常见结构处理

### 1. 强势股高位回撤

不能直接看作买点或卖期权机会。必须判断：

- 是否守住 EMA5；
- 是否守住 EMA10；
- 是否守住 EMA20；
- 是否放量下跌；
- 是否行业同步转弱。

如果高位回撤守住 EMA20 或 EMA50，并且回调缩量、相对强弱仍好，可以归类为：

```text
强趋势回踩
```

如果跌破关键前低，且反抽站不回，则归类为：

```text
破位反抽 / 结构失败
```

---

### 2. B 到 C 转折

这是较好的候选区域，但必须满足：

- B段回调缩量；
- 支撑位有效；
- C点高于A点；
- MACD / RSI / KDJ 有修复；
- 价格开始重新站回 EMA5 / EMA10；
- 相对强弱没有明显落后；
- 市场风向不是逆风。

注意：

```text
B到C下跌途中不要猜底。
真正有价值的是C点确认，而不是下跌过程中主观认为快到底了。
```

---

### 3. 假突破

如果突破后快速跌回前高下方，且放量下跌，应降级。

对应标签：

```text
顶部2B / 假突破风险
```

---

### 4. 假跌破

如果跌破前低后快速收回，且之后不再快速跌破，可以进入重点观察。

对应标签：

```text
底部2B观察
```

但底部2B不是立即转强，后续仍需观察：

- 是否站回前低上方；
- 是否形成C点；
- 是否缩量回踩不破；
- 是否重新突破B点或关键压力。

---

## 十一、一句话原则

```text
技术结构只负责判断位置和趋势，不负责决定交易动作。
```

```text
ABC负责定位，2B负责预警，123负责确认，均线和动量负责验证强弱。
```

```text
先判断当前在哪里，再判断结构等级，最后才交给总控决定是否交易。
```
