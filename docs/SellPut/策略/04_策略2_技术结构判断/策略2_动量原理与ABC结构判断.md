# 策略2_动量原理与ABC结构判断

> 本策略只判断标的的技术结构，不绑定买入、卖 Put 或卖 Call。  
> 它回答：当前标的是上涨、回调、修复、破位，还是震荡？

---

## 一、策略目标

通过动量、均线、成交量和 ABC 结构，判断某个标的当前是否处于健康结构。

本策略不直接输出交易动作，只输出：

```text
技术结构等级：A / B / C / D
```

交易动作由总控文件结合市场风向、注意事项和交易类型决定。

---

## 二、核心观察项

| 项目 | 说明 |
|---|---|
| 趋势方向 | 上涨、横盘、下跌、修复 |
| 均线结构 | 5日、10日、20日、50日、90日位置关系 |
| ABC结构 | 当前处于 A 段、B 段、C 段，还是结构不清 |
| 成交量 | 放量上涨、缩量回调、放量下跌 |
| 相对强弱 | 强于 QQQ / SPY / 行业，还是弱于 |
| 支撑位 | 最近关键支撑区 |
| 压力位 | 最近关键压力区 |
| 失效位 | 跌破哪里说明结构失败 |

---

## 三、技术结构等级

| 等级 | 状态 | 含义 |
|---|---|---|
| A | 正向共振 | 趋势健康，回调可控，强于大盘或行业 |
| B | 修复共振 | 有修复迹象，但尚未完全确认 |
| C | 中性震荡 | 方向不清，适合观察或等待更好位置 |
| D | 负向破位 | 技术结构破坏，不适合承担新风险 |

---

## 四、ABC结构定义

| 阶段 | 含义 | 判断重点 |
|---|---|---|
| A段 | 第一段启动或主升 | 动量是否真实、是否放量、是否突破关键位 |
| B段 | 回调或整理 | 是否缩量、是否守住关键均线、是否健康回踩 |
| C段 | 第二段上涨或延续 | 是否突破前高、是否强于大盘、是否过热 |
| C段后回撤 | 高位回撤验证 | 是否守住 EMA5 / EMA10 / EMA20 |
| 结构失败 | 跌破关键支撑 | 不再用原上涨结构解释 |

---

## 五、均线判断

| 状态 | 含义 |
|---|---|
| 价格在 5/10/20/50/90 日均线上方 | 强势或趋势健康 |
| 价格跌破 5日但守住10/20日 | 短线回调，需观察 |
| 价格跌破20日 | 趋势降级 |
| 价格跌破50日且无法修复 | 中期结构转弱 |
| 价格跌破90日 | 大级别趋势风险 |

---

## 六、输出格式

```markdown
## 技术结构判断：XXX

- 趋势方向：上涨 / 横盘 / 回调 / 修复 / 下跌
- 均线结构：xxx
- ABC阶段：A段 / B段 / C段 / C段后回撤 / 结构失败 / 不清晰
- 成交量状态：放量上涨 / 缩量回调 / 放量下跌 / 普通量
- 相对强弱：强于 / 弱于 QQQ、SPY、行业ETF
- 关键支撑：xxx
- 关键压力：xxx
- 失效条件：xxx
- 技术结构等级：A / B / C / D
- 是否进入下一步交易参数分析：是 / 否
```

---

## 七、常见结构处理

### 1. 强势股高位回撤

不能直接看作买点或卖期权机会。必须判断：

- 是否守住 EMA5；
- 是否守住 EMA10；
- 是否守住 EMA20；
- 是否放量下跌；
- 是否行业同步转弱。

---

### 2. B 到 C 转折

这是较好的候选区域，但必须满足：

- B段回调缩量；
- 支撑位有效；
- MACD / RSI / KDJ 有修复；
- 相对强弱没有明显落后；
- 市场风向不是逆风。

---

### 3. 假突破

如果突破后快速跌回前高下方，且放量下跌，应降级。

---

## 八、一句话原则

```text
技术结构只负责判断位置和趋势，不负责决定交易动作。
```
