# 策略3_波动率与期权参数判断

> 本策略只判断波动率和期权参数是否合理。  
> 它不直接决定是否卖 Put 或卖 Call；最终动作由总控文件决定。

---

## 一、策略目标

判断当前期权价格是否合理补偿了风险。

核心问题：

```text
当前权利金是合理风险补偿，还是风险正在变真？
```

---

## 二、适用场景

本策略适用于：

- 卖 Put；
- Covered Call；
- 买入保护性 Put；
- 判断 IV 是否过高或过低；
- 判断市场是否正在重新定价风险。

---

## 三、必须读取的参数

| 参数 | 说明 |
|---|---|
| 当前股价 | 判断 Strike 与现价距离 |
| DTE | 剩余到期日 |
| Strike | 行权价 |
| Bid / Ask / Mid | 成交价格与价差 |
| Delta | 风险距离和概率口径；自动扫描结果中的 Delta 可能为估算值，交易前需确认券商实时值 |
| IV | 当前隐含波动率 |
| HV | 历史波动率 |
| IV Rank | 当前 IV 在一年区间中的位置 |
| IV Percentile | 过去一年中低于当前 IV 的交易日占比 |
| Expected Move | 市场预期波动范围 |
| OI / Volume | 流动性 |
| 安全垫 | Strike 与当前价格的距离 |
| 权利金后盈亏平衡价 | `Strike - Mid`，表示计入收到权利金后的实际接货成本 |
| 周线 ATR(14) | 最近 14 周真实波幅均值，用于衡量价格距离是否足以覆盖标的正常周波动 |
| 行权价 ATR 倍数 | `(当前股价 - Strike) / 周线ATR(14)`，仅衡量行权价距离 |
| 盈亏平衡 ATR 倍数 | `(当前股价 - 盈亏平衡价) / 周线ATR(14)`，作为 Sell Put 的主要安全距离指标 |
| 年化收益率 | 权利金相对保证金或接货金额的年化 |

---

## 四、IV判断原则

| 状态 | 判断 | 含义 |
|---|---|---|
| IV高，技术结构健康 | 可进一步分析 | 可能存在合理补偿 |
| IV高，标的破位 | 风险正在变真 | 不应只看权利金 |
| IV低，市场顺风 | 收益不足 | 不急于卖方 |
| IV暴涨，事件临近 | 真实风险 | 默认降级 |
| IV高于HV | 可能有溢价 | 需结合事件和技术结构 |

---

## 五、Delta 与周线 ATR 联合判断

### 1. 核心原则

对于 7—10 DTE 的 Sell Put，不能只看 Delta 是否接近 0.15，还要判断计入权利金后的盈亏平衡价距离，是否足以覆盖标的正常周波动。

默认使用：

```text
盈亏平衡价 = Strike - Mid
盈亏平衡 ATR 倍数 = (当前股价 - 盈亏平衡价) / 周线ATR(14)
```

`盈亏平衡 ATR 倍数`是主要指标；`行权价 ATR 倍数`仅作为辅助观察。

### 2. 联合判断表

| Delta | 盈亏平衡 ATR 倍数 | 判断 |
|---|---:|---|
| ≤0.15 | ≥1.5 | 较理想，可进入 ABC 结构、动量、事件风险和流动性判断 |
| ≤0.15 | 1.0—1.5 | 一般，需要 ABC 止跌、动量改善或明确支撑确认 |
| ≤0.15 | <1.0 | Delta 看似安全，但实际价格距离不足，原则上不因低 Delta 单独开仓 |
| 0.15—0.20 | ≥1.5 | 可以观察，但需要更强技术结构确认并降低仓位 |
| 0.15—0.20 | <1.5 | 风险偏高，通常降级为观察 |
| >0.20 | ≥1.5 | 距离虽然较远，但需排查高 IV、财报、消息或跳空风险为何导致 Delta 偏高 |
| >0.20 | <1.0 | 偏激进，通常不符合稳健 Sell Put 逻辑 |

### 3. 使用限制

```text
ATR 只衡量历史正常波动，不覆盖财报、监管、诉讼、地缘政治和流动性崩溃造成的跳空。
```

因此即使 `盈亏平衡 ATR 倍数 ≥ 1.5`，只要出现以下情况，仍应降级：

- 财报或重大产品发布临近；
- 标的连续破位或出现加速下跌；
- IV 突然暴涨且原因不明；
- Bid / Ask 价差过宽；
- OI、Volume 太低；
- 市场风向或注意事项模块触发红灯。

---

## 七、等级判断

| 等级 | 条件 | 含义 |
|---|---|---|
| A | IV补偿合理，流动性好，价差合理，盈亏平衡 ATR 倍数通常 ≥1.5，且无明显事件风险 | 可进入总控最终决策 |
| B | 基本合理，但 ATR 距离为 1.0—1.5 或存在小瑕疵 | 小仓或选择更保守参数 |
| C | 盈亏平衡 ATR 倍数 <1.0、性价比一般、Delta 与距离冲突或数据不足 | 观察 |
| D | 流动性差、风险过高、事件风险明显或结构处于加速破位 | 不交易 |

---

## 八、特别提醒

```text
年化收益率不是第一指标。
低 Delta 也不等于实际距离足够安全。
```

优先级应为：

1. 风险是否真实可控；
2. 是否愿意承担最坏情景；
3. 技术结构是否止跌或保持健康；
4. 盈亏平衡价是否至少远离约 1.5 倍周线 ATR；
5. IV 是否补偿风险；
6. 流动性是否允许成交和退出；
7. 年化收益率是否合理。

---

## 九、一句话原则

```text
先用盈亏平衡 ATR 倍数检查真实价格距离，再用 Delta 检查概率口径；两者不能互相替代。
```
