# 策略3_波动率与期权参数判断

> 本策略只判断波动率和期权参数是否合理。  
> 它不直接决定是否卖 Put 或卖 Call；最终动作由总控文件决定。

---

## 一、策略目标

判断当前期权价格是否合理补偿了风险。

核心问题：

```text
当前权利金是合理风险补偿，还是风险正在变真？
```

---

## 二、适用场景

本策略适用于：

- 卖 Put；
- Covered Call；
- 买入保护性 Put；
- 判断 IV 是否过高或过低；
- 判断市场是否正在重新定价风险。

---

## 三、必须读取的参数

| 参数 | 说明 |
|---|---|
| 当前股价 | 判断 Strike 与现价距离 |
| DTE | 剩余到期日 |
| Strike | 行权价 |
| Bid / Ask / Mid | 成交价格与价差 |
| Delta | 风险距离和概率口径 |
| IV | 当前隐含波动率 |
| HV | 历史波动率 |
| IV Rank | 当前 IV 在一年区间中的位置 |
| IV Percentile | 过去一年中低于当前 IV 的交易日占比 |
| Expected Move | 市场预期波动范围 |
| OI / Volume | 流动性 |
| 安全垫 | Strike 与当前价格的距离 |
| 年化收益率 | 权利金相对保证金或接货金额的年化 |

---

## 四、IV判断原则

| 状态 | 判断 | 含义 |
|---|---|---|
| IV高，技术结构健康 | 可进一步分析 | 可能存在合理补偿 |
| IV高，标的破位 | 风险正在变真 | 不应只看权利金 |
| IV低，市场顺风 | 收益不足 | 不急于卖方 |
| IV暴涨，事件临近 | 真实风险 | 默认降级 |
| IV高于HV | 可能有溢价 | 需结合事件和技术结构 |

---

每个候选档位必须补一句解释：

```text
该档位为什么可以 / 为什么不可以。
```

---

## 六、等级判断

| 等级 | 条件 | 含义 |
|---|---|---|
| A | IV补偿合理，流动性好，价差合理，安全垫充足 | 可进入总控最终决策 |
| B | 基本合理，但有小瑕疵 | 小仓或更保守参数 |
| C | 性价比一般或数据不足 | 观察 |
| D | 流动性差、风险过高、事件风险明显 | 不交易 |

---

## 七、特别提醒

```text
年化收益率不是第一指标。
```

优先级应为：

1. 风险是否真实可控；
2. 是否愿意承担最坏情景；
3. IV 是否补偿风险；
4. 流动性是否允许成交和退出；
5. 年化收益率是否合理。

---

## 八、一句话原则

```text
期权参数只负责判断价格是否补偿风险，不负责替你忽略风险。
```
