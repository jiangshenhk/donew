# 策略1_市场风向与跨资产环境判断

> 本策略只判断市场环境，不绑定任何具体交易动作。  
> 它回答：当前市场是顺风、横风、逆风，还是事件风？

---

## 一、策略目标

判断未来 1–7 个交易日，当前市场环境是否适合承担风险资产敞口。

风险资产敞口包括但不限于：

- 买入股票或 ETF；
- 增加定投权重；
- 卖 Put；
- 持有高 beta 仓位；
- 持有杠杆 ETF；
- 持有 AI、半导体、BTC 相关资产。

---

## 二、风向等级

| 等级 | 状态 | 定义 | 行动含义 |
|---|---|---|---|
| A | 顺风 | 指数稳，VIX稳定或回落，10Y/DXY/油价不冲击，主线扩散 | 可进入具体策略分析 |
| B | 横风 | 指数高位震荡，部分变量有压力，但未系统性恶化 | 小仓、保守、等待确认 |
| C | 逆风苗头 | 风险变量上升，主线分化，波动率抬升 | 观察或降级 |
| D | 逆风 / 事件风 | 指数破位，VIX跳升，利率/美元/日元/油价冲击，重大事件临近 | 暂停新增高风险动作 |

---

## 三、必须观察的变量

| 变量 | 判断问题 |
|---|---|
| SPY | 美股整体是否稳定？ |
| QQQ | 科技与成长是否强于大盘？ |
| IWM | 小盘是否参与，市场宽度是否健康？ |
| VIX | 波动率是否升温？ |
| 10Y / 30Y | 利率是否压制成长股？ |
| DXY | 美元是否收紧全球流动性？ |
| USDJPY / JPY | 是否存在 carry trade 平仓或汇率干预风险？ |
| 日本10Y / JGB | 日本利率是否外溢冲击全球债市？ |
| WTI / Brent | 油价是否带来通胀压力？ |
| SMH / SOXX | 半导体和 AI 主线是否健康？ |
| BTC / IBIT | Crypto 是否确认 risk-on？ |
| EEM / FXI / KWEB | 新兴市场、中国资产、亚洲风险偏好是否支持？ |

---

## 四、市场真正交易什么

每日必须用一句话回答：

```text
今天市场真正交易的是：xxx。
```

可选主线包括：

- 利率压力；
- 降息预期；
- 美元流动性；
- 日元 / carry trade 平仓；
- AI硬件主线；
- 半导体扩散或退潮；
- BTC risk-on / risk-off；
- 油价 / 通胀冲击；
- 地缘风险；
- 财报交易；
- 期权 / gamma 驱动；
- 月末 / 季末调仓。

---

## 五、输出格式

```markdown
## 市场风向判断

- 今日市场风向：A / B / C / D
- 风向状态：顺风 / 横风 / 逆风苗头 / 逆风或事件风
- 今天市场真正交易的是：xxx
- 今天真正说真话的是：VIX / 10Y / DXY / USDJPY / SMH / BTC / 其他
- 主要有利因素：xxx
- 主要不利因素：xxx
- 是否进入技术结构分析：是 / 否
- 是否需要降低风险参数：是 / 否
```

---

## 六、评分表

| 项目 | 评分 | 判断 |
|---|---:|---|
| 短期风向稳定度 | x / 10 | 是否适合承担 1–7 天风险 |
| 波动率风险 | x / 10 | VIX / IV 是否可能突升 |
| 市场宽度 | x / 10 | 是否只有少数股票硬撑 |
| 宏观冲击风险 | x / 10 | 利率、美元、日元、油价是否冲击 |
| 事件跳空风险 | x / 10 | 是否临近重大事件 |
| 风险资产友好度 | x / 10 | 是否支持承担风险敞口 |

---

## 七、降级规则

出现以下情况，必须降级：

- QQQ / SPY / IWM 同时走弱；
- VIX 快速上行；
- 10Y 与 DXY 同时上行；
- USDJPY 或 JGB 触发日元风险；
- SMH / SOXX 明显转弱；
- BTC 快速下跌；
- 亚洲或欧洲先行大跌；
- 重大事件未落地。

---

## 八、一句话原则

```text
市场风向不是预测涨跌，而是判断今天能不能承担风险。
```
