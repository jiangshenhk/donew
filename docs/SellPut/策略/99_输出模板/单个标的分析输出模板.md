# 单个标的分析输出模板

> 本模板用于分析单个股票、ETF、BTC相关标的或其他可交易资产。  
> 它不预设交易动作，先分析市场、结构、风险，再给出可能动作。  
> 核心要求：先判断当前位置和结构图匹配，再判断技术等级，最后才进入交易动作。

### 4. 图像结构图

如果环境支持图像生成，应在文字结构图之外补充图像版结构示意图。

图像版结构图最低要求：

- 标出当前是否适合ABC；
- 如果适合ABC，标出 A / B / C 和当前价所在位置；
- 如果不适合ABC，明确标注“不适合ABC”，并画出更适合的结构图；
- 标出主路径和备选路径；
- 标出升级条件、降级条件、失效条件；
- 如果涉及卖Put，应在图中写明“只是技术候选，不等于直接交易”。

图像说明建议使用：

```text
在标准ABC模板图上，用醒目的圆点标注当前可能位置，并说明：
当前位置①：B→C回踩途中，谨慎猜底，下冲概率仍大；
当前位置②：接近C点观察区，开始判断是否企稳；
当前位置③：C点确认转强，转折概率更大，可继续看是否突破B。
```
