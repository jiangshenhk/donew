# Sell Put 模拟盘

> 用途：记录自动筛选后的纸上模拟交易，只用于复盘策略质量，不代表真实下单指令。

## 规则

- 每个交易日最多自动模拟卖出 1 张 Put。
- 只从 `sell_put_delta_alerts_YYYY-MM-DD.csv` 中选择符合公共指标的候选。
- 默认使用候选表中的 `Mid = (Bid + Ask) / 2` 作为模拟成交价。
- 重复运行同一天脚本时，不重复记录同一笔 `日期 + 标的 + 到期日 + 行权价 + Put`。
- 模拟盘只验证策略，不触发真实交易，不替代富途成交前复核。
