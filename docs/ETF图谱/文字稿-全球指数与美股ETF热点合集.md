## 开场：为什么先看指数，再看ETF？

很多人买ETF时，第一反应是看过去涨幅、看排行榜、看谁收益最高。

但长期投资的顺序应该反过来：

> 先理解指数，再理解ETF；先理解资产，再理解收益；先理解回撤，再理解复利。

ETF只是工具，真正决定长期表现的是它背后的资产：它代表哪个国家、哪个指数、哪个行业、有没有杠杆、波动有多大、能不能长期拿住。

## 宽基ETF：长期投资的地基

宽基ETF的作用，不是追求最刺激的收益，而是分享整个市场长期增长。

| ETF | 跟踪对象 | 特点 |
|---|---|---|
| SPY | 标普500 | 历史长、流动性强、期权活跃 |
| VOO | 标普500 | 费率低，适合长期持有 |
| IVV | 标普500 | 费率低，机构化程度高 |
| VTI | 美国全市场 | 覆盖范围比标普500更广 |

如果一个人刚开始做长期投资，通常应该先理解宽基，再理解行业主题和杠杆ETF。

## 纳指ETF：成长弹性更强

纳指类ETF代表科技成长，是长期复利里非常重要的一类。

| ETF | 类型 | 更适合谁 |
|---|---|---|
| QQQ | 纳斯达克100ETF | 想兼顾长期配置和期权交易的人 |
| QQQM | 低费率纳斯达克100ETF | 长期定投、不太做期权的人 |
| QLD | 2倍纳指ETF | 有风控能力、希望做进阶增强的人 |
| TQQQ | 3倍纳指ETF | 高风险承受者，只适合小仓位或交易型策略 |

我的理解是：

> QQQ是基准，QQQM是低费率长期版，QLD是更适合我做长期增强和Sell Put观察的品种，TQQQ则属于高风险进攻工具。

## 杠杆ETF：不能只看最终收益

杠杆ETF最容易让人误判。因为长期动态图的终点可能很好看，但过程中的回撤可能非常痛苦。

看QLD、TQQQ、SOXL这类资产，至少要同时看四个问题：

1. 最终收益是多少；
2. 最大回撤有多深；
3. 回撤后多久才能修复；
4. 投资者在最差阶段能不能继续执行。

没有回撤意识，只看终点收益，很容易把历史回测误读成未来承诺。

## 和Sell Put策略的关系

ETF研究不只是为了买入持有，也可以服务于接货型Sell Put。

| ETF | Sell Put适配度 | 原因 |
|---|---|---|
| SPY / QQQ | 高 | 流动性好，期权市场成熟，但资金占用较大 |
| QLD | 较高 | 波动足、价格相对更适合接货，但要严格看趋势 |
| TQQQ | 谨慎 | 权利金高，但接货后回撤压力大 |
| SOXL | 更谨慎 | 三倍杠杆叠加行业集中，不能只被高权利金吸引 |
| QQQM / VOO | 中低 | 更适合长期持有，期权活跃度不如QQQ/SPY |

## 老衲点评

这期视频的价值，是先帮读者建立ETF地图。不要一上来就问“买哪个收益最高”，而要先问：

- 我买的是宽基、成长、行业，还是杠杆？
- 这个ETF背后的指数是什么？
- 它的回撤我能不能承受？
- 它适合长期定投，还是只适合交易增强？
- 它能不能和Sell Put策略结合？

## 最后结论

ETF长期投资的核心不是代码，而是分类。

```text
宽基ETF = 地基
纳指ETF = 成长引擎
QLD = 进阶增强观察对象
TQQQ / SOXL = 高风险进攻工具
债券 / 黄金 = 周期和防守资产
BTC相关资产 = 高波动另类资产
```

先有地图，再谈选择；先看回撤，再谈收益。
