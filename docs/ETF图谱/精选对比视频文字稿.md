# ✍️ 精选对比视频文字稿

> 这个频道用于沉淀 B站「老衲有座庙」账号里的 ETF 长期投资、收益对比、平台对比、资产对比类视频。  
> B站负责展示动态图和视频节奏，网站负责沉淀文字稿、表格、结论、风险和后续更新。

## 频道定位

这个频道不是简单的视频链接合集，而是把视频整理成三层内容：

1. **视频原始入口**：嵌入 B站视频或放置待补 BV 号；
2. **文字稿整理版**：把视频口播、动态图结论、对比逻辑写成可阅读文章；
3. **投资框架升级版**：加入我的长期投资、定投、Sell Put 和风险控制点评。

## 每篇文字稿固定结构

| 模块 | 内容 |
|---|---|
| 视频入口 | 嵌入B站视频，或标记待补BV号 |
| 开场问题 | 这个视频到底解决什么投资问题 |
| 对比对象 | ETF代码、跟踪指数、资产类型 |
| 动态图结论 | 谁收益高、谁回撤大、谁更稳定 |
| 适合人群 | 新手、进阶、交易型、长期配置型 |
| Sell Put关系 | 是否适合做接货型Sell Put或Covered Call |
| 老衲点评 | 放到当前市场环境下，怎么理解 |
| 风险提示 | 强调历史收益不代表未来 |

## 后续维护规则

1. 每次新增 B站 ETF 对比视频，先进入这个频道；
2. 重要视频再单独加入「ETF长期收益图谱」首页；
3. 有 BV 号的视频要嵌入播放器；
4. 暂时找不到 BV 号的视频，先用「待补BV」方式保留文字稿；
5. 等你发来具体链接后，再补入视频和更精确的图表结论。
