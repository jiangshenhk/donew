# 杠杆ETF定投前，先问自己五个问题

> 本文是「定投是否可以用杠杆ETF？」系列的风险检查清单。  
> 在看 TQQQ、SOXL、QLD 的长期收益曲线之前，先确认自己是否真的适合承受杠杆ETF的波动。

## 一句话结论

杠杆ETF不是不能用于长期定投，但它不适合所有人。

真正的问题不是：

> 杠杆ETF历史收益高不高？

而是：

> 你能不能在它大幅回撤时继续执行？

如果做不到，历史收益曲线再漂亮，也很难变成真实账户收益。

---

## 问题一：你能接受多大的账户回撤？

普通ETF下跌 20% 时，2倍杠杆ETF可能跌得更深；普通ETF下跌 30% 时，3倍杠杆ETF的回撤可能非常难受。

所以在定投杠杆ETF前，先问自己：

| 问题 | 你的答案 |
|---|---|
| 账户回撤 30%，我还能继续投吗？ | 需要提前想清楚 |
| 回撤 50%，我会不会停止定投？ | 这是关键问题 |
| 如果回撤持续一年以上，我还能坚持吗？ | 很多人卡在这里 |

如果你只能接受小幅波动，杠杆ETF不适合做核心仓位。

## 问题二：你是用闲钱，还是用生活资金？

长期定投最怕两件事：

1. 资产在低位大跌；
2. 你刚好需要用钱。

杠杆ETF波动更大，如果资金不是长期闲钱，很容易在最差的时候被迫卖出。

所以更合理的顺序是：

```text
先保证生活现金流
再建立普通ETF底仓
最后才考虑杠杆ETF增强
```

## 问题三：你有没有普通ETF底仓？

不建议一上来就把 TQQQ、SOXL 当成全部仓位。

更稳健的结构是：

| 层级 | 代表ETF | 作用 |
|---|---|---|
| 底仓 | VOO / SPY / VTI / QQQ | 长期核心资产 |
| 增强 | QLD / SSO | 适度提高收益弹性 |
| 进攻 | TQQQ / SOXL | 小仓位承担高波动 |

如果没有底仓，直接上三倍ETF，账户波动会非常大，长期执行难度会明显提高。

## 问题四：你有没有定投纪律？

杠杆ETF定投不是只在上涨时投。

真正困难的是：

- 跌的时候继续投；
- 新闻很差时继续投；
- 账户回撤很大时继续投；
- 周围人都说“完了”的时候继续投。

如果只能在上涨时兴奋买入，下跌时停止，杠杆ETF可能会放大错误行为。

## 问题五：你是否理解路径依赖和波动损耗？

杠杆ETF不是简单的“长期收益 = 指数收益 × 杠杆倍数”。

它通常追踪的是每日收益的倍数，因此会受到路径影响。

举个简单理解：

```text
第一天跌很多
第二天涨回来
指数可能接近原点
但杠杆ETF可能仍然有损耗
```

所以杠杆ETF在持续单边上涨时表现很好，但在大幅震荡市场里，体验会差很多。

## 简单判断表

| 情况 | 是否适合杠杆ETF定投 |
|---|---|
| 没有普通ETF底仓 | 不建议直接开始 |
| 只能接受小回撤 | 不适合 |
| 有长期闲钱和稳定现金流 | 可以研究 |
| 能接受较大波动 | 可以小仓位尝试 |
| 对TQQQ/SOXL非常心动但没有风控 | 先看风险，不要重仓 |
| 已经有底仓，想增强收益 | 可以优先研究 QLD 这类2倍ETF |

## 我的基本建议

对于大多数人，顺序应该是：

```text
第一步：先定投普通ETF
第二步：理解2倍ETF，如 QLD
第三步：最后再研究3倍ETF，如 TQQQ / SOXL
```

不要反过来。

因为投资中最重要的不是短期刺激，而是长期能不能执行。

## 结论

杠杆ETF可以作为长期定投体系里的增强工具，但前提是：

1. 你已经有普通ETF底仓；
2. 你能承受深度回撤；
3. 你有长期现金流；
4. 你不会在大跌时被迫卖出；
5. 你知道杠杆ETF会放大收益，也会放大风险。

一句话总结：

> **杠杆ETF适合用来增强系统，不适合用来替代系统。**
