# 普通ETF、2倍ETF、3倍ETF，应该怎么分仓？

> 这是「定投是否可以用杠杆ETF？」系列里的仓位框架文章。  
> 核心目标不是告诉你固定比例，而是帮助你理解：不同风险承受能力的人，应该如何区分底仓、增强仓和进攻仓。

## 一句话结论

长期ETF投资不是在普通ETF和杠杆ETF之间二选一。

更合理的做法是分层：

```text
普通ETF = 底仓
2倍ETF = 增强仓
3倍ETF = 小仓位进攻仓
```

杠杆越高，仓位越要小；行业越集中，仓位越要谨慎。

---

## 三层仓位框架

| 层级 | 代表ETF | 作用 | 风险 |
|---|---|---|---|
| 底仓 | VOO / SPY / VTI / QQQ / QQQM | 长期核心资产 | 相对可控，但仍有市场回撤 |
| 增强仓 | QLD / SSO | 提高长期收益弹性 | 回撤明显放大 |
| 进攻仓 | TQQQ / SOXL / UPRO | 小仓位追求高弹性 | 极端回撤、路径依赖、心理压力 |

这个框架的重点是：

> 先有底仓，再谈增强；先能活下来，再追求更高收益。

## 保守型配置思路

适合：

- 刚开始定投ETF；
- 不喜欢太大回撤；
- 更重视长期稳定执行；
- 对杠杆ETF还不熟悉。

参考结构：

| 类型 | 比例思路 |
|---|---:|
| 普通ETF | 80% - 100% |
| 2倍ETF | 0% - 20% |
| 3倍ETF | 0% |

保守型投资者最好先把普通ETF定投做稳定，再研究2倍ETF。

## 平衡型配置思路

适合：

- 已经理解ETF长期定投；
- 能承受较大波动；
- 希望在长期底仓基础上提高收益弹性。

参考结构：

| 类型 | 比例思路 |
|---|---:|
| 普通ETF | 60% - 80% |
| 2倍ETF | 20% - 40% |
| 3倍ETF | 0% - 10% |

这个结构里，QLD 这类2倍ETF可以作为增强仓，但仍然不应该完全替代普通ETF。

## 进取型配置思路

适合：

- 风险承受能力较强；
- 有长期现金流；
- 可以接受较深回撤；
- 对杠杆ETF路径依赖有基本理解。

参考结构：

| 类型 | 比例思路 |
|---|---:|
| 普通ETF | 40% - 70% |
| 2倍ETF | 20% - 50% |
| 3倍ETF | 5% - 20% |

进取型也不等于无脑重仓三倍ETF。

TQQQ、SOXL 这类3倍ETF更适合放在进攻仓，而不是全部仓位。

## 为什么不建议三倍ETF满仓？

因为三倍ETF的问题不是最终收益，而是中途回撤。

在长期回测里，TQQQ、SOXL 看起来可能很漂亮，但真实执行时，你可能会遇到：

- 账户短期腰斩；
- 高点到低点回撤极深；
- 连续很长时间无法新高；
- 大跌时不敢继续定投；
- 因为压力过大，在最差阶段卖出。

如果一个策略你无法执行到底，它的历史收益就很难变成你的真实收益。

## 为什么2倍ETF值得重点研究？

2倍ETF处在一个中间位置。

比如 QLD：

- 比 QQQ 更有收益弹性；
- 比 TQQQ 更容易承受；
- 比 SOXL 分散度更好；
- 比三倍ETF更适合做长期增强框架。

所以对很多进阶投资者来说，研究顺序可以是：

```text
QQQ / QQQM → QLD → TQQQ / SOXL
```

而不是一开始就冲到三倍ETF。

## 和Sell Put策略怎么结合？

如果你还做 Sell Put，仓位需要更保守。

因为 Sell Put 本身已经承担了接货风险，不能再把所有仓位都压在高杠杆ETF上。

简单原则：

| 标的 | Sell Put适配度 | 仓位态度 |
|---|---|---|
| QQQ / SPY | 高 | 流动性好，但资金占用大 |
| QLD | 中高 | 适合研究，但要远OTM、分批 |
| TQQQ | 谨慎 | 权利金高，但接货压力大 |
| SOXL | 更谨慎 | 行业集中 + 三倍杠杆，风险更高 |

## 一个简单决策树

```text
你是新手吗？
  是 → 先普通ETF
  否 → 继续

你能接受50%以上回撤吗？
  否 → 不要重仓杠杆ETF
  是 → 继续

你有稳定现金流吗？
  否 → 降低杠杆仓位
  是 → 可以研究QLD

你已经有普通ETF底仓吗？
  否 → 先建立底仓
  是 → 可以考虑2倍ETF增强

你是否理解3倍ETF路径依赖？
  否 → 不碰或极小仓位
  是 → 小仓位研究TQQQ / SOXL
```

## 结论

普通ETF、2倍ETF、3倍ETF不是互相替代关系，而是不同层级的工具。

最稳妥的理解是：

```text
普通ETF解决“长期活下来”
2倍ETF解决“适度增强收益”
3倍ETF解决“小仓位进攻弹性”
```

长期投资的关键不是谁涨得最多，而是谁能被你长期执行。
