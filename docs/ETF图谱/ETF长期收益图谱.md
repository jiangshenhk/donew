# 三、财富差距来自两个选择

## 1. 选择什么标的

长期收益首先来自资产本身：

- 标普500：美国大型企业整体增长；
- 纳指100：科技创新和成长企业；
- 黄金：防御和价值储存；
- BTC等新资产：高波动、高成长潜力。

好的资产决定长期收益上限。

---

## 2. 选择什么方法

同一个资产，不同方法，最终结果也会不同：

| 方法 | 特点 |
|---|---|
| 一次投入 | 收益最大化，但需要承受波动 |
| 定投 | 降低择时压力，提高执行概率 |
| 杠杆ETF | 放大长期趋势，同时放大回撤 |
| Sell Put | 提高现金收益，增强资金效率 |

投资不是只有“买什么”，还有“怎么买”。

# 四、杠杆ETF：收益放大器，而不是普通ETF替代品

杠杆ETF最大的误解是：

> 只要长期上涨，就应该全部买杠杆。

实际上：

- 上涨趋势中，杠杆放大收益；
- 熊市中，杠杆放大亏损；
- 震荡市场中，会产生波动损耗。

因此：

```text
普通ETF = 长期底仓
QQQ = 成长增强
QLD = 2倍增强
TQQQ/SOXL = 高风险进攻仓
```
