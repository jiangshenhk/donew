## 三、看懂结果：标的和方法必须分开

### 1. 选标的：决定长期方向

| 标的类别 | 代表 | 长期角色 |
|---|---|---|
| 美国大盘 | SPY / VOO / IVV | 稳健底仓 |
| 科技成长 | QQQ / QQQM | 成长核心 |
| 黄金 | GLD / IAU | 防守与分散 |
| 中国核心权益 | 沪深300相关ETF | 中国经济权益敞口 |
| 半导体行业 | SOXX / SMH | 高成长、高周期 |

标的本身有没有长期增长，是所有方法能够成立的前提。

### 2. 选方法：决定放大方式

| 方法 | 收益特点 | 主要代价 | 更适合的定位 |
|---|---|---|---|
| 普通ETF | 完整参与长期趋势 | 收益弹性相对有限 | 核心底仓 |
| 2倍杠杆ETF | 明显提高趋势弹性 | 回撤、波动损耗上升 | 增强仓 |
| 3倍杠杆ETF | 极大放大长期结果 | 极端回撤与路径依赖 | 小仓位进攻 |
| 定投 | 降低单次择时压力 | 需要长期现金流和纪律 | 长期执行方法 |
| Sell Put | 提高闲置现金收益 | 接货与尾部风险 | 资金效率增强 |

> **杠杆ETF不是一种新的资产，而是作用在底层资产上的投资方法。**

## 四、为什么必须同时展示年化收益和最大回撤？

只看最终金额，很容易得出“收益最高的就是最好”的错误结论。

真正完整的长期回测至少要同时看四项：

| 指标 | 它回答的问题 |
|---|---|
| 最终资金 | 最后赚了多少？ |
| 累计收益 | 整个周期放大了多少倍？ |
| 平均年化收益率 | 长期复利速度有多快？ |
| 最大回撤 | 最痛苦的时候会亏多少？ |

此外还要看：

- 回撤后的修复时间；
- 是否有持续现金流继续定投；
- 极端下跌时是否会被迫卖出；
- 真实投资者能否承受账户大幅缩水。

## 五、我的长期ETF分层框架

```text
普通宽基ETF = 地基
QQQ / QQQM = 成长核心
QLD = 2倍增强
TQQQ / SOXL = 高风险进攻仓
```

这不是简单地追求最高收益，而是在收益、回撤和执行难度之间做分层。

长期财富最终来自：

```text
最终财富
=
优质标的
×
合适方法
×
足够时间
```

选标的决定方向，选方法决定放大倍数，风险控制决定你能否坚持到最后。
