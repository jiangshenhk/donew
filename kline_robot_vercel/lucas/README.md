## 1. 航班实时跟踪 — lucas_fly.html

### 功能
- 调用 OpenSky Network API，在中国区域（BBOX）内获取实时航班数据
- 使用 Leaflet 地图标注每架飞机的位置，飞机图标带航向旋转
- 按高度分色：低空 `<3000m`（绿）、中空 `3000-7000m`（橙）、高空 `>7000m`（红）
- 点击飞机弹窗显示：航班号、国家、高度、速度、航向
- 每 30 秒自动刷新，可手动刷新

### 结构
- `index.html` — 页面入口
- Leaflet 开源地图 + OpenSky REST API
- 无后端，纯浏览器端请求

## 2. 墨尔本活动地图 — melbourne-events.html

### 功能
- 侧边栏输入活动类型（如 music、food、sports），调用 DeepSeek API 搜索墨尔本近期活动
- 返回结果以卡片列表展示，同时在地图上标注活动位置
- 支持搜索词缓存，避免重复请求
- 活动卡片包含：标题、时间、地点、简介；标注点分色显示

### 结构
- 单 HTML 文件，Leaflet 地图 + DeepSeek API（通过 `/api/proxy` 代理）
- 侧边栏 UI：搜索框 + 结果列表
- 缓存策略：localStorage 缓存搜索结果

## 3. Sky Fighters 🛩️ — lucas_flyplay.html

### 类型
3D 第一人称飞行射击游戏（Three.js）

### 游戏规则
- 玩家驾驶飞机在三维世界中飞行，自动向前，鼠标/触摸控制飞机位置
- 敌机从前方来袭，自动射击，击落敌机得分
- 四种敌机类型：白（1HP，10分）→ 绿（2HP，20分）→ 蓝（5HP，50分）→ 红（10HP，100分）
- 等级随存活时间提升，每 15 秒升一级
- 威胁等级每 30 秒提升，敌机速度加快、射速提高
- 拾取武器升级道具（W星）提升武器等级（最多 10 级），弹幕更密集
- 三条命，碰撞敌机或中弹扣命，用完游戏结束

### 难度
- Easy（1×）、Medium（3×）、Hard（5×）— 影响敌机生成数量

### 控制
- 鼠标/触摸移动飞机，自动射击
- 空格/回车开始游戏

### 技术
- Three.js（CDN 引入）
- Canvas 绘制得分/生命/HUD
- localStorage 保存最高分

## 4. 深渊守护者 — lucas_zdgy.html

### 类型
2D 竖屏塔防射击游戏（Canvas 2D）

### 布局
```
┌──────────┬──────────────┐
│ 礼物通道  │  敌人通道     │
│ (50%)     │  (50%)       │
│ [8个礼物]  │ [门] 91×26   │
│ 垂直排列   │ [👹魔鬼]60×54│
│ 底部放大   │ [☠️小鬼]7列   │
│ 1.5倍     │  5-7行       │
├──────────┴──────────────┤
│    🛡️ 状态面板           │
│    🤺 战士               │
└─────────────────────────┘
```

### 核心机制

#### 礼物系统
- 8 个礼物槽垂直排列，顶部刷新、底部放大（1.5×）
- 打掉后新礼物从顶部插入，所有礼物平滑下移
- 每个礼物随机提供三种升级之一（+1 或 +2）：弹道/威力/速度
- 高加成（+2）的礼物血量为 7-12（约普通 2-3 倍），普通（+1）为 3-5
- 全满后礼物继续刷新但只加分

#### 升级属性

| 属性 | 范围 | 说明 |
|------|------|------|
| 🔫 弹道 | 1→10 | 每发子弹数，越多弹幕越密集 |
| 💥 威力 | 1→5 | 每发伤害，≥3 时带范围爆炸（AOE） |
| ⚡ 速度 | 4→10 | 战士左右移动速度 |

#### 衰减系统
- 每 10 秒倒计时归零时，按优先级减一级：💥威力 → ⚡速度 → 🔫弹道（最低 1/4/1）
- 打掉任意礼物立即重置倒计时
- 底部状态面板显示倒计时进度条

#### 敌人系统

| 敌人 | 尺寸 | HP | 分值 | 到底惩罚 |
|------|------|----|------|---------|
| ☠️小鬼 | 13×13 | 1 | 10 | 消耗 1 级效果 |
| 👹魔鬼 | 60×54 | 80+时间×0.12 | 100 | 消耗 4 级效果 |
| 🚪门 | 91×26 | 200+时间×0.25 | 200 | 消耗 2 级效果 |

- 小队（魔鬼 + 5-7 排小鬼）每 6 秒生成一个
- 门每 12 秒生成一个（每 2 个小队后）
- 所有单位速度统一：`0.35 + 帧数 × 0.00003`
- 魔鬼与第一排小鬼间距 5 行（90px）

#### 护盾抵扣
- 敌人走到底部时消耗效果等级抵扣：优先 💥威力 → ⚡速度 → 🔫弹道
- 🛡️ = (威力-1) + (速度-4) + (弹道-1)
- 效果不足则游戏结束

#### 控制
- ← → / A,D / 鼠标移动 / 触摸拖动
- 自动向上射击

#### 技术
- Canvas 2D 纯前端渲染
- 虚拟分辨率 420×780，竖屏居中缩放
- 所有单位统一速度，精确间距控制
- 礼物掉落动画 + 缩放过渡效果
