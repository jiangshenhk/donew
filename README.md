# 十方斋｜复利投资笔记

这是一个记录长期复利投资、ETF定投、Sell Put 策略、美股市场结构日报的网站。

核心关注：

- **ETF收益对比**：长期投资 ETF，到底选普通ETF、2倍ETF，还是3倍ETF；
- **ETF定投**：用长期持有理解复利；
- **Sell Put策略**：用概率、波动率、安全垫和仓位管理提高现金使用效率；
- **市场结构日报**：跟踪资金流、油价、利率、美元、黄金、BTC、VIX与风险资产联动；
- **核心标的**：QQQ / QLD / SPY / MSTR / BTC / GLD / IAU 等；
- **交易纪律**：不追高、不贴价、不满仓，优先远 OTM、分批、可接货。

网站目标不是做新闻摘要，而是沉淀一套可复盘、可执行的投资判断框架。

## ETF收益对比：带杠杆还是不带？

> 很多人长期投资 ETF 时，真正纠结的不是“要不要买 ETF”，而是：普通ETF、2倍ETF、3倍ETF，到底哪个更适合自己。

这个栏目用于整理「老衲有座庙」B站账号里与 ETF 长期收益对比、定投回测、杠杆ETF、指数基金相关的动态图内容。

重点不是简单比较谁最后涨得最多，而是把每条视频整理成：

- 普通ETF、2倍ETF、3倍ETF的收益差异；
- 长期收益排序；
- 最大回撤与修复时间；
- 定投适配度；
- Sell Put适配度；
- 当前市场下的老衲点评。
